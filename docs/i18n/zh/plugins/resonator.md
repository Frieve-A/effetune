# 共鸣器插件

这些插件会强调特定共鸣特性，为音乐加入独特的音色、存在感或扬声器般的色彩。

## 插件列表

- [Horn Resonator](#horn-resonator) - 模拟号角扬声器系统的共鸣
- [Horn Resonator Plus](#horn-resonator-plus) - 更平滑的号角扬声器共鸣，带来自然的聆听色彩
- [Modal Resonator](#modal-resonator) - 最多 5 个共鸣器的频率共鸣效果

## Horn Resonator

使用数字波导模型模拟号角负载扬声器共鸣的插件。它适合为音乐加入号角扬声器特有的存在感、亮度或投射感。

### 听觉体验指南
- 为人声、铜管或原声乐器加入号角式存在感
- 给音乐增加更直接、更有投射感的扬声器色彩
- 通过尺寸参数塑造从紧凑到宽松的共鸣特性

### 参数

- **Crossover (Hz)** - 设置低频路径（延迟）和高频号角模型路径之间的分频点（20-5000 Hz）
- **Horn Length (cm)** - 调整模拟号角长度。较长的号角会把共鸣移向更低频，并让共鸣间距更密；较短的号角会把共鸣移向更高频，并让间距更大，听起来更紧（20-120 cm）
- **Throat Diameter (cm)** - 控制号角喉口（输入端）尺寸。较小值倾向于增加亮度和上中频强调，较大值增加温暖感（0.5-50 cm）
- **Mouth Diameter (cm)** - 控制号角开口（输出端）尺寸。它影响与周围空气的阻抗匹配，以及开口处的频率相关反射。较大值通常让听感更宽，并减少低频反射；较小值更集中，并增加低频反射（5-200 cm）
- **Curve (%)** - 调整号角扩张形状（半径从喉口到开口如何增加）
  - 0%：近似圆锥
  - 正值：更接近指数型扩张
  - 负值：更接近 tractrix 型扩张
- **Damping (dB/m)** - 设置号角波导内部每米的衰减（吸音量）。较高值会减少共鸣峰和振铃，让声音更平滑、更受阻尼控制（0-10 dB/m）
- **Throat Reflection** - 调整号角喉口（输入端）的反射系数。较高值会让更多声音从喉口边界反射回号角，可能让响应更明亮并强调某些共鸣（0-0.99）
- **Output Gain (dB)** - 控制处理后高频路径在与延迟低频路径混合前的整体输出电平，可用于匹配或提升效果电平（-36 到 +36 dB）

### 快速入门
1. 设置 **Crossover**，定义送入号角模型的频率范围（例如 800-2000 Hz）。低于此分频点的频率会被延迟并混回。
2. 从约 60-70 cm 的 **Horn Length** 开始，可获得典型的中频性格。
3. 用 **Throat Diameter** 和 **Mouth Diameter** 调整核心音色（亮度/温暖、集中/宽松）
4. 用 **Curve** 微调共鸣性格（0% 圆锥，正值接近指数，负值接近 tractrix）
5. 用 **Damping** 和 **Throat Reflection** 调整号角共鸣的平滑或强调
6. 用 **Output Gain** 平衡号角声音与延迟低频的电平

## Horn Resonator Plus

Horn Resonator Plus 为音乐加入更平滑、更自然的号角扬声器色彩。它适合让人声、铜管、原声乐器或整首音乐更温暖、更有活力，同时让共鸣比标准 Horn Resonator 更不尖锐。

它基于与 [Horn Resonator](#horn-resonator) 相同的号角模型，但使用更细致的号角开口和喉口反射模型，使共鸣衰减更平滑。

### 聆听指南

- 更平滑的号角色彩：加入号角负载扬声器特性，但减少尖锐振铃
- 更温暖的存在感：可让人声、铜管和原声音乐更有生命力
- 更自然的高频行为：高频范围比标准版本更接近声学号角或号角扬声器

### 技术增强

- **2nd-order mouth reflection filter**：更平滑地模拟号角开口处的频率相关反射
- **Frequency-dependent throat reflection**：喉口反射会随频率变化，得到更自然的号角行为

### 参数和使用

Horn Resonator Plus 使用与 [Horn Resonator](#horn-resonator) 相同的参数。关于每个参数的详细说明，请参考 Horn Resonator 部分。

### 使用指南

- **Horn Resonator**：需要较轻处理并获得基本号角色彩时选择
- **Horn Resonator Plus**：想要更平滑、更自然的号角色彩，并且可以接受略高 CPU 使用量时选择

### 快速入门指南

使用与 [Horn Resonator](#horn-resonator) 相同的控制。想要更平滑的号角扬声器特性时选择 Horn Resonator Plus。

## Modal Resonator

为音乐加入调谐共鸣的效果，类似物体或扬声器部件在自身固有频率上振动。想在聆听时增加闪亮感、厚度、金属色彩或扬声器般共鸣时可以使用。

### 听觉体验指南

- **金属共鸣 (Metallic Resonance)：**
  - 创建会跟随原始素材动态变化的钟声般或金属质感音调
  - 适合为打击乐、合成器或整首音乐加入闪亮感或金属色彩
  - 使用多个经过细致调谐的共鸣器，并搭配中等 Decay 时间
- **声音增强：**
  - 为音乐加入可控的共鸣和音色
  - 增强特定频率范围的存在感
  - 可让声音更有身体感、亮度或空间感
- **扬声器特性：**
  - 模拟特定扬声器类型的共鸣特征
  - 加入箱体或材料共振般的色彩
  - 适合探索不同播放系统的味道
- **特殊效果：**
  - 创建不寻常的音色和超现实质感
  - 当您想要明显共鸣效果而不是自然增强时很有用
  - 只有在想让共鸣成为声音本身的一部分时，才尝试极端设置

### 参数

- **Resonator Selection (1-5)** - 五个独立共鸣器，可分别启用/禁用和配置。
  - 使用多个共鸣器可获得复杂、分层的共鸣效果。
  - 每个共鸣器都可以针对不同频率区域。
  - 可尝试让多个共鸣器之间形成谐波关系，使结果更有音乐性。

每个共鸣器：
- **Enable** - 开启或关闭该共鸣器
- **Freq (Hz)** - 共鸣频率（20 到 20,000 Hz）
- **Decay (ms)** - 共鸣在输入声音之后持续的时间（1 到 500 ms）
- **LPF Freq (Hz)** - 低通滤波器，用于塑造共鸣音色（20 到 20,000 Hz）
- **HPF Freq (Hz)** - 高通滤波器，用于移除共鸣中的不需要低频（20 到 20,000 Hz）
- **Gain (dB)** - 控制每个共鸣器的单独输出电平（-18 到 +18 dB）

全局控制：

- **Mix (%)** - 平衡所有已启用共鸣器的合成输出与原始声音（0 到 100%）

### 推荐设置

1. **轻微扬声器增强：**
   - 启用 2-3 个共鸣器
   - Freq：400 Hz、900 Hz、1600 Hz
   - Decay：60-100ms
   - LPF Freq：2000-4000 Hz
   - Mix：10-20%

2. **金属色彩：**
   - 启用 3-5 个共鸣器
   - Freq：分布在 1000-6500 Hz 之间
   - Decay：100-200ms
   - LPF Freq：4000-8000 Hz
   - Mix：15-30%

3. **低频增强：**
   - 启用 1-2 个共鸣器
   - Freq：50-150 Hz
   - HPF Freq：20-60 Hz，并保持低于目标共鸣
   - Decay：50-100ms
   - LPF Freq：1000-2000 Hz
   - Mix：10-25%

4. **全频扬声器模拟：**
   - 启用全部 5 个共鸣器
   - Freq：100 Hz、400 Hz、800 Hz、1600 Hz、3000 Hz
   - HPF Freq：20 Hz、120 Hz、250 Hz、500 Hz、1000 Hz
   - Decay：从低到高逐渐变短（100ms 到 30ms）
   - LPF Freq：从低到高逐渐提高（2000Hz 到 4000Hz）
   - Mix：20-40%

### 快速入门指南
1. **选择频率：**
   - 从 1 个共鸣器开始
   - 选择想要强调的频率
   - 低频用于厚度，高频用于亮度

2. **调整特性：**
   - 用 `Decay` 控制共鸣持续时间
   - 用 `LPF Freq` 塑造音色
   - 特别是低频设置时，将 `HPF Freq` 设在要保留的共鸣频率下方
   - 更长的 Decay 会产生更明显、类似钟声的音调

3. **与原声混合：**
   - 从较低 `Mix` 开始
   - 逐步增加，直到听到想要的色彩
   - 旁路比较，确认效果没有压过音乐
