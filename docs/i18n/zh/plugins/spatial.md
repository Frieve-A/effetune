# 空间音频插件

这些插件用于调整音乐在耳机或扬声器中的空间呈现，让声场更自然、更舒适，或修正左右偏移。

## 插件列表

- [Crossfeed Filter](#crossfeed-filter) - 用于自然耳机声像的交叉馈送滤波器
- [MS Matrix](#ms-matrix) - 在立体声与 Mid/Side 之间转换，用于高级立体声调整链
- [Multiband Balance](#multiband-balance) - 5 频段频率相关立体声平衡控制
- [Stereo Blend](#stereo-blend) - 从极性互换立体声、单声道到增强立体声的宽度控制

## Crossfeed Filter

耳机交叉馈送滤波器，模拟扬声器聆听时左右耳之间自然存在的声学串音。它可以减少耳机中“声音贴在两耳边”的感觉，让声像更接近自然聆听。

### 主要功能
- 为耳机聆听模拟自然声学串音
- 可调交叉馈送电平和时间
- 用低通滤波模拟频率相关串音
- 仅处理立体声（单声道或其他非立体声信号会自动旁路）

### 参数
- **Level** (-60 dB 到 0 dB)：控制交叉馈送信号量
  - 较低值：效果更微妙
  - 较高值：左右融合更明显
  - 从 -12 dB 左右开始
- **Delay** (0 ms 到 1 ms)：模拟声音到达另一只耳朵的时间差
  - 较短延迟：更紧密的声像
  - 较长延迟：更宽松的自然感
- **LPF Freq** (100 Hz 到 20000 Hz)：控制交叉馈送的频率响应
  - 较低值（500-1000 Hz）：更自然的频率相关串音
  - 较高值（1000-20000 Hz）：更宽的频率响应

### 推荐设置

1. 自然耳机聆听
   - Level: -12 dB
   - Delay: 0.3 ms
   - LPF Freq: 700 Hz
   - 效果：细微交叉馈送，适合长时间舒适聆听

2. 扬声器模拟
   - Level: -6 dB
   - Delay: 0.5 ms
   - LPF Freq: 1000 Hz
   - 效果：更明显的扬声器式呈现

3. 细微增强
   - Level: -20 dB
   - Delay: 0.2 ms
   - LPF Freq: 500 Hz
   - 效果：非常轻柔的交叉馈送，适合敏感听者

### 应用指南
- 先从较低 Level 开始，再逐步增加
- 交替开启/关闭比较是否更自然
- 如果声场变窄过多，请降低 Level
- 不同耳机可能需要不同设置

### 快速入门指南
1. 将 Level 设为 -12 dB
2. 将 Delay 设为 0.3 ms
3. 将 LPF Freq 设为 700 Hz
4. 播放熟悉的人声或现场录音
5. 微调 Level，直到声场更自然但不过度变窄

## MS Matrix

MS Matrix 可将普通立体声转换为 Mid/Side 格式，也可将 Mid/Side 音频转换回普通立体声。当您想在效果链中分别调整中心信息和侧向信息时使用，例如先编码为 M/S，改变 Mid 或 Side 电平，再解码回立体声。若只是想直接调整普通音乐的立体声宽度，[Stereo Blend](#stereo-blend) 更直接。

### 主要功能
- 分别调整 Mid 与 Side 增益（–18 dB 到 +18 dB）
- Mode 切换：Encode (Stereo→M/S) 或 Decode (M/S→Stereo)
- 可在编码前或解码后选择 Swap L/R

### 参数
- **Mode** (Encode/Decode)：Encode 会把左右立体声转换为左声道 Mid、右声道 Side。Decode 会把左声道视为 Mid、右声道视为 Side，并重建普通立体声。
- **Mid Gain** (–18 dB 到 +18 dB)：调整所选转换过程中的 Mid 电平。
- **Side Gain** (–18 dB 到 +18 dB)：调整所选转换过程中的 Side 电平。
- **Swap L/R** (Off/On)：在编码前或解码后交换左右声道。

### 推荐设置
1. **普通立体声的轻微加宽**
   - 第一个 MS Matrix：Mode: Encode, Mid Gain: 0 dB, Side Gain: +3 dB, Swap: Off
   - 后面的第二个 MS Matrix：Mode: Decode, Mid Gain: 0 dB, Side Gain: 0 dB, Swap: Off
   - 效果：稍微增强 Side 成分，然后恢复为普通立体声

2. **普通立体声的中心聚焦**
   - 第一个 MS Matrix：Mode: Encode, Mid Gain: +3 dB, Side Gain: -3 dB, Swap: Off
   - 后面的第二个 MS Matrix：Mode: Decode, Mid Gain: 0 dB, Side Gain: 0 dB, Swap: Off
   - 效果：让人声和居中声音更靠前，同时减少侧向氛围

3. **解码已有 M/S 音频**
   - Mode: Decode
   - Mid Gain: 0 dB
   - Side Gain: 0 dB
   - Swap: Off
   - 仅在输入信号已经是 Mid/Side 格式时使用

4. **创意翻转**
   - Mode: Encode
   - Mid Gain: 0 dB
   - Side Gain: 0 dB
   - Swap: On

### 快速入门指南
1. 先决定只需要一次转换，还是需要完整的 Encode -> 调整 -> Decode 链
2. 普通立体声聆听时，先放一个 Encode 模式的 MS Matrix，再在后面放一个 Decode 模式的 MS Matrix
3. 在 Encode 阶段调整 **Mid Gain** 和 **Side Gain**
4. 只有在需要通道校正或创意反转时才启用 **Swap L/R**
5. 旁路比较，确认立体声声像仍然自然

## Multiband Balance

这是一个频率相关的平衡处理器，会把音频分成五个频段，并让您分别将每个频段略微向左或向右移动。它适合低频、人声、镲片或其他频率范围偏向一侧时，只修正那部分声音，而不移动整首歌。

### 主要特点
- 5 频段频率相关立体声平衡控制
- 高质量 Linkwitz-Riley 分频滤波器
- 线性平衡控制，便于精细调整
- 左右声道独立处理
- 重置分频滤波器时自动淡变处理

### 参数

#### 分频点频率
- **Freq 1** (20-500 Hz)：Low 与 Low-Mid 的分界
- **Freq 2** (100-2000 Hz)：Low-Mid 与 Mid 的分界
- **Freq 3** (500-8000 Hz)：Mid 与 High-Mid 的分界
- **Freq 4** (1000-20000 Hz)：High-Mid 与 High 的分界

#### 频段控制
每个频段都有独立的平衡控制：
- **Band 1 Bal.** (-100% 到 +100%)：控制低频的立体声平衡
- **Band 2 Bal.** (-100% 到 +100%)：控制低中频的立体声平衡
- **Band 3 Bal.** (-100% 到 +100%)：控制中频的立体声平衡
- **Band 4 Bal.** (-100% 到 +100%)：控制高中频的立体声平衡
- **Band 5 Bal.** (-100% 到 +100%)：控制高频的立体声平衡

### 推荐设置

1. 修正高频偏右
   - Low Band (20-100 Hz): 0%（居中）
   - Low-Mid (100-500 Hz): 0%
   - Mid (500-2000 Hz): 0%
   - High-Mid (2000-8000 Hz): -10% 到 -25%
   - High (8000+ Hz): -10% 到 -30%
   - 效果：将明亮内容略向左移，同时保持低频和人声稳定

2. 修正低中频偏左
   - Low Band: 0%
   - Low-Mid: +10% 到 +25%
   - Mid: +5% 到 +15%
   - High-Mid: 0%
   - High: 0%
   - 效果：将温暖的厚度和较低的人声区域略向右移，而不改变整个立体声图像

3. 保持低频居中，同时调整空气感
   - Low Band: 0%
   - Low-Mid: 0%
   - Mid: 0%
   - High-Mid: +5% 到 +15%
   - High: +10% 到 +20%
   - 效果：轻微将高频氛围向右移，同时保持低频居中

### 应用指南

1. 聆听平衡校正
   - 低频（100 Hz 以下）保持居中以获得稳定低音
   - 只移动听起来偏离中心的频率范围
   - 先使用小的有符号数值（约 5-20%）
   - 检查单声道播放是否带来音色或电平变化

2. 问题修正
   - 重新平衡过于偏左或偏右的频率范围
   - 通过居中低频来收紧不聚焦的低音
   - 减少高频中刺耳的立体声伪影
   - 改善不同声音部位分别偏向不同侧的录音

3. 创意聆听效果
   - 创建不寻常的频率相关位置
   - 让高频偏向一侧，同时低频保持居中
   - 通过轻微移动上部频段，让氛围听起来更宽

4. 立体声场调整
   - 按频段微调立体声平衡
   - 修正不均匀的立体声分布
   - 不要把它当作整体宽度控制；要整体加宽或收窄时请使用 Stereo Blend
   - 保持单声道兼容性

### 快速入门指南
1. 先将所有 Balance 保持在 0%
2. 播放熟悉的音乐，找出偏向一侧的频率范围
3. 只调整相关频段
4. 每次以小步进调整，并频繁旁路比较

## Stereo Blend

这个效果通过调整音乐的立体声宽度，帮助获得更自然的声场。它尤其适合耳机聆听，因为它可以减少耳机中常见的过度立体声分离，让聆听体验更自然、更不容易疲劳。需要时，它也可以增强扬声器聆听的立体声图像。

### 聆听增强指南
- 耳机优化：
  - 减小立体声宽度（60-90%），获得更自然、类似扬声器的呈现
  - 减少过度立体声分离造成的聆听疲劳
  - 创建更真实、以前方为中心的声场
- 扬声器增强：
  - 保持原始立体声图像（100%），用于准确重放
  - 需要时使用轻微增强（110-130%）获得更宽声场
  - 谨慎调整以保持自然声场
- 声场控制：
  - 以自然、真实的呈现为目标
  - 避免过宽而听起来不自然
  - 负宽度只建议用于校正或创意侧向极性反转
  - 针对具体聆听环境优化

### 参数
- **Stereo** - 控制立体声宽度（-200% 到 200%）
  - 负值：在重建前反转立体声侧向（L-R）成分的极性
  - -200%：带反相侧向极性的最大宽度；只用于校正或特殊情况
  - -100%：原始立体声宽度，但左右声像互换
  - 0%：完全单声道（左右声道相加）
  - 100%：原始立体声图像
  - 200%：最大宽度增强；保留中心成分，同时强烈增强侧向差异

### 不同聆听场景的推荐设置

1. 耳机聆听（自然）
   - Stereo: 60-90%
   - 效果：减少立体声分离
   - 适合：长时间聆听、降低疲劳

2. 扬声器聆听（参考）
   - Stereo: 100%
   - 效果：原始立体声图像
   - 适合：准确重放

3. 扬声器增强
   - Stereo: 110-130%
   - 效果：轻微宽度增强
   - 适合：扬声器摆位较近的房间

### 音乐风格优化指南

- 古典音乐
  - 耳机：70-80%
  - 扬声器：100%
  - 优点：自然的音乐厅视角

- 爵士与原声
  - 耳机：80-90%
  - 扬声器：100-110%
  - 优点：亲密、真实的合奏声音

- 摇滚与流行
  - 耳机：85-95%
  - 扬声器：100-120%
  - 优点：在没有人工宽化感的情况下保持平衡冲击力

- 电子音乐
  - 耳机：90-100%
  - 扬声器：100-130%
  - 优点：保持聚焦的同时获得受控的空间感

### 快速入门指南
1. 选择您的聆听设置
   - 确认您使用的是耳机还是扬声器
   - 这会决定调整的起点

2. 从保守设置开始
   - 耳机：从 80% 开始
   - 扬声器：从 100% 开始
   - 聆听声音位置是否自然

3. 根据音乐微调
   - 每次小幅调整（5-10%）
   - 专注于获得自然声场
   - 注意聆听舒适度

请记住：目标是获得自然、舒适的聆听体验，减少疲劳并保持音乐原本的呈现。避免使用一开始听起来很惊艳、但久听容易疲劳的极端设置。
