# 分析器插件

这些插件让您用可视化方式观察音乐。它们会显示声音的不同侧面，帮助您理解正在听到的内容，也让聆听过程更有参与感。

## 插件列表

- [Level Meter](#level-meter) - 显示数字信号电平和可能的削波
- [Oscilloscope](#oscilloscope) - 实时显示波形
- [Spectrogram](#spectrogram) - 用彩色图案显示音乐变化
- [Spectrum Analyzer](#spectrum-analyzer) - 显示音乐中的不同频率
- [Stereo Meter](#stereo-meter) - 可视化立体声平衡与声道相关性

## Level Meter

实时显示音乐的数字信号电平。它适合检查加上效果后的电平，并在削波变成可闻失真之前发现风险。

### 可视化指南
- 信号电平越大，水平条越向右延伸
- 白色标记会短暂显示最近的最高电平
- OVERLOAD 表示信号超过安全的数字范围，可能产生失真
- 为了干净播放，尽量避免红色电平或 OVERLOAD 警告频繁出现；实际聆听音量请在播放设备上调节

## Oscilloscope

实时显示声音波形的形状，让您在聆听时看到节拍、尖锐起音和音量变化。波形有重复性时，可用触发设置让显示更稳定。

### 可视化指南
- 横轴表示时间（毫秒）
- 纵轴表示归一化振幅；可见范围会随 Display Level 和 Vertical Offset 改变
- 绿色线条描绘实际波形
- 网格线用于粗略观察时间和振幅
- 触发设置决定波形捕捉从哪里开始；界面不会显示单独的触发点标记

### 参数
- **Display Time** - 显示的时间长度（1 到 100 ms）
  - 较低值：更细致地显示短促变化
  - 较高值：显示更长的波形模式
- **Trigger Mode**
  - Auto：即使没有触发也持续更新
  - Normal：保持显示直到下一次触发
- 触发检测使用左右声道的平均波形。单声道输入会直接分析。
- **Trigger Level** - 开始捕捉的振幅电平
  - 范围：-1 到 1（归一化振幅）
- **Trigger Edge**
  - Rising：信号上升时触发
  - Falling：信号下降时触发
- **Holdoff** - 两次触发之间的最短时间（0.1 到 10 ms）
- **Display Level** - 垂直缩放（-96 到 0 dB）
- **Vertical Offset** - 将波形上下移动（-1 到 1）

### 波形显示说明
显示的波形会在采样点之间进行线性插值，以便更平滑地观察。请把它作为视觉参考，而不是精密测量工具。

## Spectrogram

用彩色图案显示音乐随时间的变化。颜色表示各频率成分的强弱，纵向位置表示频率高低。

### 可视化指南
- 颜色表示不同频率的强度：
  - 暗色：较安静的声音
  - 亮色：较响亮的声音
  - 图案会随着音乐变化
- 纵向位置表示频率：
  - 下方：低频
  - 中部：主要乐器和人声区域
  - 上方：高频

### 可以看到什么
- 旋律：流动的彩色线条
- 节拍：垂直条纹
- 贝斯：底部的明亮区域
- 和声：多条平行线
- 不同乐器会形成各自的图案

### 参数
- **DB Range** - 颜色显示的动态范围（-144dB 到 -48dB）
  - 较低数值：显示更细微的声音
  - 较高数值：集中显示主要声音
- **Points** - 用于显示的 FFT 大小（256 到 16384）
  - 较高数值：频率细节更多，但时间更新较慢
  - 较低数值：动作更快，但频率细节较少
- 该分析器使用左右声道平均值。单声道输入会直接分析。

## Spectrum Analyzer

实时显示音乐从深沉低频到明亮高频的频率分布。您可以把它看作观察构成音乐整体声音的各个成分。

### 可视化指南
- 左侧表示低频（鼓、贝斯等）
- 中间表示主要频段（人声、吉他、钢琴等）
- 右侧表示高频（镲片、明亮感、空气感等）
- 峰值越高，表示该频率存在感越强
- 深绿色线条表示当前声音
- 亮绿色线条会短暂保持最近峰值，便于看到刚刚经过的强声音
- 可以观察不同乐器形成不同频谱图案的方式

### 可以看到什么
- 低音下潜：左侧的大幅运动
- 人声旋律：中间区域的活动
- 清晰高频：右侧的闪亮区域
- 完整混音：所有频率如何一起组成整体声音

### 参数
- **DB Range** - 显示灵敏度（-144dB 到 -48dB）
  - 较低数值：显示更细微的声音
  - 较高数值：集中显示主要声音
- **Points** - 显示能把相邻频率分得多细（256 到 16384）
  - 较高数值：频率细节更多，但更新较慢
  - 较低数值：更新更快，但频率细节较少
- 该分析器使用左右声道平均值。单声道输入会直接分析。

### 这些工具的有趣用法

1. 探索音乐
   - 观察不同流派产生的不同图案
   - 比较原声音乐和电子音乐的差异
   - 观察乐器占据哪些频率区域

2. 理解声音
   - 用眼睛观察电子音乐中的低频
   - 看人声旋律如何穿过显示区域
   - 观察鼓声如何形成尖锐图案

3. 扩展聆听体验
   - 用 Level Meter 检查添加效果后的信号峰值
   - 看 Spectrum Analyzer 随音乐跳动
   - 用 Spectrogram 制造像灯光秀一样的视觉体验

## Stereo Meter

一个可视化工具，用于观察立体声如何形成空间感。您可以看到声音在扬声器或耳机之间如何展开，为聆听增加一层视觉维度。

### 可视化指南
- **Diamond Display** - 显示音乐运动的主窗口：
  - 中心：非常安静的瞬间，或左右合成信号接近零的瞬间
  - 上/下：左右声道共有的成分，例如居中或接近单声道的内容
  - 左/右：左右声道的差分或反相成分
  - 明显偏向某一侧的声音可能出现在带标签的角落方向
  - 绿色点会随当前音乐移动
  - 白色线条追踪音乐峰值
- **Correlation Bar**（左侧）
  - 显示左右声道相关性
  - 上方（+1.0）：左右几乎相同，通常听起来偏居中
  - 中间（0.0）：声道关系较弱，常见于宽阔氛围或左右不同内容
  - 下方（-1.0）：左右几乎相反极性，用扬声器播放时可能听起来变弱
- **Balance Bar**（底部）
  - 显示一侧扬声器是否比另一侧更响
  - 中央：两侧音量相同
  - 左/右：某一侧更强
  - 数值以分贝（dB）显示音量差

### 可以看到什么
- **中心声音**：中间附近较强的纵向运动
- **宽阔声音**：显示区域中大范围的活动
- **特殊效果**：角落附近出现的有趣图案
- **扬声器平衡**：底部条指示的方向
- **声道相关性**：左侧相关性条显示的状态

### 参数
- **Window**（10-1000 ms）- 显示中使用的最近音频长度
  - 较低值：更容易看到快速变化
  - 较高值：更容易看到整体模式
  - 默认 100 ms 适合大多数音乐

### 享受音乐
1. **观察不同风格**
   - 古典音乐常显示较平稳、均衡的图案
   - 电子音乐可能形成大胆宽阔的形状
   - 现场录音可能显示自然空间的运动

2. **发现声音特性**
   - 看不同专辑如何使用立体声效果
   - 观察不同歌曲的展开方式
   - 看乐器如何在扬声器之间移动

3. **加深体验**
   - 用不同耳机比较立体声呈现方式
   - 比较喜欢歌曲的新旧录音
   - 观察不同聆听位置带来的显示变化

这些工具的目标是在音乐聆听中加入视觉维度。试着用新的方式观察您喜欢的音乐。
