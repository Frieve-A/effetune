# Basic Audio Plugins

一组用于调整音乐播放基本要素的必备工具集合。这些插件可帮助你控制音量、平衡以及其他基础听音体验相关的方面。

## Plugin List

- [Channel Divider](#channel-divider) - 将音频拆分为跨多个通道的频段
- [DC Offset](#dc-offset) - 帮助修复听起来不平衡的音频
- [Matrix](#matrix) - 以灵活的方式路由并混合音频通道
- [Mute](#mute) - 静音输出
- [Polarity Inversion](#polarity-inversion) - 可改善立体声音乐的听感
- [Stereo Balance](#stereo-balance) - 调整音乐的左右平衡
- [Volume](#volume) - 控制音乐播放的响度

## Channel Divider

该工具可将立体声信号拆分为独立的频段，并将每个频段路由到不同的输出通道。非常适合多通道系统或自定义分频设置。

要使用此效果，需要使用桌面应用程序，在音频设置中将输出通道数量设为4或更多，并在效果总线路由中将通道设置为 "All"。

### 使用场景
- 当使用多通道音频输出（4、6 或 8 通道）时
- 创建基于频率的自定义通道路由
- 用于多放大器或多扬声器的配置

### 参数
- **Band Count** - 要创建的频段数量（2-4 个）
  - 2 个频段：低频/高频 拆分
  - 3 个频段：低频/中频/高频 拆分
  - 4 个频段：低频/低中频/高中频/高频 拆分

- **Crossover Frequencies** - 定义音频在各频段之间的分割点
  - F1：第一个分割点
  - F2：第二个分割点（针对3个及以上频段）
  - F3：第三个分割点（针对4个频段）

- **Slopes** - 控制频段的分离陡度
  - 可选：每倍频程 -12dB 至 -96dB
  - 更陡的斜率可提供更清晰的分离
  - 较低的斜率可带来更自然的过渡

### 技术说明
- 仅处理前两个输入通道
- 输出通道数量必须为2的倍数（4、6 或 8）
- 使用高品质 Linkwitz-Riley 分频滤波器
- 提供可视化频率响应图，便于配置

## DC Offset

该工具可帮助修复听起来不平衡或异常的音频。大多数情况下不常用，但在遇到音频表现异常时非常有用。

### 使用场景
- 当音乐听起来异常不平衡时
- 当某一通道听感过于响亮时
- 当其他效果无法按预期工作时

### 参数
- **Offset** - 调节音频平衡（-1.0 到 +1.0）
  - 0.0：正常设置
  - 如果听感异常，可进行调整
  - 通常细微调整效果最佳

## Matrix

该工具是一个强大的通道路由器，允许在输入和输出通道之间创建自定义信号路径。可完全灵活地连接和混合音频信号。

### 使用场景
- 创建通道间的自定义路由
- 需要以特定方式混合或拆分信号时
- 用于通过通道交互实现创意音效设计

### 功能特性
- 支持最多8个通道的灵活路由矩阵
- 可对任意输入/输出对进行单独连接控制
- 每个连接点可选相位反转
- 可视化矩阵界面，配置直观

### 工作原理
- 每个连接点表示从输入行到输出列的路由
- 已激活的连接允许信号在通道间流动
- 相位反转选项可颠倒信号极性
- 多个输入连接到同一输出时会混合在一起

### 实际应用
- 自定义下混或上混配置
- 隔离或合并特定通道
- 在通道间创建相位关系
- 解决复杂路由需求

## Mute

该工具通过将缓冲区填零来静音所有音频输出。适用于即时静音音频信号。

### 使用场景
- 即时静音，无需渐变
- 在静音段落或暂停时
- 防止产生不需要的噪音输出

## Polarity Inversion

该工具可在特定情况下改善立体声音乐的听感。其原理类似于“翻转”音频波形，以期获得更佳效果。

您还可以在效果器公共设置中限制要处理的通道，从而仅反转特定通道的极性。

### 使用场景
- 当立体声音乐听起来“空洞”或“怪异”时
- 与其他立体声效果结合使用时
- 试图改善立体声成像时

## Stereo Balance

允许您调整音乐在左右扬声器或耳机之间的分布。非常适合修复不均衡的立体声或创建所需的声场定位。

### 听感增强指南
- 完美平衡：
  - 居中位置，实现自然立体声
  - 双耳音量相等
  - 适用于大多数音乐
- 调整平衡：
  - 补偿房间声学环境
  - 适应听力差异
  - 创建理想的声场

### 参数
- **Balance** - 控制左右声道分布（-100% 到 +100%）
  - Center (0%)：两侧音量相等
  - Left (-100%)：左侧更响
  - Right (+100%)：右侧更响

### 可视化
- 易用滑块
- 清晰的数值显示
- 立体声位置可视指示

### 建议用法

1. 一般聆听
   - 保持平衡居中（0%）
   - 如果立体声不平衡，可进行调整
   - 使用细微调整

2. 耳机聆听
   - 为舒适度进行微调
   - 补偿听力差异
   - 创建理想立体声图像

3. 扬声器聆听
   - 根据房间布局进行调整
   - 为聆听位置平衡声道
   - 补偿房间声学环境

## Volume

这是一个简单但必不可少的控制，可让您调整音乐的播放响度。非常适合在不同场景中找到合适的聆听音量。

### 听感增强指南
- 根据不同的聆听场景调整：
  - 工作时的背景音乐
  - 专注聆听会话
  - 深夜安静聆听
- 将音量保持在舒适范围，以避免：
  - 听觉疲劳
  - 声音失真
  - 潜在听力损伤

### 参数
- **Volume** - 控制整体响度（-60dB 至 +24dB）
  - 较低值：更安静的播放
  - 较高值：更响亮的播放
  - 0dB：原始音量级别

请记住：这些基础控制是良好音质的基础。在使用更复杂的效果之前，请先进行这些调整！
