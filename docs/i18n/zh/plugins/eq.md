# Equalizer Plugins

一组让您调整音乐声音不同方面的插件,从深沉的低音到清脆的高音。这些工具通过增强或减弱特定声音元素帮助您个性化聆听体验。

## 插件列表

- [15Band GEQ](#15band-geq) - 具有 15 个精确控制的详细声音调节
- [5Band PEQ](#5band-peq) - 具有灵活控制的专业参数均衡器
- [Loudness Equalizer](#loudness-equalizer) - 低音量聆听时的频率平衡校正
- [Tone Control](#tone-control) - 简单的低音、中音和高音调节
- [Narrow Range](#narrow-range) - 专注于声音的特定部分

## 15Band GEQ

一个具有 15 个独立控制的详细声音调节工具,每个控制影响声音频谱的特定部分。完美适合将音乐精确调整到您喜欢的方式。

### 聆听增强指南
- 低音区域(25Hz-160Hz):
  - 增强低音鼓和深沉低音的力量
  - 调整低音乐器的饱满度
  - 控制震撼房间的次低音
- 低中频(250Hz-630Hz):
  - 调整音乐的温暖度
  - 控制整体声音的饱满度
  - 减少或增强声音的"厚度"
- 高中频(1kHz-2.5kHz):
  - 使人声更清晰和突出
  - 调整主要乐器的突出程度
  - 控制声音的"前向"感
- 高频(4kHz-16kHz):
  - 增强清晰度和细节
  - 控制音乐中的"闪烁"和"空气感"
  - 调整整体亮度

### 参数
- **Band Gains** - 每个频率范围的独立控制(-12dB 到 +12dB)
  - 深沉低音
    - 25Hz:最低低音感
    - 40Hz:深沉低音冲击
    - 63Hz:低音力量
    - 100Hz:低音饱满度
    - 160Hz:上低音
  - 低频声音
    - 250Hz:声音温暖度
    - 400Hz:声音饱满度
    - 630Hz:声音体量
  - 中频声音
    - 1kHz:主声音存在感
    - 1.6kHz:声音清晰度
    - 2.5kHz:声音细节
  - 高频声音
    - 4kHz:声音清脆度
    - 6.3kHz:声音明亮度
    - 10kHz:声音空气感
    - 16kHz:声音闪烁感

### 视觉显示
- 显示声音调整的实时图表
- 易于使用的滑块具有精确控制
- 一键重置为默认设置

## Loudness Equalizer

一个专门的均衡器，根据您的聆听音量自动调整频率平衡。这个插件补偿人耳在低音量时对低频和高频敏感度降低的特性，确保在任何播放音量下都能获得一致且愉悦的聆听体验。

### 聆听增强指南
- 低音量聆听:
  - 增强低频和高频
  - 在安静环境下保持音乐平衡
  - 补偿人耳听觉特性
- 音量相关处理:
  - 低音量时增强效果更明显
  - 随着音量增加逐渐减少处理
  - 高音量时保持自然声音
- 频率平衡:
  - 低频搁架用于增强低音(100-300Hz)
  - 高频搁架用于增强高音(3-6kHz)
  - 频率范围之间平滑过渡

### 参数
- **Average SPL** - 当前聆听音量(60dB 到 85dB)
  - 较低数值：增强效果更强
  - 较高数值：增强效果更弱
  - 代表典型聆听音量
- **低频控制**
  - 频率：低音增强中心(100Hz 到 300Hz)
  - 增益：最大低音提升(0dB 到 15dB)
  - Q值：低音增强形状(0.5 到 1.0)
- **高频控制**
  - 频率：高音增强中心(3kHz 到 6kHz)
  - 增益：最大高音提升(0dB 到 15dB)
  - Q值：高音增强形状(0.5 到 1.0)

### 视觉显示
- 实时频率响应图表
- 交互式参数控制
- 音量相关曲线可视化
- 精确数值显示

## Tone Control

一个简单的三段声音调节器,用于快速和简单的声音个性化。完美适合基本的声音塑造而不会过于技术化。

### 音乐增强指南
- 古典音乐:
  - 轻微提升高音以获得更多弦乐细节
  - 温和提升低音以获得更饱满的管弦乐声音
  - 中频保持中性以获得自然声音
- 摇滚/流行音乐:
  - 适度提升低音以获得更大冲击力
  - 轻微降低中频以获得更清晰的声音
  - 提升高音以获得清脆的镲片和细节
- 爵士音乐:
  - 温暖的低音以获得更饱满的声音
  - 清晰的中频以获得乐器细节
  - 温和的高音以获得镲片闪烁感
- 电子音乐:
  - 强劲的低音以获得深沉冲击
  - 降低中频以获得更干净的声音
  - 增强高音以获得清脆细节

### 参数
- **Bass** - 控制低频声音(-24dB 到 +24dB)
  - 增加以获得更强劲的低音
  - 降低以获得更轻盈、干净的声音
  - 影响音乐的"重量感"
- **Mid** - 控制声音的主体(-24dB 到 +24dB)
  - 增加以获得更突出的人声/乐器
  - 降低以获得更宽阔的声音
  - 影响音乐的"饱满度"
- **Treble** - 控制高频声音(-24dB 到 +24dB)
  - 增加以获得更多闪烁感和细节
  - 降低以获得更平滑、柔和的声音
  - 影响音乐的"明亮度"

### 视觉显示
- 易于阅读的调整图表
- 每个控制的简单滑块
- 快速重置按钮

## 5Band PEQ

基于科学原理的专业级参数均衡器,提供五个完全可配置的频段,具有精确的频率控制。完美适合微妙的声音改善和校正性音频处理。

### 声音增强指南
- 人声和乐器清晰度:
  - 使用 3.2kHz 频段,适中的 Q 值(1.0-2.0)获得自然存在感
  - 应用窄 Q 值(4.0-8.0)削减以消除共振
  - 用 10kHz 高架滤波器添加温和的空气感(+2 到 +4dB)
- 低音质量控制:
  - 用 100Hz 峰值滤波器塑造基频
  - 使用特定频率的窄 Q 值消除房间共振
  - 用低架滤波器创造平滑的低音延伸
- 科学声音调整:
  - 精确定位特定频率
  - 使用分析器识别问题区域
  - 应用测量校正,最小化相位影响

### 技术参数
- **精密工程频段**
  - 频段 1:100Hz(次低音和低音控制)
  - 频段 2:316Hz(低中频定义)
  - 频段 3:1.0kHz(中频存在感)
  - 频段 4:3.2kHz(高中频细节)
  - 频段 5:10kHz(高频延伸)
- **每个频段的专业控制**
  - 中心频率:对数间隔以获得最佳覆盖
  - 增益范围:精确 ±18dB 调整
  - Q 因子:从宽 0.1 到精确 10.0
  - 多种滤波器类型:
    - 峰值:对称频率调整
    - 低通/高通:12dB/倍频程斜率
    - 低架/高架:温和的频谱塑造
    - 带通:聚焦频率隔离

### 技术显示
- 高分辨率频率响应可视化
- 具有精确参数显示的交互式控制点
- 实时传递函数计算
- 校准的频率和增益网格
- 所有参数的准确数值读数

## Narrow Range

一个让您通过过滤掉不需要的频率来专注于音乐特定部分的工具。用于创建特殊声音效果或去除不需要的声音。

### 聆听增强指南
- 创建独特的声音效果:
  - "电话声音"效果
  - "老式收音机"声音
  - "水下"效果
- 专注于特定乐器:
  - 隔离低音频率
  - 专注于人声范围
  - 突出特定乐器
- 去除不需要的声音:
  - 减少低频隆隆声
  - 削减过多的高频嘶嘶声
  - 专注于音乐最重要的部分

### 参数
- **HPF Frequency** - 控制低频开始减少的位置（20Hz 到 1000Hz）
  - 较高的数值：减少更多低音
  - 较低的数值：保留更多低音
  - 从较低的数值开始，根据个人喜好调整
- **HPF Slope** - 控制低频减少的速度（0 到 -48 dB/倍频程）
  - 0dB：不减少（关闭）
  - -6dB 到 -48dB：以6dB的步进逐渐增强减少效果
- **LPF Frequency** - 控制高频开始减少的位置（200Hz 到 20000Hz）
  - 较低的数值：减少更多高音
  - 较高的数值：保留更多高音
  - 从较高的数值开始，根据需要逐步降低
- **LPF Slope** - 控制高频减少的速度（0 到 -48 dB/倍频程）
  - 0dB：不减少（关闭）
  - -6dB 到 -48dB：以6dB的步进逐渐增强减少效果

### 视觉显示
- 清晰显示频率响应的图表
- 易于调整的频率控制
- 简单的斜率选择按钮
