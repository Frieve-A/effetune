# 延迟插件

这些工具用于调整音频信号的时序，或给音乐加入延迟和回声效果。

## 插件列表

- [Delay](#delay) - 创建可控制时序、音色和 Ping-Pong 立体声运动的回声
- [Time Alignment](#time-alignment) - 为扬声器与聆听位置校准微调播放时序

## Delay

这个效果会给音频加入清晰的回声。您可以控制回声出现的速度、持续时间、音色，以及它们在立体声中的展开方式。

### 听觉体验指南
- **轻微深度与空间感：**
  - 在不冲淡声音的情况下加入轻柔的空间感
  - 可以让人声或主奏乐器感觉稍大、更有存在感
  - 使用较短的延迟时间和较低的 Feedback/Mix
- **节奏增强：**
  - 创建与音乐速度同步的回声（手动调节）
  - 增加律动和能量，尤其适合电子音乐、鼓或吉他
  - 尝试不同的 Delay Size（例如凭听感匹配八分音符或四分音符）
- **拍击式回声：**
  - 一种非常短、单次的回声，常用于摇滚和乡村音乐中的人声或吉他
  - 增加带有打击感的加倍效果
  - 使用很短的延迟时间（30-120ms）、零 Feedback 和中等 Mix
- **创意立体声扩展：**
  - 使用 Ping-Pong 控制，回声可以在左右扬声器之间来回跳动
  - 创建更宽、更有参与感的立体声图像
  - 可以让声音更有动态和趣味

### 参数

- **Pre-Delay (ms)** - 在信号进入回声延迟前额外增加的时间（0 到 100 ms）。第一次回声会在 Pre-Delay + Delay Size 之后听到。
  - 较低值（0-20ms）：回声图案几乎立即开始
  - 较高值（20-100ms）：在回声图案开始前加入明显间隔，使其与原声分开
- **Delay Size (ms)** - 每次回声之间的时间（1 到 5000 ms）
  - 短（1-100ms）：增厚或 slapback 效果
  - 中（100-600ms）：标准回声，适合节奏增强
  - 长（600ms 以上）：宽阔、氛围化的重复
- **Damping (%)** - 控制每次重复后高频和低频的衰减量（0 到 100%）
  - 0%：回声保持明亮、饱满
  - 50%：自然、均衡地衰减
  - 100%：回声很快变暗、变薄
  - 与 High Damp / Low Damp 配合塑造回声音色
- **High Damp (Hz)** - 设置回声开始失去明亮度的频率以上区域（20 到 20000 Hz）
  - 较低值（如 2000Hz）：回声很快变暗
  - 较高值（如 10000Hz）：回声保持明亮更久
  - 与 Damping 一起调整回声音色
- **Low Damp (Hz)** - 设置回声开始失去厚度的频率以下区域（20 到 20000 Hz）
  - 较低值（如 50Hz）：回声保留更多低频
  - 较高值（如 500Hz）：回声更快变薄
  - 与 Damping 一起调整回声音色
  - 为了更可预测的音色塑造，请保持 Low Damp 低于 High Damp。若数值交叉，处理器会在内部重新排序。
- **Feedback (%)** - 控制能听到多少次回声，或回声持续多久（0 到 99%）
  - 0%：只听到一次回声
  - 10-40%：有几次明显重复
  - 40-70%：持续时间更长的回声
  - 70-99%：非常长的回声尾音，接近自激振荡（请谨慎使用！）
- **Ping-Pong (%)** - 控制回声如何在立体声声道之间来回跳动（0 到 100%）（仅影响立体声播放）
  - 0%：标准延迟，左声道输入在左声道回响，右声道输入在右声道回响
  - 50%：单声道反馈，回声位于扬声器之间的中央
  - 100%：完整 Ping-Pong，回声在左右扬声器之间交替
  - 中间值会产生不同程度的立体声展开
- **Mix (%)** - 控制回声与原声的混合比例（0 到 100%）
  - 0%：无效果
  - 5-15%：轻微深度或节奏感
  - 15-30%：清楚可闻的回声（良好起点）
  - 30% 以上：更强、更明显的效果。默认值为 16%

### 推荐的听觉增强设置

1. **轻微人声/乐器深度**
   - Delay Size: 80-150ms
   - Feedback: 0-15%
   - Mix: 8-16%
   - Ping-Pong: 0%（或尝试 20-40% 获得轻微宽度）
   - Damping: 40-60%

2. **节奏增强（电子/流行）**
   - Delay Size: 按听感匹配节奏（例如 120-500ms）
   - Feedback: 20-40%
   - Mix: 15-25%
   - Ping-Pong: 0% 或 100%
   - Damping: 按口味调整（较低值让重复更明亮）

3. **经典摇滚 Slapback（吉他/人声）**
   - Delay Size: 50-120ms
   - Feedback: 0%
   - Mix: 15-30%
   - Ping-Pong: 0%
   - Damping: 20-40%

4. **宽阔立体声回声（氛围/铺底音色）**
   - Delay Size: 300-800ms
   - Feedback: 40-60%
   - Mix: 20-35%
   - Ping-Pong: 70-100%
   - Damping: 50-70%（让尾音更平滑）

### 快速入门指南

1. **设置时序**
   - 先用 `Delay Size` 设置主要回声节奏
   - 调整 `Feedback` 控制回声次数
   - 使用 `Pre-Delay` 为回声图案开始前增加额外间隔
2. **调整音色**
   - 配合使用 `Damping`、`High Damp` 和 `Low Damp`，塑造回声在衰减时的音色。可从 Damping 约 50% 开始调节。
3. **设置立体声位置（可选）**
   - 使用 `Ping-Pong` 调整回声宽度
4. **混合效果**
   - 用 `Mix` 平衡回声和原始音乐的音量。先从较低值（约 16%）开始，再增加到听感合适为止

## Time Alignment

以很小的时间量调整播放时序。它适合补偿扬声器距离差，或微调声音抵达聆听位置的方式。

### 何时使用
- 补偿扬声器与聆听位置之间的小距离差
- 微调经过此插件路由的通道时序
- 检查小延迟是否能让立体声声像更稳定、更自然

### 参数
- **Delay** - 控制应用到经过此插件路由通道的延迟时间（0 到 100 ms）
  - 0 ms：无延迟
  - 较小值：适合补偿扬声器之间极小的到达时间差
  - 较高值：产生更明显的时序偏移

### 推荐用途

1. 扬声器距离补偿
   - 当某个扬声器或通道更早到达聆听位置时，加入小幅延迟
   - 播放居中人声或其他聚焦声音时，以小步进调整

2. 聆听位置微调
   - 先尝试非常小的数值
   - 当中心声像稳定且声音仍然自然时停止调整

请记住：目标是提升聆听乐趣。尝试不同设置，找到能为喜欢的音乐增加趣味和深度、又不过度抢戏的声音。
