# 其他音频工具

这些专用工具和发生器用于补充主要效果类别。它们适合在聆听前或聆听过程中检查扬声器、耳机、通道平衡和播放行为。

## 插件列表

- [Oscillator](#oscillator) - 用于检查扬声器/耳机的测试音和噪声发生器

## Oscillator

用于检查聆听系统的测试音和噪声发生器。请以低音量使用，用来确认扬声器/耳机输出、左右位置、电平平衡、异响、共振或简单的频率响应问题。

生成的测试音或噪声会混入当前音频路径，而不是替代输入音频。启用前请先降低 Volume，尤其是在音乐已经播放时。

### 功能
- 多种波形类型：
  - Sine：简单音调检查
  - Square：丰富奇次谐波
  - Triangle：较柔和的谐波
  - Sawtooth：明亮音色
  - White Noise：宽频扬声器/耳机检查
  - Pink Noise：更平滑、更自然的噪声平衡
- Pulsed 模式用于间歇测试音或噪声脉冲

### 参数
- **Frequency (Hz)** - 控制生成音调的音高（20 Hz 到 96 kHz）
  - 低频：深沉低音测试
  - 中频：音乐常用范围
  - 高频：请谨慎使用，并保持安全聆听音量
  - 仅适用于 sine、square、triangle 和 sawtooth；white/pink noise 下禁用
  - 可用的高频输出取决于当前音频采样率；超过可用 Nyquist 频率的音调会被静音
- **Volume (dB)** - 调整输出电平（-96 dB 到 0 dB）
  - 从低音量开始，再慢慢提高
  - 较高值可能很响或令人疲劳
- **Panning (L/R)** - 控制立体声位置
  - Center：左右相同
  - Left/Right：检查通道路由和平衡
- **Waveform Type** - 选择信号类型
  - Sine：干净参考音
  - Square：富含奇次谐波
  - Triangle：较柔和的谐波内容
  - Sawtooth：完整谐波系列
  - White Noise：每 Hz 能量相同；Frequency 不影响它
  - Pink Noise：每倍频程能量相同；Frequency 不影响它
- **Mode** - 控制信号生成模式
  - Continuous：连续信号
  - Pulsed：带可控时序的间歇信号
- **Interval (ms)** - Pulsed 模式中脉冲之间的时间（100-2000 ms，步进 10 ms）
  - 较短间隔：快速脉冲序列
  - 较长间隔：相隔更远的脉冲
  - 仅 Mode 为 Pulsed 时有效
- **Width (ms)** - Pulsed 模式中的脉冲淡入/淡出时间（2-100 ms，最大为 Interval 的一半，步进 1 ms）
  - 控制每个脉冲的淡入/淡出时间
  - 生成的脉冲约为 Width 的两倍时长，没有稳定保持段
  - 较短 Width：脉冲边缘更锐利
  - 较长 Width：脉冲过渡更平滑
  - 仅 Mode 为 Pulsed 时有效

### 使用示例

1. 扬声器或耳机检查
   - 检查基本频率再现
     * 使用 sine 从低频扫到高频
     * 记录哪些位置听不见或出现失真
   - 聆听异响、嗡声或刺耳共振
     * 先使用较低 Volume
     * 一次只测试一个频率范围
   - 比较左右输出
     * 将声像完全移到左侧和右侧
     * 确认声音来自预期的扬声器或耳机单元

2. 通道与电平平衡
   - 检查立体声位置
     * 使用居中的 sine 或 pink noise
     * 确认声音位于中央
   - 比较左右音量
     * 以相同 Volume 分别移到左右两侧
     * 如果一侧明显更响，请调整播放系统
   - 检查插件链
     * 将 Oscillator 放在其他效果前或后，听效果链如何处理简单信号

3. 房间或桌面共振快速检查
   - 找出明显低频堆积或异响
     * 以安全音量播放低频 sine
     * 在聆听位置附近移动，记录强峰或凹陷
   - 检查容易振动的物品
     * 缓慢扫过低频和低中频
     * 如果出现强烈异响，立即降低 Volume

4. 噪声平衡检查
   - 使用 pink noise 作为宽频、稳定参考
     * 聆听明显的左右或音色不平衡
     * 保持舒适音量，避免长时间大音量噪声播放
   - 只有需要更亮的宽频信号时才使用 white noise

5. 脉冲信号检查
   - 使用 pulsed 模式让短促声音更容易辨认
     * 较长 Interval 让每个脉冲更容易分开听
     * 较短 Width 产生更尖锐的起止
     * 比较不同音量下的表现

请记住：Oscillator 是测试信号发生器。请从较低 Volume 开始逐步提高，并避免可能损伤设备或造成听觉疲劳的高音量/高频音调。
