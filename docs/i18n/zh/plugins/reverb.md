# Reverb Plugins

一组为音乐添加空间感和氛围的插件。这些效果可以使您的音乐听起来像是在不同的环境中播放,从私密房间到宏伟音乐厅,通过自然的环境感和深度增强您的聆听体验。

## 插件列表

- [FDN Reverb](#fdn-reverb) - 具有高级扩散矩阵的反馈延迟网络混响
- [RS Reverb](#rs-reverb) - 创造自然的房间氛围和空间感

## FDN Reverb

基于反馈延迟网络(FDN)架构的复杂混响效果,使用Hadamard扩散矩阵。这创造了丰富、复杂的混响,具有出色的密度和自然的衰减特性,非常适合通过沉浸式空间效果增强您的音乐聆听体验。

### 聆听体验指南
- 自然房间感:
  - 创造在真实声学空间中聆听的感觉
  - 为音乐添加深度和立体感
  - 使立体声录音感觉更宽阔、更生动
- 氛围增强:
  - 将平坦的录音转化为沉浸式体验
  - 为音乐音符添加美丽的尾音和延音
  - 创造身处表演空间的感觉
- 可定制氛围:
  - 从私密房间到宏伟音乐厅可调节
  - 对空间特性和色彩的精细控制
  - 温和的调制添加自然运动和生命力

### 参数
- **Reverb Time** - 混响效果持续时间(0.20到10.00秒)
  - 短(0.2-1.0秒): 快速、受控的衰减以保持清晰度
  - 中等(1.0-3.0秒): 自然的房间般混响
  - 长(3.0-10.0秒): 宽阔、有氛围的尾音
- **Density** - 复杂度的回声路径数量(4到8线)
  - 4线: 更简单、更清晰的个别回声
  - 6线: 复杂度和清晰度的良好平衡
  - 8线: 最大的平滑度和密度
- **Pre Delay** - 混响开始前的初始静音(0.0到100.0毫秒)
  - 0-20ms: 即时混响,亲密感觉
  - 20-50ms: 自然的房间距离感
  - 50-100ms: 创造更大空间的印象
- **Base Delay** - 混响网络的基础时间(10.0到60.0毫秒)
  - 较低值: 更紧凑、更集中的混响特性
  - 较高值: 更宽阔、开放的音质
  - 影响基本时间关系
- **Delay Spread** - 线间延迟时间的变化程度(0.0到25.0毫秒)
  - 实现: 每条延迟线使用幂曲线(0.8指数)逐渐变长
  - 0.0ms: 所有线具有相同的基础时间(更规律的模式)
  - 较高值: 更自然、不规则的反射模式
  - 添加真实声学空间中的现实变化
- **HF Damp** - 高频随时间衰减的程度(0.0到12.0 dB/秒)
  - 0.0: 无阻尼,整个衰减过程保持明亮声音
  - 3.0-6.0: 自然空气吸收模拟
  - 12.0: 温暖、柔和特性的重度阻尼
- **Low Cut** - 从混响中移除低频(20到500 Hz)
  - 20-50Hz: 混响中的完整低音响应
  - 100-200Hz: 避免混浊的受控低音
  - 300-500Hz: 紧实、清晰的低端
- **Mod Depth** - 合唱效果的音调调制量(0.0到10.0音分)
  - 0.0: 无调制,纯静态混响
  - 2.0-5.0: 添加生命力和真实感的微妙运动
  - 10.0: 明显的合唱般效果
- **Mod Rate** - 调制的速度(0.10到5.00 Hz)
  - 0.1-0.5Hz: 非常缓慢、温和的运动
  - 1.0-2.0Hz: 自然听起来的变化
  - 3.0-5.0Hz: 快速、更明显的调制
- **Diffusion** - Hadamard矩阵混合信号的程度(0到100%)
  - 实现: 控制应用的矩阵混合量
  - 0%: 最小混合,更清晰的回声模式
  - 50%: 自然声音的平衡扩散
  - 100%: 最平滑密度的最大混合
- **Wet Mix** - 添加到声音中的混响量(0到100%)
  - 10-30%: 微妙的空间增强
  - 30-60%: 明显的混响存在
  - 60-100%: 主导性混响效果
- **Dry Mix** - 保留的原始信号量(0到100%)
  - 正常聆听时通常保持在100%
  - 可以为特殊氛围效果而降低
- **Stereo Width** - 混响在立体声中的扩展宽度(0到200%)
  - 实现: 0% = 单声道,100% = 正常立体声,200% = 夸张宽度
  - 0%: 混响出现在中央(单声道)
  - 100%: 自然立体声扩展
  - 200%: 超宽立体声声像

### 不同聆听体验的推荐设置

1. 古典音乐增强
   - Reverb Time: 2.5-3.5秒
   - Density: 8线
   - Pre Delay: 30-50ms
   - HF Damp: 4.0-6.0
   - 完美适合: 管弦乐录音、室内乐

2. 爵士俱乐部氛围
   - Reverb Time: 1.2-1.8秒
   - Density: 6线
   - Pre Delay: 15-25ms
   - HF Damp: 2.0-4.0
   - 完美适合: 原声爵士、亲密表演

3. 流行/摇滚增强
   - Reverb Time: 1.0-2.0秒
   - Density: 6-7线
   - Pre Delay: 10-30ms
   - Wet Mix: 20-40%
   - 完美适合: 现代录音、添加空间

4. 环境声景
   - Reverb Time: 4.0-8.0秒
   - Density: 8线
   - Mod Depth: 3.0-6.0
   - Wet Mix: 60-80%
   - 完美适合: 氛围音乐、放松

### 快速入门指南

1. 设置空间特性
   - 从Reverb Time开始匹配期望的空间大小
   - 将Density设置为6-8以获得平滑、自然的声音
   - 调整Pre Delay以控制距离感知

2. 塑造音色
   - 使用HF Damp模拟自然空气吸收
   - 设置Low Cut防止低音堆积
   - 调整Diffusion以获得平滑度(尝试70-100%)

3. 添加自然运动
   - 将Mod Depth设置为2-4音分以获得微妙的生命力
   - 使用约0.3-1.0 Hz的Mod Rate获得温和变化
   - 调整Stereo Width以获得空间印象

4. 平衡效果
   - 从30%的Wet Mix开始
   - 正常聆听时保持Dry Mix在100%
   - 根据音乐和偏好进行微调

FDN Reverb通过为任何录音添加美丽、自然听起来的混响和现实的声学空间来改变您的聆听体验。对于想要用美丽、自然声音的混响增强喜爱曲目的音乐爱好者来说是完美的!

## RS Reverb

一个可以将您的音乐带入不同空间的效果器,从舒适的房间到宏伟的大厅。它添加自然的回声和反射,使您的音乐感觉更加立体和沉浸。

### 聆听体验指南
- 私密空间:
  - 使音乐感觉像在温暖、舒适的房间中
  - 完美适合近距离、私密的聆听
  - 添加细微的深度而不失清晰度
- 音乐厅体验:
  - 重现现场演出的宏伟感
  - 为古典和管弦乐音乐添加宏伟的空间感
  - 创造沉浸式的音乐会体验
- 氛围增强:
  - 添加梦幻、空灵的品质
  - 完美适合环境和氛围音乐
  - 创造引人入胜的声景

### 参数
- **Pre-Delay** - 控制空间效果开始的速度(0 到 50 ms)
  - 较低值:即时、私密的感觉
  - 较高值:更多的距离感
  - 从 10ms 开始获得自然声音
- **Room Size** - 设置空间感觉的大小(2.0 到 50.0 m)
  - 小(2-5m):舒适的房间感
  - 中(5-15m):现场房间氛围
  - 大(15-50m):音乐厅的宏伟感
- **Reverb Time** - 回声持续的时间(0.1 到 10.0 s)
  - 短(0.1-1.0s):清晰、聚焦的声音
  - 中(1.0-3.0s):自然的房间声音
  - 长(3.0-10.0s):宽阔、富有氛围
- **Density** - 空间感觉的丰富程度(4 到 8)
  - 较低值:更清晰的回声
  - 较高值:更平滑的氛围
  - 从 6 开始获得自然声音
- **Diffusion** - 声音扩散的方式(0.2 到 0.8)
  - 较低值:更清晰的回声
  - 较高值:更平滑的混合
  - 尝试 0.5 获得平衡的声音
- **Damping** - 回声衰减的方式(0 到 100%)
  - 较低值:更明亮、开放的声音
  - 较高值:更温暖、私密
  - 从约 40% 开始获得自然感觉
- **High Damp** - 控制空间的明亮度(1000 到 20000 Hz)
  - 较低值:更暗、更温暖的空间
  - 较高值:更明亮、更开放
  - 从约 8000Hz 开始获得自然声音
- **Low Damp** - 控制空间的饱满度(20 到 500 Hz)
  - 较低值:更饱满、丰富的声音
  - 较高值:更清晰、受控
  - 从约 100Hz 开始获得平衡的低音
- **Mix** - 平衡效果与原始声音(0 到 100%)
  - 10-30%:细微增强
  - 30-50%:明显的空间感
  - 50-100%:戏剧性效果

### 不同音乐风格的推荐设置

1. 音乐厅中的古典音乐
   - Room Size: 30-40m
   - Reverb Time: 2.0-2.5s
   - Mix: 30-40%
   - 完美适合:管弦乐作品、钢琴协奏曲

2. 私密爵士俱乐部
   - Room Size: 8-12m
   - Reverb Time: 1.0-1.5s
   - Mix: 20-30%
   - 完美适合:爵士乐、原声表演

3. 现代流行/摇滚
   - Room Size: 15-20m
   - Reverb Time: 1.2-1.8s
   - Mix: 15-25%
   - 完美适合:当代音乐

4. 环境/电子音乐
   - Room Size: 25-40m
   - Reverb Time: 3.0-6.0s
   - Mix: 40-60%
   - 完美适合:氛围电子音乐

### 快速入门指南

1. 选择您的空间
   - 从 Room Size 开始设置基本空间
   - 调整 Reverb Time 获得理想氛围
   - 微调 Mix 获得适当平衡

2. 塑造声音
   - 使用 Damping 控制温暖度
   - 调整 High/Low Damp 获得音色
   - 设置 Density 和 Diffusion 获得纹理

3. 微调效果
   - 添加 Pre-Delay 获得更多深度
   - 调整 Mix 获得最终平衡
   - 相信您的耳朵并根据喜好调整

记住:目标是用自然的空间感和氛围增强您的音乐。从细微设置开始调整,直到找到完美平衡的聆听体验!
