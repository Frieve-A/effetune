# Reverb Plugins

음악에 공간감과 분위기를 더하는 플러그인 모음입니다. 이러한 이펙트들은 음악이 아늑한 방에서부터 웅장한 콘서트홀까지 다양한 환경에서 연주되는 것처럼 들리게 만들어, 자연스러운 공간감과 깊이감으로 청취 경험을 향상시킵니다.

## Plugin List

- [Dattorro Plate Reverb](#dattorro-plate-reverb) - Dattorro 알고리즘 기반의 클래식 플레이트 리버브
- [FDN Reverb](#fdn-reverb) - 고급 확산 매트릭스를 사용한 피드백 딜레이 네트워크 리버브
- [RS Reverb](#rs-reverb) - 자연스러운 공간 분위기와 공간감 생성

## Dattorro Plate Reverb

Jon Dattorro의 1997년 논문 "Effect Design, Part 1: Reverberator and Other Filters"를 기반으로 한 클래식 플레이트 리버브 구현입니다. 이 알고리즘은 풍부하고 부드러운 음질로 유명하며 디지털 리버브 설계의 참조 표준이 되었습니다. 음악에 풍부하고 반짝이는 앰비언스를 추가하는 데 완벽합니다.

### Listening Experience Guide
- 풍부한 플레이트 사운드:
  - 클래식 스튜디오 플레이트 리버브 특성
  - 금속적인 아티팩트 없는 부드럽고 밀도 높은 리버브 테일
  - 플레이트 리버브 특유의 아름다운 쉬머와 따뜻함
- 다양한 앰비언스:
  - 미묘한 공간 향상부터 광활한 홀까지
  - 모든 음악 장르와 아름답게 어울림
  - 녹음에 프로페셔널한 마감 추가
- 자연스러운 움직임:
  - 모듈레이션이 리버브에 유기적인 생명력 추가
  - 정적이고 인공적인 테일 방지
  - 음악 주위에 숨쉬는 듯한 살아있는 공간 생성

### Parameters
- **Pre Delay** - 리버브 시작 전 초기 무음(0.0~100.0ms)
  - 0-10ms: 즉각적인 리버브, 친밀한 느낌
  - 10-30ms: 자연스러운 공간감
  - 30-100ms: 더 큰 공간의 인상 생성
- **Bandwidth** - 입력 신호 필터링(0.0~1.0)
  - 낮은 값: 더 어둡고 따뜻한 입력 톤
  - 높은 값(1.0에 가까움): 더 밝고 풀 주파수 입력
  - 기본값 0.9995: Dattorro가 제안한 최적값
- **Input Diff 1** - 1단계 입력 확산(0.0~1.0)
  - 입력 신호의 초기 스미어링 제어
  - 기본값 0.75: Dattorro 논문 권장값
  - 높은 값: 더 확산된, 부드러운 초기 반사
- **Input Diff 2** - 2단계 입력 확산(0.0~1.0)
  - 입력 신호를 추가로 확산
  - 기본값 0.625: Dattorro 논문 권장값
  - Input Diff 1과 함께 복잡한 확산 생성
- **Decay** - 리버브 테일의 지속 시간(0.0~1.0)
  - 낮음(0.1-0.3): 짧고 제어된 감쇠
  - 중간(0.4-0.6): 자연스러운 공간 같은 감쇠
  - 높음(0.7-1.0): 길고 확장적인 테일
- **Decay Diff 1** - 탱크 내 감쇠 확산(0.0~1.0)
  - 감쇠 단계의 밀도 제어
  - 기본값 0.70: Dattorro 논문 권장값
  - 리버브 테일의 부드러움에 영향
- **Damping** - 시간에 따른 고주파 흡수(0.0~1.0)
  - 0.0: 댐핑 없음, 전체적으로 밝은 리버브
  - 0.0005(기본값): 매우 미묘하고 자연스러운 댐핑
  - 높은 값: 더 어둡고 따뜻한 감쇠
- **Mod Depth** - 딜레이 모듈레이션 양(0.0~16.0샘플)
  - 0.0: 모듈레이션 없음, 정적 리버브
  - 1.0-4.0: 미묘한 움직임, 생명력 추가
  - 8.0-16.0: 더 눈에 띄는 코러스 같은 이펙트
- **Mod Rate** - 모듈레이션 속도(0.0~10.0Hz)
  - 0.5-1.5Hz: 느리고 부드러운 움직임
  - 2.0-4.0Hz: 더 활발한 모듈레이션
  - 높은 값: 빠르고 반짝이는 이펙트
- **Wet Mix** - 추가되는 리버브 양(0~100%)
  - 10-30%: 미묘한 향상
  - 30-50%: 눈에 띄는 존재감
  - 50-100%: 지배적인 리버브 이펙트
- **Dry Mix** - 원래 신호의 양(0~100%)
  - 일반적인 청취 시 보통 100%로 유지
  - 특수 효과나 앰비언트 워시를 위해 줄임

### 다양한 음악 스타일을 위한 권장 설정

1. 클래식 피아노
   - Decay: 0.6-0.7
   - Damping: 0.001
   - Mod Depth: 1.0
   - Wet Mix: 25-35%
   - 완벽한 용도: 솔로 피아노, 실내악

2. 보컬과 어쿠스틱
   - Decay: 0.4-0.5
   - Damping: 0.002
   - Pre Delay: 15-25ms
   - Wet Mix: 20-30%
   - 완벽한 용도: 보컬, 어쿠스틱 기타

3. 앰비언트와 분위기
   - Decay: 0.8-0.95
   - Mod Depth: 4.0-8.0
   - Mod Rate: 0.5-1.0Hz
   - Wet Mix: 50-70%
   - 완벽한 용도: 앰비언트, 일렉트로닉, 사운드스케이프

4. 범용 향상
   - Decay: 0.5
   - Damping: 0.0005
   - Mod Depth: 1.0
   - Wet Mix: 20-30%
   - 완벽한 용도: 올라운드 사용, 미묘한 마감

### Quick Start Guide

1. 기본 특성 설정
   - Decay로 리버브 길이 제어
   - Pre Delay로 인지되는 거리 조정
   - Wet Mix로 원하는 리버브 존재감 설정

2. 톤 형성
   - Bandwidth로 입력 밝기 제어
   - Damping으로 고주파 감쇠 조정
   - 밀도를 위해 확산 파라미터 미세 조정

3. 움직임 추가
   - 미묘한 변화를 위해 Mod Depth 설정(1.0 시도)
   - 속도를 위해 Mod Rate 조정(1.0Hz 시도)
   - 이 파라미터들이 리버브에 생명력 추가

4. 최종 밸런스
   - 취향에 맞게 Wet/Dry 믹스 조정
   - 최종 설정은 귀를 믿을 것
   - 기본값이 훌륭한 시작점

Dattorro Plate Reverb는 클래식하고 프로페셔널한 품질의 리버브를 청취 경험에 제공합니다. 부드럽고 풍부한 특성으로 아름답고 자연스럽게 들리는 앰비언스로 모든 녹음을 향상시키는 데 완벽합니다!

## FDN Reverb

Hadamard 확산 매트릭스를 사용한 피드백 딜레이 네트워크(FDN) 아키텍처를 기반으로 한 정교한 리버브 이펙트입니다. 뛰어난 밀도와 자연스러운 감쇠 특성을 가진 풍부하고 복잡한 잔향을 만들어내며, 몰입감 있는 공간 이펙트로 음악 청취 경험을 향상시키는 데 완벽합니다.

### Listening Experience Guide
- 자연스러운 공간감:
  - 실제 음향 공간에서 듣는 느낌을 만들어냄
  - 음악에 깊이와 입체감을 추가
  - 스테레오 녹음을 더 넓고 생생하게 느끼게 만듦
- 분위기 향상:
  - 평면적인 녹음을 몰입감 있는 경험으로 변환
  - 음악적 음표에 아름다운 테일과 서스테인을 추가
  - 연주 공간에 있는 듯한 느낌을 만들어냄
- 사용자 정의 가능한 분위기:
  - 친밀한 방에서 웅장한 콘서트홀까지 조절 가능
  - 공간의 특성과 색채에 대한 세밀한 제어
  - 부드러운 모듈레이션이 자연스러운 움직임과 생명력을 추가

### Parameters
- **Reverb Time** - 리버브 이펙트가 지속되는 시간(0.20~10.00초)
  - 짧음(0.2-1.0초): 명료함을 위한 빠르고 제어된 감쇠
  - 보통(1.0-3.0초): 자연스러운 방과 같은 잔향
  - 김(3.0-10.0초): 확장적이고 분위기 있는 테일
- **Density** - 복잡성을 위한 에코 경로 수(4~8라인)
  - 4라인: 더 간단하고 명확한 개별 에코
  - 6라인: 복잡성과 명료성의 좋은 균형
  - 8라인: 최대 부드러움과 밀도
- **Pre Delay** - 리버브 시작 전 초기 무음(0.0~100.0ms)
  - 0-20ms: 즉각적인 리버브, 친밀한 느낌
  - 20-50ms: 자연스러운 방 거리감
  - 50-100ms: 더 큰 공간의 인상 생성
- **Base Delay** - 리버브 네트워크의 기초 타이밍(10.0~60.0ms)
  - 낮은 값: 더 타이트하고 집중된 리버브 특성
  - 높은 값: 더 넓고 개방적인 음질
  - 기본적인 타이밍 관계에 영향
- **Delay Spread** - 라인 간 딜레이 시간의 변화량(0.0~25.0ms)
  - 구현: 각 딜레이 라인은 파워 커브(0.8 지수)를 사용하여 점진적으로 길어짐
  - 0.0ms: 모든 라인이 같은 기본 타이밍(더 규칙적인 패턴)
  - 높은 값: 더 자연스럽고 불규칙한 반사 패턴
  - 실제 음향 공간에서 발견되는 현실적인 변화 추가
- **HF Damp** - 시간이 지남에 따라 고주파수가 사라지는 정도(0.0~12.0 dB/s)
  - 0.0: 댐핑 없음, 감쇠 전체에서 밝은 사운드
  - 3.0-6.0: 자연스러운 공기 흡수 시뮬레이션
  - 12.0: 따뜻하고 부드러운 특성을 위한 헤비 댐핑
- **Low Cut** - 리버브에서 저주파수 제거(20~500 Hz)
  - 20-50Hz: 리버브에서 완전한 베이스 응답
  - 100-200Hz: 흐림을 피하기 위한 제어된 베이스
  - 300-500Hz: 타이트하고 명확한 저음역
- **Mod Depth** - 코러스 이펙트를 위한 피치 모듈레이션 양(0.0~10.0센트)
  - 0.0: 모듈레이션 없음, 순수한 정적 리버브
  - 2.0-5.0: 생명력과 리얼리즘을 추가하는 미묘한 움직임
  - 10.0: 눈에 띄는 코러스 같은 이펙트
- **Mod Rate** - 모듈레이션의 속도(0.10~5.00 Hz)
  - 0.1-0.5Hz: 매우 느리고 부드러운 움직임
  - 1.0-2.0Hz: 자연스럽게 들리는 변화
  - 3.0-5.0Hz: 빠르고 더 명백한 모듈레이션
- **Diffusion** - Hadamard 매트릭스가 신호를 믹스하는 정도(0~100%)
  - 구현: 적용되는 매트릭스 믹싱의 양을 제어
  - 0%: 최소 믹싱, 더 뚜렷한 에코 패턴
  - 50%: 자연스러운 사운드를 위한 균형 잡힌 확산
  - 100%: 가장 부드러운 밀도를 위한 최대 믹싱
- **Wet Mix** - 사운드에 추가되는 리버브 양(0~100%)
  - 10-30%: 미묘한 공간 향상
  - 30-60%: 눈에 띄는 리버브 존재감
  - 60-100%: 지배적인 리버브 이펙트
- **Dry Mix** - 보존되는 원래 신호의 양(0~100%)
  - 일반적인 청취 시 보통 100%로 유지
  - 특별한 분위기 이펙트를 위해 줄일 수 있음
- **Stereo Width** - 리버브가 스테레오로 퍼지는 폭(0~200%)
  - 구현: 0% = 모노, 100% = 일반 스테레오, 200% = 과장된 폭
  - 0%: 리버브가 중앙에 나타남(모노)
  - 100%: 자연스러운 스테레오 확산
  - 200%: 초광각 스테레오 이미지

### 다양한 청취 경험을 위한 권장 설정

1. 클래식 음악 향상
   - Reverb Time: 2.5-3.5초
   - Density: 8라인
   - Pre Delay: 30-50ms
   - HF Damp: 4.0-6.0
   - 완벽한 용도: 오케스트라 녹음, 실내악

2. 재즈 클럽 분위기
   - Reverb Time: 1.2-1.8초
   - Density: 6라인
   - Pre Delay: 15-25ms
   - HF Damp: 2.0-4.0
   - 완벽한 용도: 어쿠스틱 재즈, 친밀한 공연

3. 팝/록 향상
   - Reverb Time: 1.0-2.0초
   - Density: 6-7라인
   - Pre Delay: 10-30ms
   - Wet Mix: 20-40%
   - 완벽한 용도: 현대 녹음, 공간 추가

4. 앰비언트 사운드스케이프
   - Reverb Time: 4.0-8.0초
   - Density: 8라인
   - Mod Depth: 3.0-6.0
   - Wet Mix: 60-80%
   - 완벽한 용도: 분위기 있는 음악, 휴식

### Quick Start Guide

1. 공간 특성 설정
   - 원하는 공간 크기에 맞춰 Reverb Time부터 시작
   - 부드럽고 자연스러운 사운드를 위해 Density를 6-8로 설정
   - 거리 인식을 제어하기 위해 Pre Delay 조정

2. 톤 형성
   - 자연스러운 공기 흡수를 시뮬레이션하기 위해 HF Damp 사용
   - 베이스 빌드업을 방지하기 위해 Low Cut 설정
   - 부드러움을 위해 Diffusion 조정(70-100% 시도)

3. 자연스러운 움직임 추가
   - 미묘한 생명력을 위해 Mod Depth를 2-4센트로 설정
   - 부드러운 변화를 위해 Mod Rate를 0.3-1.0 Hz 정도로 사용
   - 공간적 인상을 위해 Stereo Width 조정

4. 이펙트 균형
   - 30% Wet Mix에서 시작
   - 일반적인 청취 시 Dry Mix를 100%로 유지
   - 음악과 취향에 따라 미세 조정

FDN Reverb는 어떤 녹음에도 아름답고 자연스럽게 들리는 잔향과 현실적인 음향 공간을 추가하여 청취 경험을 변화시킵니다. 아름답고 자연스러운 잔향으로 좋아하는 곡을 향상시키고 싶은 음악 애호가들에게 완벽합니다!

## RS Reverb

음악을 아늑한 방에서부터 웅장한 홀까지 다양한 공간으로 이동시킬 수 있는 이펙트입니다. 음악을 더 입체적이고 몰입감 있게 만드는 자연스러운 에코와 반사음을 추가합니다.

### Listening Experience Guide
- 친밀한 공간:
  - 음악이 따뜻하고 아늑한 방에서 연주되는 것 같은 느낌
  - 가까운 거리의 개인적인 청취에 완벽
  - 선명도를 잃지 않으면서 미묘한 깊이감 추가
- 콘서트홀 경험:
  - 라이브 공연의 웅장함을 재현
  - 클래식과 오케스트라 음악에 웅장한 공간감 추가
  - 몰입감 있는 콘서트 경험 생성
- 분위기 향상:
  - 몽환적이고 환상적인 특질 추가
  - 앰비언트와 분위기 있는 음악에 완벽
  - 매력적인 사운드스케이프 생성

### Parameters
- **Pre-Delay** - 공간 이펙트가 시작되는 시점 제어(0에서 50 ms)
  - 낮은 값: 즉각적이고 친밀한 느낌
  - 높은 값: 더 많은 거리감
  - 자연스러운 사운드를 위해 10ms에서 시작
- **Room Size** - 공간이 얼마나 크게 느껴지는지 설정(2.0에서 50.0 m)
  - 작음(2-5m): 아늑한 방 느낌
  - 중간(5-15m): 라이브룸 분위기
  - 큼(15-50m): 콘서트홀의 웅장함
- **Reverb Time** - 에코가 지속되는 시간(0.1에서 10.0 s)
  - 짧음(0.1-1.0s): 선명하고 집중된 사운드
  - 중간(1.0-3.0s): 자연스러운 공간 사운드
  - 김(3.0-10.0s): 공간감 있고 분위기 있는
- **Density** - 공간이 얼마나 풍성하게 느껴지는지(4에서 8)
  - 낮은 값: 더 정의된 에코
  - 높은 값: 더 부드러운 분위기
  - 자연스러운 사운드를 위해 6에서 시작
- **Diffusion** - 사운드가 퍼져나가는 방식(0.2에서 0.8)
  - 낮은 값: 더 구분되는 에코
  - 높은 값: 더 부드러운 블렌드
  - 균형 잡힌 사운드를 위해 0.5 시도
- **Damping** - 에코가 사라지는 방식(0에서 100%)
  - 낮은 값: 더 밝고 개방적인 사운드
  - 높은 값: 더 따뜻하고 친밀한
  - 자연스러운 느낌을 위해 40% 정도에서 시작
- **High Damp** - 공간의 밝기 제어(1000에서 20000 Hz)
  - 낮은 값: 더 어둡고 따뜻한 공간
  - 높은 값: 더 밝고 개방적
  - 자연스러운 사운드를 위해 8000Hz 정도에서 시작
- **Low Damp** - 공간의 풍성함 제어(20에서 500 Hz)
  - 낮은 값: 더 풍성하고 풍부한 사운드
  - 높은 값: 더 선명하고 제어된
  - 균형 잡힌 베이스를 위해 100Hz 정도에서 시작
- **Mix** - 이펙트와 원음의 밸런스(0에서 100%)
  - 10-30%: 미묘한 향상
  - 30-50%: 눈에 띄는 공간감
  - 50-100%: 극적인 이펙트

### Recommended Settings for Different Music Styles

1. 콘서트홀의 클래식 음악
   - Room Size: 30-40m
   - Reverb Time: 2.0-2.5s
   - Mix: 30-40%
   - 적합한 장르: 오케스트라 작품, 피아노 협주곡

2. 친밀한 재즈 클럽
   - Room Size: 8-12m
   - Reverb Time: 1.0-1.5s
   - Mix: 20-30%
   - 적합한 장르: 재즈, 어쿠스틱 공연

3. 현대 팝/록
   - Room Size: 15-20m
   - Reverb Time: 1.2-1.8s
   - Mix: 15-25%
   - 적합한 장르: 현대 음악

4. 앰비언트/일렉트로닉
   - Room Size: 25-40m
   - Reverb Time: 3.0-6.0s
   - Mix: 40-60%
   - 적합한 장르: 분위기 있는 일렉트로닉 음악

### Quick Start Guide

1. 공간 선택
   - Room Size로 기본 공간 설정
   - Reverb Time으로 원하는 분위기 조정
   - Mix로 적절한 밸런스 조정

2. 사운드 쉐이핑
   - Damping으로 따뜻함 제어
   - High/Low Damp로 톤 조정
   - Density와 Diffusion으로 텍스처 설정

3. 이펙트 미세 조정
   - Pre-Delay로 더 많은 깊이감 추가
   - Mix로 최종 밸런스 조정
   - 귀를 믿고 취향에 맞게 조정
