# 분석 플러그인

음악을 흥미로운 방식으로 시각화할 수 있는 플러그인 모음입니다. 이러한 시각적 도구들은 소리의 다양한 측면을 보여줌으로써 청취 경험을 더욱 몰입감 있고 상호작용적으로 만들어줍니다.

## 플러그인 목록

- [Level Meter](#level-meter) - 음악의 디지털 신호 레벨을 표시
- [Oscilloscope](#oscilloscope) - 실시간 파형 시각화
- [Spectrogram](#spectrogram) - 시간에 따른 음악의 주파수 변화를 표시
- [Spectrum Analyzer](#spectrum-analyzer) - 음악의 주파수 분포를 실시간으로 표시
- [Stereo Meter](#stereo-meter) - 스테레오 밸런스와 채널 상관을 시각화

## Level Meter

음악의 디지털 신호 레벨을 실시간으로 보여주는 디스플레이입니다. 이펙트 적용 후 레벨을 확인하고, 들리는 왜곡으로 번지기 전에 클리핑 가능성을 파악하는 데 도움이 됩니다.

### 표시 가이드
- 수평 막대가 오른쪽으로 길어질수록 신호 레벨이 큽니다
- 흰색 마커는 최근 최고 레벨을 잠시 표시합니다
- OVERLOAD는 신호가 안전한 디지털 범위를 넘어 왜곡될 수 있음을 뜻합니다
- 깨끗한 재생을 위해 빨간색 레벨이나 OVERLOAD 경고가 자주 뜨지 않게 조정하세요. 실제 청취 음량은 기기 볼륨에서 맞춥니다

## Oscilloscope

소리의 파형을 실시간으로 보여주어 음악을 들으며 비트, 날카로운 타격음, 음량 변화를 눈으로 확인할 수 있습니다. 파형이 반복될 때는 트리거 설정으로 표시를 더 안정시킬 수 있습니다.

### 표시 가이드
- 가로축은 시간 표시(밀리초)
- 세로축은 정규화된 진폭 표시. 실제 표시 범위는 Display Level과 Vertical Offset에 따라 달라집니다
- 녹색 선이 실제 파형을 추적
- 그리드 선은 시간과 진폭 값 측정에 도움
- 트리거 설정은 파형 캡처가 시작되는 위치를 결정하며 별도 마커는 표시되지 않습니다

### 파라미터
- **Display Time** - 표시할 시간 범위(1에서 100ms)
  - 낮은 값: 짧은 이벤트의 더 자세한 관찰
  - 높은 값: 더 긴 패턴 관찰
- **Trigger Mode**
  - Auto: 트리거 없이도 지속적 업데이트
  - Normal: 다음 트리거까지 디스플레이 고정
- 트리거 검출은 왼쪽과 오른쪽 채널의 평균 파형을 사용합니다. 모노 입력은 그대로 사용됩니다.
- **Trigger Level** - 캡처를 시작하는 진폭 레벨
  - 범위: -1에서 1(정규화된 진폭)
- **Trigger Edge**
  - Rising: 신호가 상승할 때 트리거
  - Falling: 신호가 하강할 때 트리거
- **Holdoff** - 트리거 간 최소 시간(0.1에서 10ms)
- **Display Level** - 수직 스케일(dB 단위)(-96에서 0dB)
- **Vertical Offset** - 파형을 위/아래로 이동(-1에서 1)

### 파형 표시에 대한 참고
표시 파형은 캡처된 지점을 시간 순서대로 연결합니다. Display Time이 길 때는 각 구간의 첫 번째와 마지막 샘플뿐 아니라 최솟값과 최댓값 및 원래 위치도 유지하므로, 표시 해상도 범위에서 연속성과 짧은 피크가 보존됩니다. 정밀 측정 도구라기보다는 시각적 참고용으로 보세요.

## Spectrogram

시간에 따른 음악의 변화를 다채로운 패턴으로 보여줍니다. 색상은 각 주파수의 강도를, 세로 위치는 주파수를 나타냅니다.

### 표시 가이드
- 색상은 다양한 주파수의 강도를 보여줌:
  - 어두운 색상: 조용한 소리
  - 밝은 색상: 큰 소리
  - 음악에 따라 변화하는 패턴 관찰
- 세로 위치는 주파수를 나타냄:
  - 하단: 베이스 사운드
  - 중간: 주요 악기
  - 상단: 고주파

### 볼 수 있는 것
- 멜로디: 흐르는 색상 선
- 비트: 수직 줄무늬
- 베이스: 하단의 밝은 색상
- 하모니: 여러 개의 평행선
- 각기 다른 악기가 만드는 고유한 패턴

### 파라미터
- **DB Range** - 색상의 선명도(-144dB에서 -48dB)
  - 낮은 숫자: 더 미묘한 디테일 관찰
  - 높은 숫자: 주요 소리에 집중
- **Points** - 표시에 사용하는 FFT 크기(256에서 16384)
  - 높은 숫자: 더 세밀한 주파수 정보, 더 느린 시간 업데이트
  - 낮은 숫자: 더 빠른 움직임, 더 낮은 주파수 해상도
- 분석기는 왼쪽과 오른쪽 채널의 평균을 사용합니다. 모노 입력은 그대로 분석됩니다.

## Spectrum Analyzer

깊은 베이스부터 높은 트레블까지 음악의 주파수를 실시간으로 시각화합니다. 음악의 완전한 사운드를 구성하는 개별 요소들을 보는 것과 같습니다.

### 표시 가이드
- 왼쪽은 저주파수 표시(드럼, 베이스 기타)
- 중간은 주요 주파수 표시(보컬, 기타, 피아노)
- 오른쪽은 고주파수 표시(심벌즈, 반짝임, 공기감)
- 높은 피크는 해당 주파수의 강한 존재감을 의미
- 어두운 녹색 선은 현재 소리를 표시
- 밝은 녹색 선은 최근 피크를 잠시 유지하여 방금 지나간 강한 소리를 보여줌
- 다양한 악기가 만드는 서로 다른 패턴 관찰

### 볼 수 있는 것
- 베이스 드롭: 왼쪽의 큰 움직임
- 보컬 멜로디: 중간 영역의 활동
- 선명한 고음: 오른쪽의 반짝임
- 전체 믹스: 모든 주파수가 함께 작동하는 방식

### 파라미터
- **DB Range** - 디스플레이의 감도(-144dB에서 -48dB)
  - 낮은 숫자: 더 미묘한 디테일 관찰
  - 높은 숫자: 주요 소리에 집중
- **Points** - 가까운 주파수를 얼마나 세밀하게 분리할지 설정(256에서 16384)
  - 높은 숫자: 더 세밀한 주파수 정보, 더 느린 업데이트
  - 낮은 숫자: 더 빠른 업데이트, 더 낮은 주파수 해상도
- 분석기는 왼쪽과 오른쪽 채널의 평균을 사용합니다. 모노 입력은 그대로 분석됩니다.

### 이 도구들을 즐기는 방법

1. 음악 탐험하기
   - 다양한 장르가 만드는 서로 다른 패턴 관찰
   - 어쿠스틱과 일렉트로닉 음악의 차이점 확인
   - 악기들이 서로 다른 주파수 범위를 차지하는 방식 관찰

2. 사운드에 대해 배우기
   - 일렉트로닉 음악의 베이스 관찰
   - 보컬 멜로디가 디스플레이를 가로지르는 모습 관찰
   - 드럼이 만드는 날카로운 패턴 관찰

3. 경험 향상시키기
   - Level Meter를 사용해 이펙트 추가 후 신호 피크 확인
   - Spectrum Analyzer가 음악과 함께 춤추는 모습 감상
   - Spectrogram으로 시각적 라이트쇼 만들기

## Stereo Meter

스테레오 사운드가 만들어내는 공간감을 시각화하는 매력적인 도구입니다. 스피커나 헤드폰 사이에서 움직이는 소리를 관찰할 수 있어, 청취 경험에 흥미로운 시각적 차원을 더해줍니다.

### 표시 가이드
- **다이아몬드 디스플레이** - 음악이 생생하게 표현되는 메인 창:
  - 중앙: 매우 조용한 순간 또는 합산 신호가 거의 0에 가까운 순간
  - 상하: 중앙에 있는 소리나 모노에 가까운 소리처럼 좌우 채널이 공유하는 성분
  - 좌우: 채널 간 차이 성분 또는 역상 성분
  - 한쪽이 훨씬 강한 소리는 표시된 모서리 쪽으로 나타날 수 있음
  - 녹색 점이 현재 음악에 맞춰 춤추듯 움직임
  - 흰색 선이 음악의 피크를 추적
- **Correlation Bar** (왼쪽)
  - 좌우 채널의 상관 관계를 표시
  - 상단(+1.0): 좌우가 거의 같아 보통 중앙에 모여 들림
  - 중앙(0.0): 채널 관계가 약하며 넓은 앰비언스나 서로 다른 좌우 성분에서 자주 나타남
  - 하단(-1.0): 좌우가 거의 반대 극성이라 스피커에서 약하게 들릴 수 있음
- **밸런스 바** (하단)
  - 한쪽 스피커가 다른 쪽보다 더 큰 소리를 내는지 표시
  - 중앙: 양쪽 스피커가 동일한 볼륨
  - 좌/우: 한쪽 스피커의 소리가 더 강함
  - 숫자는 데시벨(dB) 단위로 차이를 표시

### 볼 수 있는 것
- **중앙 사운드**: 중앙에서의 강한 수직 움직임
- **공간감 있는 사운드**: 디스플레이 전체에 퍼진 활동
- **특수 효과**: 모서리 부분의 흥미로운 패턴
- **스피커 밸런스**: 하단 바가 가리키는 방향
- **채널 상관**: 왼쪽 Correlation Bar가 보여주는 관계

### 파라미터
- **Window** (10-1000 ms) - 표시할 최근 오디오 구간
  - 낮은 값: 빠른 음악 변화 확인
  - 높은 값: 전체적인 사운드 패턴 확인
  - 기본값: 100 ms가 대부분의 음악에 적합

### 음악 즐기기
1. **다양한 스타일 관찰**
   - 클래식 음악은 부드럽고 균형 잡힌 패턴을 보여줌
   - 일렉트로닉 음악은 대담하게 퍼지는 디자인을 만듦
   - 라이브 녹음은 자연스러운 공간 움직임을 보여줌

2. **사운드 특성 발견**
   - 앨범마다 다른 스테레오 효과 사용법 확인
   - 곡마다 다른 소리의 넓이감 관찰
   - 악기가 스피커 사이를 이동하는 모습 관찰

3. **경험 향상**
   - 다른 헤드폰으로 스테레오 표현의 차이 비교
   - 좋아하는 곡의 새로운 버전과 옛 버전 비교
   - 다른 청취 위치에서의 디스플레이 변화 관찰

기억하세요: 이러한 도구들은 청취 경험에 시각적 차원을 더함으로써 음악을 더욱 즐겁게 만들어주기 위한 것입니다. 좋아하는 음악을 새로운 방식으로 탐험하고 발견하는 즐거움을 누려보세요!
