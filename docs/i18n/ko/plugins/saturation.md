# 새츄레이션 플러그인

음악에 따뜻함과 특성을 더하는 플러그인 모음입니다. 이러한 이펙트들은 디지털 음악을 더 아날로그처럼 들리게 만들고, 빈티지 오디오 장비가 사운드에 색채를 입히는 것과 유사하게 즐거운 풍성함을 더할 수 있습니다.

## 플러그인 목록

- [Dynamic Saturation](#dynamic-saturation) - 스피커 콘의 비선형 이동을 시뮬레이션
- [Exciter](#exciter) - 명료도와 존재감을 높이는 하모닉 콘텐츠를 추가
- [Hard Clipping](#hard-clipping) - 사운드에 강도와 날카로움을 추가
- [Harmonic Distortion](#harmonic-distortion) - 2차에서 5차까지의 비선형 항을 조절해 파형을 형성
- [Multiband Saturation](#multiband-saturation) - 서로 다른 주파수 대역을 독립적으로 형성하고 강화
- [Saturation](#saturation) - 빈티지 장비와 같은 따뜻함과 풍성함을 추가
- [Sub Synth](#sub-synth) - 저역 향상을 위한 추가 저주파 신호 생성 및 블렌드

## Dynamic Saturation

물리학 기반 이펙트로 다양한 조건에서 스피커 콘의 비선형 이동을 시뮬레이션합니다. 스피커의 기계적 동작을 모델링한 다음 그 이동에 새츄레이션을 적용하여, 음악에 역동적으로 응답하는 독특한 형태의 디스토션을 만들어냅니다.

### 청취 향상 가이드
- **미묘한 향상:**
  - 부드러운 따뜻함과 약간 둥글어진 피크 느낌을 더함
  - 명확한 디스토션 없이 자연스럽게 "푸시된" 사운드 생성
  - 사운드에 미묘한 깊이감과 차원감 추가
- **적당한 이펙트:**
  - 더 역동적이고 반응성 있는 디스토션 생성
  - 지속적인 사운드에 독특한 움직임과 생동감 추가
  - 트랜지언트에 움직임 있고 반응적인 캐릭터를 부여
- **창의적인 이펙트:**
  - 입력에 따라 진화하는 복잡한 디스토션 패턴 생성
  - 공명하는 스피커 같은 동작 생성
  - 실험적인 청취에 사용할 수 있는 극적인 질감 제공

### 파라미터
- **Speaker Drive** (0.0-10.0) - 오디오 신호가 콘을 얼마나 강하게 움직이는지 제어
  - 낮은 값: 미묘한 움직임과 부드러운 효과
  - 높은 값: 극적인 움직임과 더 강한 특성
- **Speaker Stiffness** (0.0-10.0) - 콘의 서스펜션 강성을 시뮬레이션
  - 낮은 값: 느슨하고 자유로운 움직임과 더 긴 감쇠
  - 높은 값: 단단하고 제어된 움직임과 빠른 응답
- **Speaker Damping** (0.1-10.0) - 콘의 움직임이 얼마나 빨리 정착하는지 제어
  - 낮은 값: 연장된 진동과 공명
  - 높은 값: 제어된 사운드를 위한 빠른 감쇠
- **Speaker Mass** (0.1-5.0) - 콘의 관성을 시뮬레이션
  - 낮은 값: 빠르고 반응성 있는 움직임
  - 높은 값: 더 느리고 두드러진 움직임
- **Distortion Drive** (0.0-10.0) - 변위 새츄레이션의 강도 제어
  - 낮은 값: 미묘한 비선형성
  - 높은 값: 강한 새츄레이션 특성
- **Distortion Bias** (-1.0-1.0) - 새츄레이션 곡선의 대칭성 조정
  - 0: 대칭적 새츄레이션
  - 양수/음수: 변위의 어느 쪽이 더 강하게 포화되는지 바꿔 비대칭 캐릭터를 더함
- **Distortion Mix** (0-100%) - 선형 및 포화 변위 사이의 블렌드
  - 낮은 값: 더 선형적인 응답
  - 높은 값: 더 포화된 특성
- **Cone Motion Mix** (0-100%) - 콘의 움직임이 원래 사운드에 얼마나 영향을 주는지 제어
  - 낮은 값: 미묘한 향상
  - 높은 값: 극적인 효과
- **Output Gain** (-18.0-18.0dB) - 최종 출력 레벨 조정

### 시각적 디스플레이
- 변위가 어떻게 포화되는지 보여주는 인터랙티브 전달 곡선 그래프
- 디스토션 특성의 명확한 시각적 피드백
- Distortion Drive와 Bias가 사운드에 어떤 영향을 미치는지에 대한 시각적 표현

### 음악 향상 팁
- 미묘한 따뜻함을 위해:
  - Speaker Drive: 2.0-3.0
  - Speaker Stiffness: 1.5-2.5
  - Speaker Damping: 0.5-1.5
  - Distortion Drive: 1.0-2.0
  - Cone Motion Mix: 20-40%
  - Distortion Mix: 30-50%

- 역동적인 특성을 위해:
  - Speaker Drive: 3.0-5.0
  - Speaker Stiffness: 2.0-4.0
  - Speaker Mass: 0.5-1.5
  - Distortion Drive: 3.0-6.0
  - Distortion Bias: 비대칭 특성을 위해 ±0.2 시도
  - Cone Motion Mix: 40-70%

- 실험적인 청취를 위해:
  - Speaker Drive: 6.0-10.0
  - Speaker Stiffness: 극단적인 값 시도(매우 낮거나 높은)
  - Speaker Mass: 과장된 움직임을 위해 2.0-5.0
  - Distortion Drive: 5.0-10.0
  - Bias 값으로 실험
  - Cone Motion Mix: 70-100%

### 빠른 시작 가이드
1. 적당한 Speaker Drive(3.0)와 Stiffness(2.0)로 시작
2. 공명을 제어하기 위해 Speaker Damping 설정(균형 잡힌 응답을 위해 1.0)
3. 취향에 맞게 Distortion Drive 조정(적당한 효과를 위해 3.0)
4. 처음에는 Distortion Bias를 0.0으로 유지
5. Distortion Mix를 50%, Cone Motion Mix를 50%로 설정
6. 이펙트의 특성을 변경하기 위해 Speaker Mass 조정
7. 레벨 균형을 맞추기 위해 Output Gain으로 미세 조정

## Exciter

명료도와 존재감을 높이는 하모닉 콘텐츠를 더하는 이펙트입니다. 고주파수 콘텐츠를 필터링하고 새츄레이션을 적용하여 음악을 더 밝고 또렷하게 느끼게 하는 추가 하모닉을 생성합니다.

### 청취 향상 가이드
- **미묘한 향상:**
  - 보컬에 명료도와 공기감을 추가
  - 전체 재생 신호의 존재감을 향상
  - 더 열린, 상세한 사운드 생성
- **적당한 이펙트:**
  - 믹스에서 숨겨진 디테일을 끌어냄
  - 반짝임과 밝기를 추가
  - 음악을 더 "하이파이"로 만듦
- **창의적인 이펙트:**
  - 밝고 날카로운 톤을 생성
  - 강한 존재감을 추가
  - 더 밝고 앞으로 나온 사운드를 원할 때 유용하지만, 과하게 쓰지 않는 것이 좋음

### 파라미터
- **HPF Freq** (500-10000Hz) - 하이패스 필터링의 컷오프 주파수를 설정
  - 낮은 값 (500-2000Hz): 신호의 더 많은 부분에 영향
  - 중간 값 (2000-5000Hz): 프레젠스 주파수를 타겟
  - 높은 값 (5000-10000Hz): 공기감과 밝기에 집중
- **HPF Slope** - 필터의 급경사를 조절
  - Off: 필터링 없이 전체 스펙트럼 처리
  - 6dB/oct: 부드러운 필터링
  - 12dB/oct: 더 급경사인 필터링
- **Drive** (0.0-10.0) - 새츄레이션 강도를 조절
  - 가벼움 (0.0-3.0): 미묘한 하모닉 향상
  - 중간 (3.0-6.0): 눈에 띄는 밝기
  - 높음 (6.0-10.0): 강한 익사이트 효과
- **Bias** (-0.3에서 0.3) - 새츄레이션 비대칭성을 조정
  - 음수: 신호의 음수 부분을 강조
  - 영: 대칭적 새츄레이션
  - 양수: 신호의 양수 부분을 강조
- **Mix** (0-100%) - 생성된 하모닉 향상 신호를 원음에 더하는 양을 조절
  - 낮음 (0-30%): 미묘한 향상
  - 중간 (30-60%): 균형 잡힌 이펙트
  - 높음 (60-100%): 강한 하모닉 향상; 거칠어지지 않도록 주의해서 사용

### 시각적 디스플레이
- 하이패스 필터 주파수 응답 그래프
- 새츄레이션 전달 곡선 시각화
- 필터와 새츄레이션 모두에 대한 명확한 시각적 피드백

### 음악 향상 팁
- 보컬 프레젠스를 위해:
  - HPF Freq: 3000-5000Hz
  - HPF Slope: 6dB/oct
  - Drive: 2.0-4.0
  - Bias: 0.05에서 0.1
  - Mix: 20-40%

- 악기 명료도를 위해:
  - HPF Freq: 2000-4000Hz
  - HPF Slope: 12dB/oct
  - Drive: 3.0-5.0
  - Bias: 0.0
  - Mix: 30-50%

- 전체 트랙의 은은한 밝기 향상을 위해:
  - HPF Freq: 5000-8000Hz
  - HPF Slope: 6dB/oct
  - Drive: 1.0-3.0
  - Bias: 0.0에서 0.1
  - Mix: 10-25%

### 빠른 시작 가이드
1. 원하는 주파수 범위를 타겟하도록 HPF Freq 설정
2. HPF Slope 선택 (6dB/oct로 시작)
3. 적당한 Drive (3.0)로 시작
4. 약간의 비대칭 특성을 위해 Bias를 0.1 근처에 유지
5. Mix를 25%로 설정하고 취향에 맞게 조정
6. 들으면서 모든 파라미터를 미세 조정

## Hard Clipping

설정한 Threshold를 넘는 피크를 제한하는 디지털 클리핑 이펙트입니다. 사운드에 더 강한 엣지나 밀도, 창의적인 왜곡을 더하고 싶을 때 사용하세요. 가벼운 피크 제어에는 Threshold를 높게 두고, 더 강한 캐릭터가 필요할수록 조금씩 낮추면 됩니다.

### 청취 향상 가이드
- 미묘한 향상:
  - Threshold가 높게 유지될 때 약간의 엣지와 밀도를 더함
  - 가볍게 사용하면 날카로운 피크를 다듬을 수 있음
  - 클리핑은 과하게 걸면 거칠어질 수 있으므로 바이패스 상태와 비교
- 중간 정도의 이펙트:
  - 더 에너지 넘치는 사운드 생성
  - 리듬 요소에 흥분감 추가
  - 음악이 더 "드라이브가 걸린" 느낌
- 창의적인 이펙트:
  - 극적인 사운드 변형 생성
  - 음악에 공격적인 특성 추가
  - 실험적인 청취에 완벽

### 파라미터
- **Threshold** - 사운드가 영향을 받는 정도 제어(-60dB에서 0dB)
  - 높은 값(-6dB에서 0dB): 가벼운 피크 제어 또는 은은한 엣지
  - 중간 값(-24dB에서 -6dB): 뚜렷한 클리핑 캐릭터와 밀도
  - 낮은 값(-60dB에서 -24dB): 강한 왜곡과 극적인 이펙트
- **Mode** - 사운드의 어느 부분에 영향을 줄지 선택
  - Both Sides: 양수와 음수 피크를 대칭적으로 클리핑하며 가장 예측하기 쉬운 모드
  - Positive Only: 양수 피크만 클리핑하여 비대칭 클리핑과 다른 음색 캐릭터를 만듦
  - Negative Only: 음수 피크만 클리핑하여 Positive Only와는 다른 느낌의 비대칭 클리핑을 만듦

### 시각적 표시
- 사운드가 어떻게 쉐이핑되는지 보여주는 실시간 그래프
- 설정을 조정할 때 명확한 시각적 피드백
- 조정을 돕는 참조선

### 청취 팁
- 미묘한 향상을 위해:
  1. Threshold를 0dB에서 시작
  2. "Both Sides" 모드 사용
  3. -3dB에서 -6dB 방향으로 점진적으로 낮추고 이펙트가 막 들리기 시작하면 멈춤
- 창의적인 이펙트를 위해:
  1. Threshold를 점진적으로 낮춤
  2. 다양한 Mode 시도
  3. 독특한 사운드를 위해 다른 이펙트와 결합

## Harmonic Distortion

Harmonic Distortion 플러그인은 조절 가능한 2차에서 5차 비선형 항으로 파형을 형성합니다. 짝수차와 홀수차 디스토션 캐릭터를 은은한 따뜻함부터 더 강한 색채감까지 조정할 수 있어, 너무 깨끗하거나 얇거나 평평하게 들리는 음악을 더 생생하게 느끼도록 도와줍니다.

### 청취 향상 가이드
- **미묘한 효과:**
  - 은은한 고조파의 따뜻함 층을 추가합니다
  - 원 신호를 압도하지 않으면서 자연스러운 톤을 강화합니다
  - 아날로그 감성의 은은한 깊이를 추가하기에 이상적입니다
- **적당한 효과:**
  - 더 뚜렷한 하모닉 캐릭터를 더합니다
  - 전체 녹음에 바디감, 밝기 또는 엣지를 더할 수 있습니다
  - 소리가 너무 평평하거나 억제되어 느껴질 때 유용합니다
- **공격적인 효과:**
  - 여러 비선형 항을 더 강하게 만듭니다
  - 강한 하모닉 질감으로 실험적인 청취 효과를 만들 수 있습니다
  - 독특하고 엣지 있는 질감을 추가하는 데 완벽합니다
- **양수와 음수 값의 차이:**
  - 양수와 음수 값은 각 비선형 항의 방향을 뒤집습니다
  - 짝수 차수 항은 주로 비대칭성과 음색을 바꿉니다
  - 홀수 차수 항은 주로 대칭적인 왜곡 캐릭터를 바꿉니다

### 파라미터
- **2nd Harm (%):** 두 번째 고조파 추가량을 조절합니다 (-30에서 30%, 기본값: 2%)
- **3rd Harm (%):** 세 번째 고조파의 기여도를 조절합니다 (-30에서 30%, 기본값: 3%)
- **4th Harm (%):** 네 번째 고조파의 강도를 조절합니다 (-30에서 30%, 기본값: 0.5%)
- **5th Harm (%):** 다섯 번째 고조파의 수준을 설정합니다 (-30에서 30%, 기본값: 0.3%)
- **Sensitivity (x):** 전반적인 입력 민감도를 조절합니다 (0.1–2.0, 기본값: 0.5)
  - 낮은 민감도는 보다 미묘한 효과를 제공합니다
  - 높은 민감도는 왜곡 강도를 증가시킵니다
  - 모든 고조파의 강도에 영향을 미치는 전반적인 제어 역할을 합니다

### 시각적 디스플레이
- 입력 레벨이 출력 레벨로 어떻게 형성되는지 보여주는 전달 곡선입니다.
- 직관적인 슬라이더와 입력 필드가 즉각적인 피드백을 제공합니다.
- 하모닉과 Sensitivity 설정이 바뀔 때 그래프가 함께 갱신됩니다.

### 빠른 시작 가이드
1. **초기화:** 기본 설정(2nd: 2%, 3rd: 3%, 4th: 0.5%, 5th: 0.3%, Sensitivity: 0.5)으로 시작합니다.
2. **파라미터 조정:** 거칠어짐이나 선명도 저하를 들으면서 한 번에 한두 개의 하모닉 컨트롤만 바꿉니다.
3. **사운드 혼합:** Sensitivity를 이용해 효과의 균형을 맞추어 은은한 따뜻함 또는 두드러진 왜곡 효과를 구현합니다.

## Multiband Saturation

전체 재생 신호의 특정 주파수 범위에 따뜻함과 캐릭터를 더할 수 있는 다목적 이펙트입니다. 사운드를 저역, 중역, 고역으로 나누고 각 범위를 독립적으로 다듬어 정밀하게 사운드를 보강합니다.

### 청취 향상 가이드
- 저역의 따뜻함:
  - 저주파 범위에 따뜻함과 펀치감 추가
  - 전체 재생 신호의 저주파 범위에 충실함과 부드러운 펀치를 더함
  - 더 풍부하고 충실한 저역 생성
- 중역 명료도:
  - 여러 목소리와 악기가 모이는 중역에 바디감과 윤곽을 더함
  - 복잡한 녹음이 더 명료하게 느껴지도록 도움
  - 더 선명하고 정의된 사운드 생성
- 고역 보정:
  - 고주파 범위에 반짝임 추가
  - 공기감과 밝기를 향상
  - 선명하고 디테일한 고역 생성

이 처리는 주파수 대역을 기준으로 동작하므로 선택한 범위 안의 모든 소리에 영향을 주며, 특정 악기나 보컬만 분리해서 처리하지 않습니다.

### 파라미터
- **크로스오버 주파수**
  - Freq 1 (20Hz-2kHz): 저역 밴드가 끝나고 중역 밴드가 시작되는 지점 설정
  - Freq 2 (200Hz-20kHz, 항상 Freq 1 이상으로 유지): 중역 밴드가 끝나고 고역 밴드가 시작되는 지점 설정
  - Freq 2가 Freq 1보다 낮게 설정되면 저역-중역-고역 밴드 순서를 유지하기 위해 자동으로 올라갑니다
- **밴드 컨트롤** (저역, 중역, 고역 각 밴드용):
  - **Drive** (0.0-10.0): 새츄레이션 강도 제어
    - 가벼움(0.0-3.0): 미묘한 향상
    - 중간(3.0-6.0): 눈에 띄는 따뜻함
    - 강함(6.0-10.0): 강한 특성
  - **Bias** (-0.3에서 0.3): 새츄레이션 커브의 대칭성 조정
    - 0: 대칭적 새츄레이션
    - 양수/음수: 파형의 어느 쪽이 더 강하게 포화되는지 바꿔 비대칭 캐릭터를 더함
  - **Mix** (0-100%): 이펙트와 원음 블렌드
    - 낮음(0-30%): 미묘한 향상
    - 중간(30-70%): 균형 잡힌 이펙트
    - 높음(70-100%): 강한 특성
  - **Gain** (-18dB에서 +18dB): 밴드 볼륨 조정
    - 밴드 간의 균형을 맞추는 데 사용
    - 볼륨 변화 보상

### 시각적 표시
- 인터랙티브한 밴드 선택 탭
- 각 밴드의 실시간 전달 커브 그래프
- 설정 조정 시 명확한 시각적 피드백

### 음악 향상 팁
- 전체 믹스 향상:
  1. 모든 밴드에서 부드러운 Drive(2.0-3.0)로 시작
  2. 자연스러운 새츄레이션을 위해 Bias를 0.0으로 유지
  3. 자연스러운 블렌드를 위해 Mix를 40-50%로 설정
  4. 각 밴드의 Gain 미세 조정

- 저역 향상:
  1. 저역 밴드에 집중
  2. 중간 정도의 Drive(3.0-5.0) 사용
  3. 일관된 반응을 위해 Bias를 중립으로 유지
  4. Mix를 50-70%로 유지

- 중역 존재감:
  1. 중역 밴드에 집중
  2. 가벼운 Drive(1.0-3.0) 사용
  3. 자연스러운 사운드를 위해 Bias를 0.0으로 유지
  4. Mix를 취향에 맞게 조정(30-50%)

- 밝기 추가:
  1. 고역 밴드에 집중
  2. 부드러운 Drive(1.0-2.0) 사용
  3. 깨끗한 새츄레이션을 위해 Bias를 중립으로 유지
  4. Mix를 미묘하게 유지(20-40%)

### 빠른 시작 가이드
1. 크로스오버 주파수를 설정하여 사운드 분할
2. 모든 밴드에서 낮은 Drive 값으로 시작
3. 처음에는 Bias를 0.0으로 유지
4. Mix를 사용하여 이펙트를 자연스럽게 블렌드
5. Gain 컨트롤로 미세 조정
6. 귀를 믿고 취향에 맞게 조정하세요!

## Saturation

빈티지 튜브 장비의 따뜻하고 즐거운 사운드를 시뮬레이션하는 이펙트입니다. 음악에 풍성함과 특성을 더하여 더 "아날로그"처럼 들리고 덜 "디지털"처럼 들리게 만들 수 있습니다.

### 청취 향상 가이드
- 따뜻함 추가:
  - 디지털 음악을 더 자연스럽게 만듦
  - 사운드에 즐거운 풍성함 추가
  - 재즈와 어쿠스틱 음악에 완벽
- 풍부한 특성:
  - 더 "빈티지한" 사운드 생성
  - 깊이감과 차원감 추가
  - 록과 일렉트로닉 음악에 훌륭함
- 강한 이펙트:
  - 사운드를 극적으로 변형
  - 대담하고 특징적인 톤 생성
  - 실험적인 청취에 이상적

### 파라미터
- **Drive** - 따뜻함과 특성의 양 제어(0.0에서 10.0)
  - 가벼움(0.0-3.0): 미묘한 아날로그 따뜻함
  - 중간(3.0-6.0): 풍부한 빈티지 특성
  - 강함(6.0-10.0): 대담하고 극적인 이펙트
- **Bias** - 새츄레이션 커브의 대칭성 조정(-0.3에서 0.3)
  - 0.0: 대칭적 새츄레이션
  - 양수: waveform의 negative side를 더 prominent하게 만듦
  - 음수: waveform의 positive side를 더 prominent하게 만듦
- **Mix** - 이펙트와 원음의 밸런스(0%에서 100%)
  - 0-30%: 미묘한 향상
  - 30-70%: 균형 잡힌 이펙트
  - 70-100%: 강한 특성
- **Gain** - 전체적인 볼륨 조정(-18dB에서 +18dB)
  - 이펙트가 너무 크면 음수 값 사용
  - 이펙트가 너무 작으면 양수 값 사용

### 시각적 표시
- 사운드가 어떻게 쉐이핑되는지 보여주는 명확한 그래프
- 실시간 시각적 피드백
- 읽기 쉬운 컨트롤

### 음악 향상 팁
- 클래식 & 재즈:
  - 자연스러운 따뜻함을 위한 가벼운 Drive(1.0-2.0)
  - 깨끗한 새츄레이션을 위해 Bias를 0.0으로 유지
  - 미묘함을 위한 낮은 Mix(20-40%)
- 록 & 팝:
  - 풍부한 특성을 위한 중간 Drive(3.0-5.0)
  - 일관된 반응을 위해 Bias를 중립으로 유지
  - 밸런스를 위한 중간 Mix(40-60%)
- 일렉트로닉:
  - 대담한 이펙트를 위한 높은 Drive(4.0-7.0)
  - 다양한 Bias 값으로 실험
  - 특성을 위한 높은 Mix(60-80%)

### 빠른 시작 가이드
1. 부드러운 따뜻함을 위해 낮은 Drive로 시작
2. 처음에는 Bias를 0.0으로 유지
3. Mix로 이펙트의 밸런스 조정
4. 적절한 볼륨을 위해 필요한 경우 Gain 조정
5. 실험하고 귀를 믿으세요!

## Sub Synth

원본 오디오에서 파생한 추가 저주파 신호를 생성하고 블렌드하여 음악의 저역을 보강하는 이펙트입니다. 저역이 부족한 녹음에 깊이감과 힘을 더하거나 더 풍부한 베이스 느낌을 만들 때 유용합니다.

### 청취 향상 가이드
- 저역 향상:
  - 얇은 녹음에 깊이감과 파워 추가
  - 더 충실하고 풍부한 저역 생성
  - 헤드폰 청취에 완벽
- 주파수 제어:
  - 추가 저주파 신호의 정밀한 제어
  - 깨끗한 베이스를 위한 독립적인 필터링
  - 파워를 더하면서도 선명도 유지

### 파라미터
- **Sub Level** - 추가 저주파 신호 레벨 제어(0-200%)
  - 가벼움(0-50%): 미묘한 저역 향상
  - 중간(50-100%): 균형 잡힌 저역 부스트
  - 강함(100-200%): 극적인 저역 이펙트
- **Dry Level** - 원음 신호 레벨 조정(0-200%)
  - 추가 저주파 신호와의 밸런스를 위해 사용
  - 원음의 선명도 유지
- **Sub LPF** - 추가 저주파 신호용 로우패스 필터(5-400Hz)
  - 주파수: 서브의 상한 제어
  - 기울기: 필터 기울기 조정(Off에서 -24dB/oct)
- **Sub HPF** - 추가 저주파 신호용 하이패스 필터(5-400Hz)
  - 주파수: 원치 않는 럼블 제거
  - 기울기: 필터 기울기 제어(Off에서 -24dB/oct)
- **Dry HPF** - 원음 신호용 하이패스 필터(5-400Hz)
  - 주파수: 저역 쌓임 방지
  - 기울기: 필터 기울기 조정(Off에서 -24dB/oct)

### 시각적 표시
- 인터랙티브한 주파수 응답 그래프
- 필터 커브의 명확한 시각화
- 실시간 시각적 피드백

### 음악 향상 팁
- 일반적인 저역 향상:
  1. Sub Level을 50%로 시작
  2. Sub LPF를 100Hz 주변으로 설정(-12dB/oct)
  3. Sub HPF를 20Hz에 유지(-6dB/oct)
  4. Dry Level을 취향에 맞게 조정

- 깨끗한 베이스 부스트:
  1. Sub Level을 70-100%로 설정
  2. Sub LPF를 80Hz에서 사용(-18dB/oct)
  3. Sub HPF를 30Hz로 설정(-12dB/oct)
  4. Dry HPF를 40Hz (-6dB/oct)로 설정

- 최대 임팩트:
  1. Sub Level을 150%까지 증가
  2. Sub LPF를 120Hz로 설정(-24dB/oct)
  3. Sub HPF를 15Hz에 유지(-6dB/oct)
  4. Dry Level로 밸런스 조정

### 빠른 시작 가이드
1. 적당한 Sub Level(50-70%)로 시작
2. Sub LPF를 100Hz 주변으로 설정
3. Sub HPF를 20Hz 주변에서 활성화
4. Dry Level로 밸런스 조정
5. 필요에 따라 필터 미세 조정
6. 귀를 믿고 점진적으로 조정하세요!
