# 공간 오디오 플러그인

스테레오(좌우) 밸런스를 조정하여 헤드폰이나 스피커에서 음악이 어떻게 들리는지 향상시키는 플러그인 모음입니다. 이러한 이펙트들은 특히 헤드폰으로 청취할 때 음악을 더 공간감 있고 자연스럽게 들리게 만들 수 있습니다.

## 플러그인 목록

- [Crossfeed Filter](#crossfeed-filter) - 자연스러운 스테레오 이미지를 위한 헤드폰 크로스피드 필터
- [MS Matrix](#ms-matrix) - 고급 스테레오 조정 체인을 위해 스테레오를 Mid/Side로 변환하고 다시 되돌림
- [Multiband Balance](#multiband-balance) - 5밴드 주파수 의존 스테레오 밸런스 제어
- [Stereo Blend](#stereo-blend) - Side 극성이 반전된 스테레오부터 모노, 향상된 스테레오까지 스테레오 폭 제어

## Crossfeed Filter

스피커를 통해 들을 때 발생하는 자연스러운 음향 크로스토크를 시뮬레이션하는 헤드폰용 크로스피드 필터입니다. 이 효과는 헤드폰으로 들을 때 자주 경험하는 과도한 스테레오 분리를 줄여서, 실제 음향 환경에서 소리가 우리 귀에 도달하는 방식을 모방한 더 자연스럽고 편안한 청취 경험을 만듭니다.

### 주요 기능
- 헤드폰 청취를 위한 자연스러운 음향 크로스토크 시뮬레이션
- 조정 가능한 크로스피드 레벨과 타이밍
- 주파수 의존 크로스토크를 모방하는 로우패스 필터링
- 스테레오 전용 처리 (모노 신호는 자동으로 바이패스)

### 파라미터
- **Level** (-60 dB ~ 0 dB): 크로스피드 신호의 양을 제어
  - 낮은 값 (-20 dB ~ -6 dB): 미묘하고 자연스러운 크로스피드
  - 높은 값 (-6 dB ~ 0 dB): 더 뚜렷한 효과
- **Delay** (0 ms ~ 1 ms): 음향 크로스토크의 시간 차이를 시뮬레이션
  - 낮은 값 (0.1-0.3 ms): 더 타이트하고 포커스된 이미지
  - 높은 값 (0.3-1.0 ms): 더 넓은 스피커와 같은 표현
- **LPF Freq** (100 Hz ~ 20000 Hz): 크로스피드의 주파수 응답을 제어
  - 낮은 값 (500-1000 Hz): 더 자연스럽고 주파수 의존적인 크로스토크
  - 높은 값 (1000-20000 Hz): 더 넓은 주파수 응답

### 권장 설정

1. 자연스러운 헤드폰 청취
   - Level: -12 dB
   - Delay: 0.3 ms
   - LPF Freq: 700 Hz
   - 효과: 편안한 장시간 청취를 위한 미묘한 크로스피드

2. 스피커 시뮬레이션
   - Level: -6 dB
   - Delay: 0.5 ms
   - LPF Freq: 1000 Hz
   - 효과: 더 뚜렷한 스피커와 같은 표현

3. 미묘한 향상
   - Level: -20 dB
   - Delay: 0.2 ms
   - LPF Freq: 500 Hz
   - 효과: 민감한 청취자를 위한 매우 부드러운 크로스피드

### 응용 가이드

1. 헤드폰 최적화
   - 보수적인 설정 (-15 dB level, 0.3 ms delay)으로 시작
   - 편안함과 자연스러움을 위해 레벨 조정
   - 공간적 인식을 위해 딜레이 미세 조정
   - 주파수 응답을 제어하기 위해 LPF 사용

2. 음악 스타일 고려사항
   - 클래식/재즈: 자연스러운 표현을 위한 낮은 레벨 (-15 ~ -10 dB)
   - 록/팝: 중간 레벨 (-12 ~ -8 dB)은 음악의 생동감을 유지하면서 hard-panned 기타나 보컬을 부드럽게 만들 수 있습니다
   - 일렉트로닉 또는 매우 넓은 믹스: 폭을 유지하려면 낮은 레벨부터 중간 레벨 (-18 ~ -10 dB)을 사용하고, 과도한 좌우 분리를 줄이고 싶을 때만 더 높은 레벨을 사용하세요

3. 청취 환경
   - 조용한 환경: 미묘한 효과를 위한 낮은 레벨
   - 시끄러운 환경: 더 나은 포커스를 위한 높은 레벨
   - 긴 청취 세션: 피로를 줄이기 위한 보수적인 설정

### 빠른 시작 가이드

1. 초기 설정
   - Level을 -12 dB로 설정
   - Delay를 0.3 ms로 설정
   - LPF Freq를 700 Hz로 설정

2. 미세 조정
   - 원하는 크로스피드 양을 위해 Level 조정
   - 공간적 인식을 위해 Delay 수정
   - 주파수 응답을 위해 LPF Freq 조정

3. 최적화
   - 자연스럽고 편안한 표현을 위해 청취
   - 인공적으로 들리는 과도한 설정 피하기
   - 다양한 음악 스타일로 테스트

기억하세요: Crossfeed Filter는 헤드폰 청취를 더 자연스럽고 편안하게 만들기 위해 설계되었습니다. 보수적인 설정으로 시작하고 청취 선호도와 음악 자료에 최적의 균형을 찾기 위해 점진적으로 조정하세요.

## MS Matrix

MS Matrix는 일반 스테레오 오디오를 Mid/Side 형식으로 변환하거나, Mid/Side 오디오를 다시 일반 스테레오로 변환합니다. M/S로 인코딩하고 Mid 또는 Side 레벨을 바꾼 뒤 다시 스테레오로 디코딩하는 것처럼, 이펙트 체인 안에서 중앙 정보와 사이드 정보를 따로 조정하고 싶을 때 사용합니다. 일반 음악의 단순한 스테레오 폭 조정에는 [Stereo Blend](#stereo-blend)가 더 직접적인 도구입니다.

### 주요 기능
- Mid 및 Side 게인(–18 dB~+18 dB) 분리 제어  
- Mode 전환: Encode (스테레오→M/S) 또는 Decode (M/S→스테레오)  
- 인코딩 전 또는 디코딩 후 선택적 왼쪽/오른쪽 전환  

### 파라미터
- **Mode** (Encode/Decode): Encode는 좌우 스테레오를 왼쪽 채널의 Mid와 오른쪽 채널의 Side로 변환합니다. Decode는 왼쪽 채널을 Mid, 오른쪽 채널을 Side로 처리한 뒤 일반 스테레오를 다시 구성합니다.
- **Mid Gain** (–18 dB~+18 dB): 선택한 변환 중 Mid 레벨을 조정합니다.
- **Side Gain** (–18 dB~+18 dB): 선택한 변환 중 Side 레벨을 조정합니다.
- **Swap L/R** (Off/On): 인코딩 전 또는 디코딩 후 좌우 채널 전환  

### 권장 설정
1. **일반 스테레오의 은은한 확장**
   - 첫 번째 MS Matrix: Mode: Encode, Mid Gain: 0 dB, Side Gain: +3 dB, Swap: Off
   - 그 뒤의 두 번째 MS Matrix: Mode: Decode, Mid Gain: 0 dB, Side Gain: 0 dB, Swap: Off
   - 효과: Side 성분을 약간 강화한 뒤 결과를 일반 스테레오로 되돌립니다.
2. **일반 스테레오의 중앙 집중**
   - 첫 번째 MS Matrix: Mode: Encode, Mid Gain: +3 dB, Side Gain: -3 dB, Swap: Off
   - 그 뒤의 두 번째 MS Matrix: Mode: Decode, Mid Gain: 0 dB, Side Gain: 0 dB, Swap: Off
   - 효과: 보컬과 중앙에 위치한 소리를 앞으로 가져오면서 사이드 앰비언스를 줄입니다.
3. **기존 M/S 오디오 디코딩**
   - Mode: Decode
   - Mid Gain: 0 dB
   - Side Gain: 0 dB
   - Swap: Off
   - 입력 신호가 이미 Mid/Side 형식일 때만 사용합니다.
4. **창의적 반전**
   - Mode: Encode  
   - Mid Gain: 0 dB  
   - Side Gain: 0 dB  
   - Swap: On  

### 빠른 시작 가이드
1. 단일 변환이 필요한지, 전체 Encode -> 조정 -> Decode 체인이 필요한지 결정합니다.
2. 일반 스테레오 청취에는 MS Matrix 하나를 Encode 모드로 배치하고, 뒤쪽에 두 번째 MS Matrix를 Decode 모드로 배치합니다.
3. Encode 단계에서 **Mid Gain**과 **Side Gain**을 조정합니다.
4. 채널 보정 또는 창의적 반전을 위해서만 **Swap L/R**을 활성화합니다.
5. 바이패스와 비교하여 스테레오 이미지가 여전히 자연스럽게 느껴지는지 확인합니다.

## Multiband Balance

오디오를 5개의 주파수 대역으로 나누고 각 대역을 왼쪽이나 오른쪽으로 조금씩 이동할 수 있는 주파수 의존 밸런스 프로세서입니다. 저역, 보컬, 심벌 또는 다른 주파수 범위가 한쪽으로 치우쳐 들릴 때, 트랙 전체를 움직이지 않고 해당 부분만 다시 균형 있게 맞추는 데 사용합니다.

### 주요 기능
- 5밴드 주파수 의존 스테레오 밸런스 제어
- 고품질 Linkwitz-Riley 크로스오버 필터
- 정확한 스테레오 조정을 위한 선형 밸런스 제어
- 좌우 채널 독립 처리
- 자동 페이드 처리로 클릭 없는 파라미터 변경

### 파라미터

#### 크로스오버 주파수
- **Freq 1** (20-500 Hz): 저역과 중저역 분리
- **Freq 2** (100-2000 Hz): 중저역과 중역 분리
- **Freq 3** (500-8000 Hz): 중역과 중고역 분리
- **Freq 4** (1000-20000 Hz): 중고역과 고역 분리

#### 밴드 컨트롤
각 밴드는 독립적인 밸런스 컨트롤을 가집니다:
- **Band 1 Bal.** (-100%~+100%): 저역 스테레오 밸런스 제어
- **Band 2 Bal.** (-100%~+100%): 중저역 스테레오 밸런스 제어
- **Band 3 Bal.** (-100%~+100%): 중역 스테레오 밸런스 제어
- **Band 4 Bal.** (-100%~+100%): 중고역 스테레오 밸런스 제어
- **Band 5 Bal.** (-100%~+100%): 고역 스테레오 밸런스 제어

### 권장 설정

1. 오른쪽으로 치우친 고역 보정
   - 저역 (20-100 Hz): 0% (중앙)
   - 중저역 (100-500 Hz): 0%
   - 중역 (500-2000 Hz): 0%
   - 중고역 (2000-8000 Hz): -10% ~ -25%
   - 고역 (8000+ Hz): -10% ~ -30%
   - 효과: 저역과 보컬은 안정적으로 유지하면서 밝은 성분을 약간 왼쪽으로 이동

2. 왼쪽으로 치우친 중저역 보정
   - 저역: 0%
   - 중저역: +10% ~ +25%
   - 중역: +5% ~ +15%
   - 중고역: 0%
   - 고역: 0%
   - 효과: 전체 스테레오 이미지를 바꾸지 않고 따뜻한 바디감과 낮은 보컬 대역을 약간 오른쪽으로 이동

3. 저역을 중앙에 유지하면서 공기감 조정
   - 저역: 0%
   - 중저역: 0%
   - 중역: 0%
   - 중고역: +5% ~ +15%
   - 고역: +10% ~ +20%
   - 효과: 저역은 중앙에 유지하면서 상부 앰비언스를 부드럽게 오른쪽으로 이동

### 응용 가이드

1. 청취 밸런스 보정
   - 안정적인 베이스를 위해 저주파(100 Hz 이하)는 중앙에 유지
   - 중심에서 벗어나 들리는 주파수 범위만 이동
   - 먼저 작은 부호 있는 값(약 5-20%) 사용
   - 음색이나 레벨 변화가 있는지 모노 재생 확인

2. 문제 해결
   - 너무 왼쪽이나 오른쪽으로 치우친 주파수 범위 재조정
   - 저주파를 중앙에 두어 포커스가 흐린 베이스를 타이트하게 조정
   - 고주파의 거친 스테레오 아티팩트 감소
   - 서로 다른 소리 부분이 각기 다른 쪽으로 기울어진 녹음 개선

3. 창의적 청취 효과
   - 주파수에 따라 달라지는 독특한 위치감 만들기
   - 저주파는 중앙에 두고 고주파만 한쪽으로 기울이기
   - 상위 밴드의 작은 밸런스 이동으로 더 넓게 느껴지는 앰비언스 만들기

4. 스테레오 필드 조정
   - 주파수 대역별 스테레오 밸런스 미세 조정
   - 불균일한 스테레오 분포 보정
   - 전체 이미지를 넓히거나 좁히는 스테레오 폭 컨트롤로 취급하지 말고, 그럴 때는 Stereo Blend 사용
   - 모노 호환성 유지

### 빠른 시작 가이드

1. 초기 설정
   - 모든 밴드를 중앙(0%)에서 시작
   - 크로스오버 주파수를 표준 지점으로 설정:
     * Freq 1: 100 Hz
     * Freq 2: 500 Hz
     * Freq 3: 2000 Hz
     * Freq 4: 8000 Hz

2. 기본 향상
   - Band 1(저역)은 중앙에 유지
   - 높은 밴드부터 작은 조정
   - 공간 이미지의 변화 청취
   - 모노 호환성 확인

3. 미세 조정
   - 소스 material에 맞게 크로스오버 포인트 조정
   - 밴드 위치를 점진적으로 변경
   - 원치 않는 아티팩트 청취
   - 바이패스와 비교하여 관점 확인

주의: Multiband Balance는 신중한 조정이 필요한 강력한 도구입니다. 미묘한 설정으로 시작하여 필요에 따라 복잡성을 높여가세요. 항상 스테레오와 모노 모두에서 조정을 확인하여 호환성을 확보하세요.

## Stereo Blend

음악의 스테레오 폭을 조정하여 더 자연스러운 음장을 달성하는 데 도움을 주는 이펙트입니다. 특히 헤드폰 청취 시 유용하며, 헤드폰에서 자주 발생하는 과장된 스테레오 분리를 줄여 청취 경험을 더 자연스럽고 덜 피로하게 만들 수 있습니다. 또한 필요할 때 스피커 청취를 위한 스테레오 이미지를 향상시킬 수도 있으며, 음수 폭은 Side 성분의 극성을 반전해 보정 또는 창의적 용도로 사용할 수 있습니다.

### 청취 향상 가이드
- 헤드폰 최적화:
  - 더 자연스럽고 스피커와 같은 표현을 위해 스테레오 폭 감소(60-90%)
  - 과도한 스테레오 분리로 인한 청취 피로도 최소화
  - 더 현실적인 전방 중심의 사운드스테이지 생성
- 스피커 향상:
  - 정확한 재생을 위해 원래의 스테레오 이미지 유지(100%)
  - 필요할 때 더 넓은 사운드스테이지를 위한 미묘한 향상(110-130%)

### 파라미터
- **Stereo** - 스테레오 폭 제어(-200%~200%)
  - 음수 값: 재구성하기 전에 스테레오 Side (L-R) 성분의 극성을 반전
  - -200%: Side 극성이 반전된 최대 폭. 보정 또는 특수한 경우에만 사용
  - -100%: 원래의 스테레오 폭이지만 좌우 이미지가 서로 바뀜
  - 0%: 완전한 모노(좌우 채널 합산)
  - 100%: 원래의 스테레오 이미지
  - 200%: 중앙 성분은 유지하면서 스테레오 Side 차이를 강하게 확대
