# Basic Audio Plugins

음악 재생의 기본 요소를 조정하기 위한 필수 도구 모음입니다. 이 플러그인들은 재생 볼륨, 밸런스 및 기타 기본 설정을 제어하는 데 도움을 줍니다.

## 플러그인 목록

* [Channel Divider](#channel-divider) - 오디오를 여러 채널에 걸친 주파수 대역으로 분리
* [DC Offset](#dc-offset) - 비대칭으로 들리는 오디오를 수정하는 데 도움
* [Matrix](#matrix) - 유연한 제어로 오디오 채널을 라우팅 및 믹싱
* [MultiChannel Panel](#multichannel-panel) - 개별 설정으로 여러 오디오 채널을 제어
* [Mute](#mute) - 오디오 출력을 음소거
* [Polarity Inversion](#polarity-inversion) - 스테레오 음악의 음향을 향상시킬 수 있음
* [Stereo Balance](#stereo-balance) - 음악의 좌우 밸런스를 조정
* [Volume](#volume) - 음악 재생 볼륨을 제어

## Channel Divider

특정 주파수 대역으로 스테레오 신호를 분할하여 각 대역을 서로 다른 출력 채널로 라우팅하는 전문 도구입니다. 멀티 채널 시스템이나 커스텀 크로스오버 구성에 적합합니다.

이 효과를 사용하려면 데스크톱 앱에서 오디오 설정의 출력 채널 수를 4개 이상으로 설정하고, 이펙트 버스 라우팅의 채널을 "All"로 지정해야 합니다。

### 사용할 때

* 멀티 채널 오디오 출력(4, 6 또는 8 채널) 사용 시
* 커스텀 주파수 기반 채널 라우팅 생성 시
* 멀티 앰프 또는 멀티 스피커 구성 시

### 매개변수

* **Band Count** - 생성할 주파수 대역 수 (2~4 밴드)

  * 2 밴드: 저역/고역 분할
  * 3 밴드: 저역/중역/고역 분할
  * 4 밴드: 저역/저중역/중고역/고역 분할

* **Crossover Frequencies** - 대역 간 오디오 분할 지점 정의

  * F1: 첫 번째 크로스오버 포인트
  * F2: 두 번째 크로스오버 포인트 (3개 대역 이상)
  * F3: 세 번째 크로스오버 포인트 (4개 대역)

* **Slopes** - 대역 분리의 기울기 제어

  * 옵션: 옥타브당 -12dB~ -96dB
  * 기울기가 가파를수록 분리가 더 깔끔함
  * 기울기가 낮을수록 자연스러운 전환 제공

### 기술 참고

* 첫 두 입력 채널만 처리
* 출력 채널 수는 2의 배수(4, 6 또는 8)여야 함
* 고품질 Linkwitz-Riley 크로스오버 필터 사용
* 쉬운 설정을 위한 시각적 주파수 응답 그래프 제공

## DC Offset

비대칭으로 들리거나 이상하게 들리는 오디오를 수정하는 데 도움을 주는 유틸리티입니다. 대부분의 사용자는 자주 필요하지 않지만, 오디오가 제대로 들리지 않을 때 유용합니다。

### 사용할 때

* 음악이 비정상적으로 한쪽으로 치우쳐 들릴 때
* 한 채널이 지나치게 크게 들릴 때
* 다른 이펙트가 예상대로 작동하지 않을 때

### 매개변수

* **Offset** - 오디오 밸런스 조정 (-1.0 ~ +1.0)

  * 0.0: 기본 설정
  * 이상한 소리가 날 때 조정
  * 미세 조정이 가장 효과적임

## Matrix

입력 채널과 출력 채널 간의 커스텀 신호 경로를 생성할 수 있는 강력한 채널 라우팅 도구입니다. 오디오 신호 연결 및 믹싱에 완전한 유연성을 제공합니다。

### 사용할 때

* 채널 간 커스텀 라우팅 생성 시
* 신호를 특정 방식으로 믹스하거나 분할해야 할 때
* 채널 상호 작용을 이용한 창의적 사운드 디자인 시

### 특징

* 최대 8개 채널을 위한 유연한 라우팅 매트릭스
* 모든 입력/출력 쌍 간 개별 연결 제어
* 각 연결에 대한 위상 반전 옵션
* 직관적인 설정을 위한 시각적 매트릭스 인터페이스

### 작동 방식

* 각 연결 지점은 입력 행에서 출력 열로의 라우팅을 나타냄
* 활성 연결은 채널 간 신호 흐름을 허용
* 위상 반전 옵션은 신호 위상(폴라리티)을 반전시킴
* 여러 입력이 하나의 출력에 연결되면 혼합됨

### 실제 적용 예

* 커스텀 다운믹싱 또는 업믹싱 구성
* 특정 채널 분리 또는 결합
* 채널 간 위상 관계 생성
* 복잡한 라우팅 요구 사항 해결

## MultiChannel Panel

여러 오디오 채널을 개별적으로 관리하기 위한 종합적인 제어 패널입니다. 이 플러그인은 최대 8개 채널의 볼륨, 음소거, 솔로, 딜레이를 완벽하게 제어할 수 있으며, 각 채널에 시각적 레벨 미터를 제공합니다.

### 사용할 때
- 멀티 채널 오디오(최대 8개 채널)로 작업할 때
- 서로 다른 채널 간 커스텀 볼륨 밸런스를 만들 때
- 특정 채널에 개별 딜레이를 적용해야 할 때
- 여러 채널의 레벨을 동시에 모니터링할 때

### 특징
- 최대 8개 오디오 채널의 개별 제어
- 피크 홀드 기능이 있는 실시간 레벨 미터로 시각적 모니터링 제공
- 그룹 파라미터 변경을 위한 채널 링크 기능

### 매개변수

#### 채널별 제어
- **Mute (M)** - 개별 채널 음소거
  - 각 채널별 켜기/끄기 전환
  - 솔로 기능과 연동하여 작동

- **Solo (S)** - 개별 채널 분리
  - 어떤 채널이라도 솔로 상태이면 솔로 채널만 재생됨
  - 여러 채널을 동시에 솔로 상태로 설정 가능

- **Volume** - 개별 채널 볼륨 조정 (-20dB ~ +10dB)
  - 슬라이더 또는 직접 값 입력을 통한 세밀한 제어
  - 링크된 채널은 동일한 볼륨 유지

- **Delay** - 개별 채널에 시간 지연 추가 (0-30ms)
  - 밀리초 단위의 정밀한 딜레이 제어
  - 채널 간 시간 정렬에 유용
  - 채널 간 위상 조정 가능

#### 채널 링크
- **Link** - 인접 채널을 동기화된 제어를 위해 연결
  - 링크된 채널 중 하나에 대한 변경이 연결된 모든 채널에 영향을 줌
  - 링크된 채널 그룹 간에 일관된 설정 유지
  - 스테레오 페어 또는 멀티 채널 그룹에 유용

### 시각적 모니터링
- 실시간 레벨 미터가 현재 신호 강도를 표시
- 피크 홀드 인디케이터가 최대 레벨을 표시
- 명확한 수치 dB 피크 레벨 표시
- 레벨 인식이 쉬운 색상 코드 미터:
  - 녹색: 안전 레벨
  - 노란색: 최대값에 근접
  - 빨간색: 최대값 근처 또는 최대값

### 실제 적용 예
- 서라운드 사운드 시스템 밸런싱
- 커스텀 헤드폰 믹스 생성
- 멀티 마이크 설정의 시간 정렬
- 멀티 채널 오디오 소스 모니터링 및 조정

## Mute

버퍼를 0으로 채워 모든 오디오 출력을 즉시 음소거하는 간단한 유틸리티입니다. 오디오 신호를 즉시 음소거할 때 유용합니다。

### 사용할 때

* 페이드 없이 즉시 오디오 음소거 필요 시
* 무음 구간 또는 일시 중지 중
* 원치 않는 노이즈 출력 방지 시

## Polarity Inversion

특정 상황에서 스테레오 음악의 음향을 개선할 수 있는 도구입니다. 오디오 파형을 "뒤집는" 것과 유사합니다。

효과 공통 설정에서 처리할 채널을 제한하여 특정 채널만 위상을 반전할 수도 있습니다。

### 사용할 때

* 스테레오 음악이 "공허"하거나 "이상"하게 들릴 때
* 다른 스테레오 이펙트와 결합할 때
* 스테레오 이미징을 개선하려 할 때

## Stereo Balance

음악이 좌우 스피커 또는 헤드폰에 어떻게 분배되는지 조정할 수 있습니다. 불균형한 스테레오를 수정하거나 원하는 사운드 배치를 만들기에 완벽합니다。

### 청취 향상 가이드

* 완벽한 밸런스:

  * 자연스러운 스테레오를 위한 중앙 위치
  * 양쪽 귀에 동일한 볼륨
  * 대부분의 음악에 최적
* 조정된 밸런스:

  * 룸 어쿠스틱 보정
  * 청력 차이 보정
  * 선호하는 사운드 스테이지 생성

### 매개변수

* **Balance** - 좌우 분배 제어 (-100% ~ +100%)

  * 중앙(0%): 양쪽 동일
  * 왼쪽(-100%): 왼쪽에서 더 큰 소리
  * 오른쪽(+100%): 오른쪽에서 더 큰 소리

### 시각적 표시

* 사용하기 쉬운 슬라이더
* 명확한 숫자 표시
* 스테레오 위치 시각 표시

### 권장 사용 사례

1. 일반 청취

   * 밸런스를 중앙(0%)으로 유지
   * 스테레오가 고르지 않다고 느껴지면 조정
   * 미세 조정 사용

2. 헤드폰 청취

   * 편안함을 위해 미세 조정
   * 청력 차이 보정
   * 선호하는 스테레오 이미지 생성

3. 스피커 청취

   * 룸 설정에 맞게 조정
   * 청취 위치에 맞춰 밸런스 조정
   * 룸 어쿠스틱 보정

## Volume

음악 재생 볼륨을 조정할 수 있는 간단하지만 필수적인 컨트롤입니다. 다양한 상황에 맞는 청취 레벨을 찾기에 완벽합니다。

### 청취 향상 가이드

* 다양한 청취 상황에 맞춰 조정:

  * 작업 중 백그라운드 음악
  * 집중 청취 세션
  * 늦은 밤 조용한 청취
* 다음을 피하기 위해 볼륨을 적절한 수준으로 유지:

  * 청취 피로
  * 소리 왜곡
  * 청력 손상 위험

### 매개변수

* **Volume** - 전체 음량 제어 (-60dB ~ +24dB)

  * 낮은 값: 더 조용한 재생
  * 높은 값: 더 큰 소리
  * 0dB: 원래 음량

참고: 이 기본 컨트롤은 좋은 사운드의 토대입니다. 더 복잡한 이펙트를 사용하기 전에 이 조정부터 시작하세요！
