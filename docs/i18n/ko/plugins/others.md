# 기타 오디오 도구

주요 이펙트 카테고리를 보완하는 특수 오디오 도구와 제너레이터 모음입니다. 현재 재생 경로에 테스트 톤이나 노이즈를 섞어 스피커, 헤드폰, 채널 밸런스를 확인하는 데 사용할 수 있습니다.

## 플러그인 목록

- [Oscillator](#oscillator) - 테스트 톤과 노이즈를 현재 오디오 경로에 더하는 신호 제너레이터

## Oscillator

다양한 파형을 생성해 현재 오디오 경로에 섞는 신호 제너레이터입니다. 오디오 입력을 대체하지 않으므로, 음악이 재생 중이면 생성된 톤이나 노이즈가 음악 위에 더해집니다. 큰 소리의 테스트 톤은 듣기 피로와 장비 부담을 줄 수 있으므로, 켜기 전에 **Volume**을 낮게 설정하세요.

### 기능
- 다양한 파형 타입:
  - 특정 주파수 확인을 위한 순수 사인파
  - 풍부한 하모닉을 가진 사각파
  - 더 부드러운 하모닉을 가진 삼각파
  - 밝은 질감의 톱니파
  - 전체 대역 확인을 위한 White Noise
  - 청감상 더 고르게 느껴지는 Pink Noise
- 버스트 테스트와 간헐적 신호를 위한 펄스 동작 모드

### 파라미터
- **Frequency (Hz)** - 생성된 톤의 피치를 제어합니다(20 Hz에서 96 kHz). White Noise와 Pink Noise에서는 비활성화됩니다. 실제로 들리거나 출력되는 최고 주파수는 현재 샘플 레이트의 Nyquist 한계에 따라 제한됩니다.
  - 저주파수: 깊은 베이스 톤
  - 중간 주파수: 음악적 범위
  - 고주파수: 트위터나 고역 응답 확인
- **Volume (dB)** - 출력 레벨 조정(-96 dB에서 0 dB)
  - 처음에는 낮은 값으로 시작
  - 필요할 때만 천천히 올림
- **Panning (L/R)** - 스테레오 배치 제어
  - 중앙: 양쪽 채널에서 동일
  - 좌/우: 채널 밸런스 테스트
- **Waveform Type** - 신호 타입 선택
  - Sine: 깨끗한 레퍼런스 톤
  - Square: 홀수 하모닉이 풍부
  - Triangle: 더 부드러운 하모닉 콘텐츠
  - Sawtooth: 완전한 하모닉 시리즈
  - White Noise: Hz당 동일한 에너지
  - Pink Noise: 옥타브당 동일한 에너지
- **Mode** - 신호 생성 패턴 제어
  - Continuous: 표준 연속 신호 생성
  - Pulsed: 제어 가능한 타이밍의 간헐적 신호
- **Interval (ms)** - 펄스 모드에서 펄스 간 시간 간격(100-2000 ms, 단계 10 ms)
  - 짧은 간격: 빠른 펄스 시퀀스
  - 긴 간격: 넓은 간격의 펄스
  - 모드가 Pulsed로 설정된 경우에만 활성
- **Width (ms)** - 펄스 모드에서 각 펄스의 램프 시간입니다(2-100 ms, 단계 1 ms). 값은 Interval의 절반을 넘지 않도록 제한되며, 실제 펄스는 대략 Width의 두 배 길이로 들립니다.
  - 각 펄스의 페이드 인/페이드 아웃 시간 제어
  - 짧은 폭: 날카로운 펄스 에지
  - 긴 폭: 더 부드러운 펄스 전환
  - 모드가 Pulsed로 설정된 경우에만 활성

### 사용 예

1. 스피커 테스트
   - 주파수 재생 범위 확인
     * 저주파에서 고주파까지 사인파 스윕 사용
     * 소리가 들리지 않거나 왜곡되는 지점 확인
   - 거슬리는 울림이나 왜곡 확인
     * 다양한 주파수에서 순수 사인파 사용
     * 원치 않는 하모닉이나 왜곡 청취
     * 다양한 볼륨 레벨에서의 동작 비교
   - 버스트 신호 테스트
     * 짧은 간격과 폭으로 펄스 모드 사용
     * 간헐적 신호에 대한 시스템 응답 분석

2. 청취 공간 확인
   - 정재파 식별
     * 의심되는 룸 모드 주파수에서 사인파 사용
     * 노드와 안티노드를 찾기 위해 공간 이동
   - 공진과 잔향 확인
     * 문제가 되는 공진을 찾기 위해 다양한 주파수 테스트
     * 전체적인 밸런스를 확인하기 위해 Pink Noise 사용
   - 다양한 위치에서 주파수 응답 매핑
     * 청취 영역 전체의 일관성을 확인하기 위해 사인 스윕 사용
   - 에코와 반사 분석
     * 펄스 모드를 사용하여 직접음과 반사음을 명확히 분리

3. 헤드폰/이어폰 테스트
   - 채널 간 크로스토크 평가
     * 한 채널에만 신호 전송
     * 다른 채널로의 원치 않는 누출 확인
   - 주파수 응답 테스트
     * 주파수 밸런스를 확인하기 위해 사인 스윕 사용
     * 좌우 채널 응답 비교
   - 과도 응답 테스트
     * 펄스 모드를 사용하여 버스트 신호에서의 시스템 동작 평가

4. 청취 범위 확인
   - 개인 청취 범위 확인
     * 상한과 하한을 찾기 위해 주파수 스윕
     * 주파수 간격이나 약점 확인
   - 최소 가청 볼륨 결정
     * 다양한 볼륨에서 여러 주파수 테스트
     * 주파수별 민감도 차이 확인
   - 시간 처리 평가
     * 다양한 간격의 펄스 모드를 사용하여 시간 분해능 테스트

5. 시스템 확인
   - 컴포넌트 간 레벨 매칭
     * 레퍼런스 주파수에서 사인파 사용
     * 신호 체인 전체에서 일관된 레벨 확보
   - 채널 밸런스 검증
     * 다양한 주파수에서 좌/우 밸런스 테스트
     * 적절한 스테레오 이미징 확보
   - 버스트 신호 응답 테스트
     * 펄스 모드를 사용하여 간헐적 신호에 대한 시스템 응답 테스트
     * 짧은 소리에 대한 반응 확인

Oscillator는 테스트용 도구입니다. 장비 손상이나 청취 피로를 방지하기 위해 낮은 Volume에서 시작하여 필요한 만큼만 천천히 올리세요.
