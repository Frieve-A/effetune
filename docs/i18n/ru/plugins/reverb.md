# Плагины реверберации

Коллекция плагинов, добавляющих пространство и атмосферу вашей музыке. Эти эффекты могут заставить вашу музыку звучать так, как будто она воспроизводится в различных помещениях, от уютных комнат до грандиозных концертных залов, улучшая ваше прослушивание естественной атмосферой и глубиной.

## Список плагинов

- [Dattorro Plate Reverb](#dattorro-plate-reverb) - Классический плейт-реверб на основе алгоритма Датторро
- [FDN Reverb](#fdn-reverb) - Сложный реверб с архитектурой сети обратной связи и диффузией Адамара
- [IR Reverb](#ir-reverb) - Свёрточная реверберация с импортированной импульсной характеристикой
- [RS Reverb](#rs-reverb) - Создает естественную атмосферу помещения и пространства

## Dattorro Plate Reverb

Классическая реализация плейт-реверба на основе знаменитого алгоритма Джона Датторро из статьи 1997 года "Effect Design, Part 1: Reverberator and Other Filters." Этот алгоритм известен своим богатым, мягким звучанием и стал эталоном в проектировании цифровых ревербов. Идеально подходит для добавления богатой, мерцающей атмосферы к вашей музыке.

Примечание по маршрутизации: Dattorro Plate Reverb — это стерео-модель плейт-реверба. При маршрутизации более чем двух каналов все маршрутизированные входные каналы подаются в одну общую plate-схему преобразования mono в stereo, но wet/dry mix записывается только в первую маршрутизированную стереопару. Дополнительные каналы участвуют во входе plate и в остальном проходят без изменений, даже когда Dry Mix равен 0%; они не получают wet return и не являются независимыми plate tanks.

### Руководство по прослушиванию
- Богатый звук плейта:
  - Классический характер плейт-реверба
  - Гладкий, плотный хвост реверба без металлических артефактов
  - Красивое мерцание и тепло, характерные для плейт-ревербов
- Универсальная атмосфера:
  - От тонкого улучшения помещения до обширных залов
  - Прекрасно работает с любым музыкальным жанром
  - Добавляет музыке гладкий блеск и пространство
- Естественное движение:
  - Модуляция добавляет органическую жизнь реверберации
  - Предотвращает статичные, искусственно звучащие хвосты
  - Создает дышащее, живое пространство вокруг вашей музыки

### Параметры
- **Pre Delay** - Начальная тишина перед началом реверба (управление 0.0-100.0 мс; для эффективного pre-delay используйте значения ниже 100.0 мс)
  - 0-10мс: Немедленный реверб, интимное ощущение
  - 10-30мс: Естественное ощущение пространства
  - 30-99.9мс: Создает впечатление больших пространств
  - Избегайте ровно 100.0мс, если нужен максимальный pre-delay; текущая реализация обрабатывает эту конечную точку как отсутствие эффективного pre-delay
- **Bandwidth** - Фильтрация входного сигнала (0.0 до 1.0)
  - Меньшие значения: Более темный, теплый входной тон
  - Большие значения (около 1.0): Более яркий, полночастотный вход
  - По умолчанию 0.9995: Оптимально по рекомендации Датторро
- **Input Diff 1** - Первая ступень входной диффузии (0.0 до 1.0)
  - Контролирует начальное размытие входного сигнала
  - По умолчанию 0.75: Рекомендованное значение из статьи Датторро
  - Большие значения: Более диффузные, мягкие ранние отражения
- **Input Diff 2** - Вторая ступень входной диффузии (0.0 до 1.0)
  - Дополнительно распределяет входной сигнал
  - По умолчанию 0.625: Рекомендованное значение из статьи Датторро
  - Работает с Input Diff 1 для создания сложной диффузии
- **Decay** - Продолжительность хвоста реверба (0.0 до 1.0)
  - Низкий (0.1-0.3): Короткое, контролируемое затухание
  - Средний (0.4-0.6): Естественное затухание типа помещения
  - Высокий (0.7-1.0): Длинные, расширенные хвосты
- **Decay Diff 1** - Диффузия затухания в резервуаре (0.0 до 1.0)
  - Контролирует плотность во время фазы затухания
  - По умолчанию 0.70: Рекомендованное значение из статьи Датторро
  - Влияет на гладкость хвоста реверба
- **Damping** - Поглощение высоких частот со временем (0.0 до 1.0)
  - 0.0: Без затухания, яркий реверб повсюду
  - 0.0005 (по умолчанию): Очень тонкое, естественное затухание
  - Большие значения: Более темное, теплое затухание
- **Mod Depth** - Количество модуляции задержки (0.0 до 16.0 сэмплов)
  - 0.0: Без модуляции, статический реверб
  - 1.0-4.0: Тонкое движение, добавляет жизнь
  - 8.0-16.0: Более заметный эффект типа хоруса
- **Mod Rate** - Скорость модуляции (0.0 до 10.0 Гц)
  - 0.5-1.5Гц: Медленное, мягкое движение
  - 2.0-4.0Гц: Более активная модуляция
  - Большие значения: Быстрый, мерцающий эффект
- **Wet Mix** - Количество добавляемого реверба (0 до 100%)
  - 10-30%: Тонкое улучшение
  - 30-50%: Заметное присутствие
  - 50-100%: Доминирующий эффект реверба
- **Dry Mix** - Количество оригинального сигнала (0 до 100%)
  - Обычно держится на 100% для нормального прослушивания
  - Уменьшить для специальных эффектов или эмбиент-подложек

### Рекомендуемые настройки для различных музыкальных стилей

1. Классическое фортепиано
   - Decay: 0.6-0.7
   - Damping: 0.001
   - Mod Depth: 1.0
   - Wet Mix: 25-35%
   - Идеально для: Сольное фортепиано, камерная музыка

2. Вокал и акустика
   - Decay: 0.4-0.5
   - Damping: 0.002
   - Pre Delay: 15-25мс
   - Wet Mix: 20-30%
   - Идеально для: Вокал, акустическая гитара

3. Эмбиент и атмосферный
   - Decay: 0.8-0.95
   - Mod Depth: 4.0-8.0
   - Mod Rate: 0.5-1.0Гц
   - Wet Mix: 50-70%
   - Идеально для: Эмбиент, электроника, звуковые ландшафты

4. Общее улучшение
   - Decay: 0.5
   - Damping: 0.0005
   - Mod Depth: 1.0
   - Wet Mix: 20-30%
   - Идеально для: Универсальное использование, тонкая отделка

### Руководство быстрого старта

1. Установите базовый характер
   - Начните с Decay для контроля длины реверба
   - Настройте Pre Delay для воспринимаемого расстояния
   - Установите Wet Mix для желаемого присутствия реверба

2. Сформируйте тональность
   - Используйте Bandwidth для контроля яркости входа
   - Настройте Damping для затухания высоких частот
   - Точно настройте параметры диффузии для плотности

3. Добавьте движение
   - Установите Mod Depth для тонкой вариации (попробуйте 1.0)
   - Настройте Mod Rate для скорости (попробуйте 1.0Гц)
   - Эти параметры добавляют жизнь реверберации

4. Финальный баланс
   - Настройте микс Wet/Dry по вкусу
   - Доверяйте своим ушам для финальных настроек
   - Значения по умолчанию - отличная отправная точка

Dattorro Plate Reverb добавляет к прослушиванию классическую plate-атмосферу. Его гладкий и насыщенный характер полезен, когда записи нужно красивое, естественно звучащее пространство.

## FDN Reverb

Сложный эффект реверберации, использующий архитектуру сети обратной связи с задержкой (FDN) с диффузионной матрицей Адамара. Этот алгоритм создает богатые, естественно звучащие пространства с превосходным контролем над характеристиками реверберации.

Примечание по маршрутизации: FDN Reverb — это стерео-модель реверберации с одним общим feedback tank. При маршрутизации более чем двух каналов каждый канал последовательно продвигает этот общий tank вместо независимых per-channel tanks. Канал 1 получает left wet, а каналы 2+ получают right wet.

### Руководство по прослушиванию
- Естественное ощущение помещения:
  - Создает реалистичные акустические пространства
  - Добавляет органичную глубину и размерность
  - Сохраняет четкость исходного сигнала
- Атмосферное улучшение:
  - Обогащает музыку пространственными характеристиками
  - Создает иммерсивные звуковые ландшафты
  - Идеально для создания настроения и атмосферы
- Настраиваемая атмосфера:
  - Гибкий контроль от интимных до просторных пространств
  - Точная настройка частотной характеристики
  - Адаптируется к любому музыкальному стилю

### Параметры
- **Reverb Time** - Продолжительность эффекта реверберации (0.20 до 10.00 с)
  - Короткое (0.2-1.0с): Быстрое, контролируемое затухание для четкости
  - Среднее (1.0-3.0с): Естественная комнатная реверберация
  - Длинное (3.0-10.0с): Расширенные, атмосферные хвосты
- **Density** - Количество путей эха для сложности (4 до 8 линий)
  - 4 линии: Более простые, четко определенные отдельные эхо
  - 6 линий: Хороший баланс сложности и четкости
  - 8 линий: Максимальная гладкость и плотность
- **Pre Delay** - Первоначальная тишина перед началом реверберации (0.0 до 100.0 мс)
  - 0-20мс: Немедленная реверберация, интимное ощущение
  - 20-50мс: Естественное ощущение расстояния помещения
  - 50-100мс: Создает впечатление больших пространств
- **Base Delay** - Основное время для сети реверберации (10.0 до 60.0 мс)
  - Меньшие значения: Более плотный, сфокусированный характер реверберации
  - Большие значения: Более просторное, открытое качество звука
  - Влияет на фундаментальные временные соотношения
- **Delay Spread** - Добавляет прогрессивный разброс времени между линиями задержки поверх небольших случайных смещений каждой линии (0.0 до 25.0 мс)
  - 0.0мс: Используется базовая задержка плюс небольшие рандомизированные смещения линий, поэтому отражения все равно остаются слегка нерегулярными
  - Большие значения: Добавляют больший прогрессивный разброс между линиями для более крупного и менее регулярного хвоста
  - Добавляет реалистичные вариации, найденные в реальных акустических пространствах
- **HF Damp** - Как высокие частоты затухают со временем (0.0 до 12.0 дБ/с)
  - 0.0: Без затухания, яркий звук на протяжении всего затухания
  - 3.0-6.0: Симуляция естественного поглощения воздуха
  - 12.0: Сильное затухание для теплого, мягкого характера
- **Low Cut** - Удаляет низкие частоты из реверберации (20 до 500 Гц)
  - 20-50Гц: Полная басовая реакция в реверберации
  - 100-200Гц: Контролируемые басы для избежания замутнения
  - 300-500Гц: Плотный, четкий низ
- **Mod Depth** - Количество питч-модуляции для эффекта хоруса (0.0 до 10.0 центов)
  - 0.0: Без модуляции, чистая статичная реверберация
  - 2.0-5.0: Тонкое движение, которое добавляет жизнь и реализм
  - 10.0: Заметный эффект, похожий на хорус
- **Mod Rate** - Скорость модуляции (0.10 до 5.00 Гц)
  - 0.1-0.5Гц: Очень медленное, нежное движение
  - 1.0-2.0Гц: Естественно звучащие вариации
  - 3.0-5.0Гц: Быстрая, более очевидная модуляция
- **Diffusion** - Управляет тем, какая часть смешанной обратной связи возвращается в сеть задержек (0 до 100%)
  - 0%: Отключает диффузию обратной связи; звук становится намного более разреженным, а хвост реверберации сильно сокращается
  - 50%: Сбалансированная диффузия для естественного звука
  - 100%: Максимальная диффузия обратной связи для самой гладкой плотности
- **Wet Mix** - Количество реверберации, добавленной к звуку (0 до 100%)
  - 10-30%: Тонкое пространственное улучшение
  - 30-60%: Заметное присутствие реверберации
  - 60-100%: Доминирующий эффект реверберации
- **Dry Mix** - Количество сохраненного оригинального сигнала (0 до 100%)
  - Обычно держится на 100% для нормального прослушивания
  - Может быть уменьшено для специальных атмосферных эффектов
- **Stereo Width** - Смешивает wet-реверберацию от моно к раздельным левому/правому wet-отводам (0 до 200%)
  - 0%: Wet-реверберация появляется в центре (моно)
  - 100%: Стандартная умеренная стереоширина wet-сигнала
  - 200%: Полное разделение левого/правого wet-отвода, а не дополнительное усиление боковой составляющей

### Рекомендуемые настройки для различных опытов прослушивания

1. Улучшение классической музыки
   - Reverb Time: 2.5-3.5с
   - Density: 8 линий
   - Pre Delay: 30-50мс
   - HF Damp: 4.0-6.0
   - Идеально для: Оркестровых записей, камерной музыки

2. Атмосфера джаз-клуба
   - Reverb Time: 1.2-1.8с
   - Density: 6 линий
   - Pre Delay: 15-25мс
   - HF Damp: 2.0-4.0
   - Идеально для: Акустического джаза, интимных выступлений

3. Улучшение поп/рока
   - Reverb Time: 1.0-2.0с
   - Density: 6-7 линий
   - Pre Delay: 10-30мс
   - Wet Mix: 20-40%
   - Идеально для: Современных записей, добавления пространства

4. Эмбиент-пейзажи
   - Reverb Time: 4.0-8.0с
   - Density: 8 линий
   - Mod Depth: 3.0-6.0
   - Wet Mix: 60-80%
   - Идеально для: Атмосферной музыки, релаксации

### Руководство быстрого старта

1. Установите базовое пространство
   - Начните с Reverb Time для размера пространства
   - Установите Pre Delay для типа помещения
   - Настройте Density для характера отражений

2. Сформируйте тональность
   - Используйте HF Damp для контроля яркости
   - Настройте Low Cut для четкости низких частот
   - Установите Diffusion для гладкости

3. Добавьте движение и финальную настройку
   - Настройте Mod Depth и Rate для живости
   - Сбалансируйте Wet/Dry Mix
   - Настройте Stereo Width для пространственности

4. Точная настройка
   - Настройте Base Delay и Delay Spread для уникального характера
   - Доверяйте своим ушам и настраивайте по вкусу
   - Начинайте с пресетов и адаптируйте под свои нужды

FDN Reverb добавляет к музыке насыщенное акустическое пространство. Он подходит для слушателей, которые хотят придать любимым трекам красивую, естественно звучащую реверберацию без потери ясности.

## IR Reverb

IR Reverb выполняет свёртку сигнала с импортированной импульсной характеристикой (IR), воспроизводя измеренное затухание и пространственный характер комнаты, зала, пластины или другой акустической системы. Он полезен, когда нужен воспроизводимый характер конкретной записи.

### Руководство по улучшению звучания

- Для лёгкой комнатной атмосферы возьмите короткую IR, задайте **Dry** 0 dB, **Wet** от -18 до -12 dB и небольшой **Pre Delay**.
- Для большого зала используйте стерео- или True Stereo IR, а слишком длинный хвост сократите параметрами **Decay** и **Trim**.
- В схеме send/return скопируйте источник на другую шину через **Matrix**, задайте **Dry** -96 dB, оставьте **Wet** на 0 dB и регулируйте эффект уровнем посыла.
- Для точного воспроизведения сохраните исходный IR-файл, источник и лицензию: одинаковые байты дают одинаковый ID.

### Параметры

- **Channel Mode**: Auto, Mono, Independent, True Stereo (маршруты LL/LR/RL/RR) или Diagonal Matrix без перекрёстной связи.
- **Latency**: Zero либо 128/256/512/1024 samples. Большие значения снижают нагрузку, но задерживают wet-сигнал; Zero требует Full.
- **Convolution Rate**: Auto, Full, Half или Quarter. Понижение частоты уменьшает нагрузку и полосу wet-сигнала; Quarter требует не менее 176,4 kHz.
- **Dry**: задаёт уровень исходного сигнала. Значение -96 dB полностью его заглушает для полностью wet-эффекта или возврата.
- **Wet**: уровень свёрнутого сигнала от -96 до +12 dB.
- **Pre Delay**: задерживает только wet-сигнал на 0–500 ms.
- **Direct Cut** удаляет найденный прямой импульс; **Cut Offset** смещает точку среза на -20…+50 ms. Для нормализации по-прежнему используется IR до среза, поэтому включение Direct Cut не усиливает оставшийся хвост реверберации.
- **Decay** меняет затухание от 10% до 400%; 100% сохраняет исходную запись.
- **Trim** оставляет 1–100% IR после среза; короткий хвост требует меньше CPU и памяти.

### Как читать график затухания

Время идёт слева направо, уровень — от 0 до -90 dB. Сплошная EDC показывает потерю энергии; крутой спад означает короткий хвост. Метки обозначают onset, cut, pre-delay и trim. RT60 оценивает время спада на 60 dB. При изменении **Decay** новая кривая сплошная, исходная — пунктирная.

### Маршрутизация, библиотека и обмен

Mono применяет одну IR, Independent сохраняет каналы раздельными, True Stereo использует LL/LR/RL/RR, а Diagonal Matrix соединяет только соответствующие входы и выходы. Для пары True Stereo выберите вместе файлы с согласованными окончаниями `L`/`R` или `Left`/`Right`.

Оригиналы хранятся в **Impulse Response Library**. Веб-версия использует приватное OPFS сайта, настольная — управляемое хранилище приложения. Библиотека показывает исходные имена файлов и позволяет искать по ним, загружать или удалять записи. При смене sample rate данные готовятся заново из оригинала. Храните отдельную копию: очистка данных сайта или нехватка места может удалить данные браузера.

URL и пресеты содержат только ID, но не звук IR и не имя файла. Получателю нужно импортировать тот же файл или выбрать замену. Без IR нет wet-сигнала; без WASM проходит только настроенный dry-сигнал. Материалы доступны в [OpenAIR](https://www.openair.hosted.york.ac.uk/), [EchoThief](https://www.echothief.com/downloads/) и [Freesound](https://freesound.org/), однако проверяйте лицензию каждого файла (например, CC0, CC BY или CC BY-NC) и сохраняйте автора, источник, требования атрибуции и разрешение на коммерческое использование вне EffeTune: приложение не хранит и не проверяет сведения о лицензии.

## RS Reverb

Эффект, способный перенести вашу музыку в различные пространства, от уютных комнат до величественных залов. Он добавляет естественные эхо и отражения, которые делают вашу музыку более трехмерной и иммерсивной.

### Руководство по прослушиванию
- Интимное пространство:
  - Создает ощущение, будто музыка звучит в теплой, уютной комнате
  - Идеально для близкого, личного прослушивания
  - Добавляет тонкую глубину без потери четкости
- Опыт концертного зала:
  - Воссоздает величие живых выступлений
  - Добавляет величественное пространство классической и оркестровой музыке
  - Создает иммерсивный концертный опыт
- Атмосферное улучшение:
  - Добавляет мечтательные, эфирные качества
  - Идеально для эмбиент и атмосферной музыки
  - Создает захватывающие звуковые ландшафты

### Параметры
- **Pre-Delay** - Сохраняемое и отображаемое значение управления (от 0 до 50 ms)
  - В текущей реализации это значение не используется обработкой реверберации
  - Его изменение не меняет дистанцию или глубину; для слышимых изменений пространства используйте Room Size, Reverb Time и Mix
- **Room Size** - Устанавливает, насколько большим ощущается пространство (от 2.0 до 50.0 m)
  - Маленький (2-5m): Ощущение уютной комнаты
  - Средний (5-15m): Атмосфера живого помещения
  - Большой (15-50m): Величие концертного зала
- **Reverb Time** - Как долго длится эхо (от 0.1 до 10.0 s)
  - Короткое (0.1-1.0s): Четкий, сфокусированный звук
  - Среднее (1.0-3.0s): Естественный звук помещения
  - Длинное (3.0-10.0s): Просторное, атмосферное
- **Density** - Насколько богатым ощущается пространство (от 4 до 8)
  - Меньшие значения: Более определенные эхо
  - Большие значения: Более гладкая атмосфера
  - Начните с 6 для естественного звука
- **Diffusion** - Как распространяется звук (от 0.2 до 0.8)
  - Меньшие значения: Более различимые эхо
  - Большие значения: Более гладкое смешение
  - Попробуйте 0.5 для сбалансированного звука
- **Damping** - Как затухает эхо (от 0 до 100%)
  - Меньшие значения: Более яркий, открытый звук
  - Большие значения: Более теплый, интимный
  - Начните около 40% для естественного ощущения
- **High Damp** - Контролирует яркость пространства (от 1000 до 20000 Hz)
  - Меньшие значения: Более темное, теплое пространство
  - Большие значения: Более яркое, открытое
  - Начните около 8000Hz для естественного звука
- **Low Damp** - Контролирует полноту пространства (от 20 до 500 Hz)
  - Меньшие значения: Более полный, богатый звук
  - Большие значения: Более четкий, контролируемый
  - Начните около 100Hz для сбалансированного баса
- **Mix** - Балансирует эффект с оригинальным звуком (от 0 до 100%)
  - 10-30%: Тонкое улучшение
  - 30-50%: Заметное пространство
  - 50-100%: Драматический эффект

### Рекомендуемые настройки для различных музыкальных стилей

1. Классическая музыка в концертном зале
   - Room Size: 30-40m
   - Reverb Time: 2.0-2.5s
   - Mix: 30-40%
   - Идеально для: Оркестровых произведений, фортепианных концертов

2. Уютный джаз-клуб
   - Room Size: 8-12m
   - Reverb Time: 1.0-1.5s
   - Mix: 20-30%
   - Идеально для: Джаза, акустических выступлений

3. Современный поп/рок
   - Room Size: 15-20m
   - Reverb Time: 1.2-1.8s
   - Mix: 15-25%
   - Идеально для: Современной музыки

4. Эмбиент/электроника
   - Room Size: 25-40m
   - Reverb Time: 3.0-6.0s
   - Mix: 40-60%
   - Идеально для: Атмосферной электронной музыки

### Руководство быстрого старта

1. Выберите ваше пространство
   - Начните с Room Size для установки базового пространства
   - Настройте Reverb Time для желаемой атмосферы
   - Точно настройте Mix для правильного баланса

2. Сформируйте звук
   - Используйте Damping для контроля теплоты
   - Настройте High/Low Damp для тона
   - Установите Density и Diffusion для текстуры

3. Точная настройка эффекта
   - Для глубины используйте Room Size и Reverb Time; значение Pre-Delay сейчас не отражается в обработке
   - Настройте Mix для финального баланса
   - Доверяйте своим ушам и настраивайте по вкусу

Помните: Цель состоит в том, чтобы улучшить вашу музыку естественным пространством и атмосферой. Начните с тонких настроек и регулируйте, пока не найдете идеальный баланс для вашего прослушивания!
