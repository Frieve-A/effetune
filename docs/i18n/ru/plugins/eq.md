# Equalizer Plugins

Коллекция плагинов, позволяющих настраивать различные аспекты звучания вашей музыки, от глубоких басов до четких верхов. Эти инструменты помогают персонализировать ваше прослушивание, усиливая или ослабляя определенные элементы звука.

## Plugin List

- [15Band GEQ](#15band-geq) - Детальная настройка звука с 15 точными регуляторами
- [5Band PEQ](#5band-peq) - Профессиональный параметрический эквалайзер с гибкими настройками
- [Tone Control](#tone-control) - Простая регулировка басов, средних и высоких частот
- [Narrow Range](#narrow-range) - Фокусировка на определенных частях звука

## 15Band GEQ

Инструмент детальной настройки звука с 15 отдельными регуляторами, каждый из которых влияет на определенную часть звукового спектра. Идеально подходит для точной настройки музыки именно так, как вам нравится.

### Listening Enhancement Guide
- Басовый диапазон (25Hz-160Hz):
  - Усиление мощности бас-барабанов и глубоких басов
  - Настройка полноты басовых инструментов
  - Контроль сотрясающих комнату суб-басов
- Нижняя середина (250Hz-630Hz):
  - Настройка теплоты звучания музыки
  - Контроль полноты общего звучания
  - Уменьшение или усиление "плотности" звука
- Верхняя середина (1kHz-2.5kHz):
  - Делает вокал более четким и присутствующим
  - Настройка выраженности основных инструментов
  - Контроль "переднего" ощущения звука
- Высокие частоты (4kHz-16kHz):
  - Усиление четкости и детализации
  - Контроль "искристости" и "воздуха" в музыке
  - Настройка общей яркости

### Parameters
- **Band Gains** - Индивидуальные регуляторы для каждого частотного диапазона (от -12dB до +12dB)
  - Глубокий бас
    - 25Hz: Самое низкое басовое ощущение
    - 40Hz: Глубокий басовый удар
    - 63Hz: Мощность баса
    - 100Hz: Полнота баса
    - 160Hz: Верхний бас
  - Нижний звук
    - 250Hz: Теплота звука
    - 400Hz: Полнота звука
    - 630Hz: Тело звука
  - Средний звук
    - 1kHz: Присутствие основного звука
    - 1.6kHz: Четкость звука
    - 2.5kHz: Детализация звука
  - Высокий звук
    - 4kHz: Четкость звука
    - 6.3kHz: Блеск звука
    - 10kHz: Воздушность звука
    - 16kHz: Искристость звука

### Visual Display
- График в реальном времени, показывающий ваши настройки звука
- Удобные в использовании слайдеры с точным контролем
- Сброс к настройкам по умолчанию одним кликом

## Tone Control

Простой трехполосный регулятор звука для быстрой и легкой персонализации звучания. Идеально подходит для базовой настройки звука без излишних технических сложностей.

### Music Enhancement Guide
- Классическая музыка:
  - Легкое усиление верхов для большей детализации струнных
  - Мягкое усиление басов для более полного оркестрового звучания
  - Нейтральные средние частоты для естественного звука
- Рок/Поп музыка:
  - Умеренное усиление басов для большей ударности
  - Легкое снижение средних частот для более чистого звука
  - Усиление верхов для четких тарелок и деталей
- Джазовая музыка:
  - Теплый бас для более полного звучания
  - Четкие средние частоты для детализации инструментов
  - Мягкие верха для искристости тарелок
- Электронная музыка:
  - Сильный бас для глубокого воздействия
  - Пониженные средние частоты для более чистого звука
  - Усиленные верха для четких деталей

### Parameters
- **Bass** - Управляет низкими звуками (от -24dB до +24dB)
  - Увеличение для более мощного баса
  - Уменьшение для более легкого, чистого звука
  - Влияет на "вес" музыки
- **Mid** - Управляет основным телом звука (от -24dB до +24dB)
  - Увеличение для более выраженного вокала/инструментов
  - Уменьшение для более просторного звука
  - Влияет на "полноту" музыки
- **Treble** - Управляет высокими звуками (от -24dB до +24dB)
  - Увеличение для большей искристости и детализации
  - Уменьшение для более гладкого, мягкого звука
  - Влияет на "яркость" музыки

### Visual Display
- Легко читаемый график, показывающий ваши настройки
- Простые слайдеры для каждого регулятора
- Кнопка быстрого сброса

## 5Band PEQ

Профессиональный параметрический эквалайзер, основанный на научных принципах, предлагающий пять полностью настраиваемых полос с точным частотным контролем. Идеально подходит как для тонкой настройки звука, так и для корректирующей обработки аудио.

### Sound Enhancement Guide
- Четкость вокала и инструментов:
  - Используйте полосу 3.2kHz с умеренным Q (1.0-2.0) для естественного присутствия
  - Применяйте узкие вырезы с Q (4.0-8.0) для удаления резонансов
  - Добавьте мягкий воздух с помощью высокочастотной полки 10kHz (+2 до +4dB)
- Контроль качества баса:
  - Формируйте основы с помощью пикового фильтра 100Hz
  - Удаляйте резонансы помещения, используя узкий Q на конкретных частотах
  - Создавайте плавное расширение баса с помощью низкочастотной полки
- Научный подход к настройке звука:
  - Точное воздействие на конкретные частоты
  - Использование анализаторов для выявления проблемных областей
  - Применение измеренных коррекций с минимальным фазовым воздействием

### Technical Parameters
- **Precision-Engineered Bands**
  - Band 1: 100Hz (Контроль суб-баса и баса)
  - Band 2: 316Hz (Определение нижней середины)
  - Band 3: 1.0kHz (Присутствие средних частот)
  - Band 4: 3.2kHz (Детализация верхней середины)
  - Band 5: 10kHz (Расширение высоких частот)
- **Professional Controls Per Band**
  - Center Frequency: Логарифмически распределенные для оптимального покрытия
  - Gain Range: Точная регулировка ±18dB
  - Q Factor: От широкого 0.1 до точного 10.0
  - Multiple Filter Types:
    - Peaking: Симметричная частотная регулировка
    - Low/High Pass: Наклон 12dB/октава
    - Low/High Shelf: Мягкое спектральное формирование
    - Band Pass: Фокусированная частотная изоляция

### Technical Display
- Визуализация частотной характеристики высокого разрешения
- Интерактивные точки управления с точным отображением параметров
- Расчет передаточной функции в реальном времени
- Калиброванная сетка частот и усиления
- Точные числовые показания для всех параметров

## Narrow Range

Инструмент, позволяющий сфокусироваться на определенных частях музыки путем фильтрации нежелательных частот. Полезен для создания специальных звуковых эффектов или удаления нежелательных звуков.

### Listening Enhancement Guide
- Создание уникальных звуковых эффектов:
  - Эффект "телефонного голоса"
  - Звук "старого радио"
  - Эффект "под водой"
- Фокусировка на определенных инструментах:
  - Изоляция басовых частот
  - Фокус на вокальном диапазоне
  - Выделение конкретных инструментов
- Удаление нежелательных звуков:
  - Уменьшение низкочастотного гула
  - Срез чрезмерного высокочастотного шипения
  - Фокус на самых важных частях музыки

### Parameters
- **HPF Frequency** - Управляет тем, с какой частоты начинают уменьшаться низкие звуки (20Hz до 1000Hz)
  - Более высокие значения: Удаляют больше баса
  - Более низкие значения: Сохраняют больше баса
  - Начните с низких значений и настраивайте по вкусу
- **HPF Slope** - Определяет, насколько быстро уменьшаются низкие звуки (от 0 до -48 dB/октава)
  - 0dB: Без уменьшения (выключено)
  - От -6dB до -48dB: Постепенно усиливающееся уменьшение с шагом 6dB
- **LPF Frequency** - Управляет тем, с какой частоты начинают уменьшаться высокие звуки (200Hz до 20000Hz)
  - Более низкие значения: Удаляют больше высоких частот
  - Более высокие значения: Сохраняют больше высоких частот
  - Начните с высоких значений и при необходимости уменьшайте
- **LPF Slope** - Определяет, насколько быстро уменьшаются высокие звуки (от 0 до -48 dB/октава)
  - 0dB: Без уменьшения (выключено)
  - От -6dB до -48dB: Постепенно усиливающееся уменьшение с шагом 6dB
