# Frieve EffeTune <img src="../../../images/icon.png" alt="EffeTune Icon" width="30" heignt="30" align="bottom">

[Open App](https://frieve-a.github.io/effetune/effetune.html)

Веб-приложение для обработки аудио в реальном времени, разработанное для энтузиастов аудио, чтобы улучшить их опыт прослушивания музыки. EffeTune позволяет обрабатывать любой источник аудио через различные высококачественные эффекты, позволяя настраивать и совершенствовать ваше прослушивание в реальном времени.

[![Screenshot](../../../images/screenshot.png)](https://frieve-a.github.io/effetune/effetune.html)

## Concept

EffeTune создан для энтузиастов аудио, которые хотят поднять свой опыт прослушивания музыки на новый уровень. Независимо от того, слушаете ли вы потоковую музыку или воспроизводите с физических носителей, EffeTune позволяет добавлять профессиональные эффекты для настройки звука точно под ваши предпочтения. Превратите ваш компьютер в мощный процессор аудиоэффектов, который располагается между источником аудио и вашими колонками или усилителем.

Никаких аудиофильских мифов, только чистая наука.

## Features

- Обработка аудио в реальном времени
- Интерфейс drag-and-drop для создания цепочек эффектов
- Расширяемая система плагинов с категоризированными эффектами
- Визуализация аудио в реальном времени
- Аудио конвейер, который можно модифицировать в реальном времени
- Офлайн обработка аудиофайлов с использованием текущей цепочки эффектов

## Setup Guide

Перед использованием EffeTune вам нужно настроить маршрутизацию аудио. Вот как настроить различные источники аудио:

### Streaming Service Setup

Для обработки аудио из стриминговых сервисов (Spotify, YouTube Music и т.д.):

1. Prerequisites:
   - Установите виртуальное аудиоустройство (например, VB Cable, Voice Meeter или ASIO Link Tool)
   - Настройте ваш стриминговый сервис на вывод аудио через виртуальное аудиоустройство

2. Configuration:
   - Запустите EffeTune
   - Выберите виртуальное аудиоустройство как источник входа
   - Начните воспроизведение музыки из вашего стримингового сервиса
   - Убедитесь, что аудио проходит через EffeTune
   - Добавьте эффекты в Pipeline для улучшения вашего прослушивания

### Physical Audio Source Setup

Для использования EffeTune с CD-плеерами, сетевыми плеерами или другими физическими источниками:

1. Configuration:
   - Подключите ваш аудиоинтерфейс к компьютеру
   - Запустите EffeTune
   - Выберите ваш аудиоинтерфейс как источник входа
   - Настройте вывод аудио браузера на ваш аудиоинтерфейс
   - Ваш аудиоинтерфейс теперь функционирует как мультиэффект-процессор:
     * Input: Ваш CD-плеер, сетевой плеер или другой источник аудио
     * Processing: Эффекты реального времени через EffeTune
     * Output: Обработанное аудио на ваш усилитель или колонки

## Usage

### Building Your Effect Chain

1. Доступные плагины перечислены в левой части экрана
2. Перетаскивайте плагины из списка в область Effect Pipeline
3. Плагины обрабатываются в порядке сверху вниз
4. Используйте ручку (⋮) для изменения порядка плагинов перетаскиванием
5. Нажмите на название плагина для развертывания/свертывания его настроек
6. Используйте кнопку ON/OFF для обхода отдельных эффектов
7. Удаляйте плагины с помощью значка корзины

### Использование пресетов

1. Сохранение цепочки эффектов:
   - Настройте желаемую цепочку эффектов и параметры
   - Введите имя в поле пресета
   - Нажмите кнопку Сохранить для сохранения пресета

2. Загрузка пресета:
   - Введите или выберите имя пресета из выпадающего списка
   - Пресет будет загружен автоматически
   - Все плагины и их настройки будут восстановлены

3. Удаление пресета:
   - Выберите пресет, который хотите удалить
   - Нажмите кнопку Удалить
   - Подтвердите удаление при запросе

4. Информация о пресетах:
   - Каждый пресет хранит полную конфигурацию цепочки эффектов
   - Включает порядок плагинов, параметры и состояния

### Plugin Selection and Keyboard Shortcuts

1. Методы выбора плагинов:
   - Нажмите на заголовки плагинов для выбора отдельных плагинов
   - Удерживайте Ctrl при нажатии для выбора нескольких плагинов
   - Нажмите на пустое пространство в области Pipeline для отмены выбора всех плагинов

2. Keyboard Shortcuts:
   - Ctrl + A: Выбрать все плагины в Pipeline
   - Ctrl + C: Копировать выбранные плагины
   - Ctrl + V: Вставить плагины из буфера обмена
   - ESC: Отменить выбор всех плагинов

3. Plugin Documentation:
   - Нажмите кнопку ? на любом плагине, чтобы открыть его подробную документацию в новой вкладке

### Обработка аудиофайлов

1. Область для перетаскивания файлов:
    - Специальная область всегда видна под областью Effect Pipeline
    - Поддерживает одиночные или множественные аудиофайлы
    - Файлы обрабатываются с использованием текущих настроек Pipeline
    - Вся обработка выполняется на частоте дискретизации Pipeline

2. Статус обработки:
    - Индикатор прогресса показывает текущий статус обработки
    - Время обработки зависит от размера файла и сложности цепочки эффектов

3. Варианты загрузки:
    - Одиночные файлы загружаются в формате WAV
    - Множественные файлы автоматически упаковываются в ZIP-файл

### Sharing Effect Chains

Вы можете поделиться вашей конфигурацией цепочки эффектов с другими пользователями:
1. После настройки желаемой цепочки эффектов нажмите кнопку "Share" в правом верхнем углу области Effect Pipeline
2. URL будет автоматически скопирован в ваш буфер обмена
3. Поделитесь скопированным URL с другими - они смогут воссоздать вашу точную цепочку эффектов, открыв его
4. Все настройки эффектов хранятся в URL, что делает их легкими для сохранения и обмена

### Audio Reset

Если вы испытываете проблемы с аудио (выпадения, глитчи):
1. Нажмите кнопку "Reset Audio" в левом верхнем углу
2. Аудио конвейер будет перестроен автоматически
3. Ваша конфигурация цепочки эффектов будет сохранена

## Common Effect Combinations

Вот некоторые популярные комбинации эффектов для улучшения вашего прослушивания:

### Headphone Enhancement
1. Stereo Blend -> RS Reverb
   - Stereo Blend: Настраивает стерео ширину для комфорта (90-110%)
   - RS Reverb: Добавляет тонкую атмосферу помещения (10-20% mix)
   - Результат: Более естественное, менее утомительное прослушивание в наушниках

### Vinyl Simulation
1. Wow Flutter -> Noise Blender -> Simple Tube
   - Wow Flutter: Добавляет мягкие колебания высоты тона
   - Noise Blender: Создает виниловую атмосферу
   - Simple Tube: Добавляет аналоговую теплоту
   - Результат: Аутентичный опыт виниловой пластинки

### FM Radio Style
1. Multiband Compressor -> 5Band PEQ -> Hard Clipping
   - Multiband Compressor: Создает тот "радио" звук
   - 5Band PEQ: Улучшает присутствие и четкость
   - Hard Clipping: Добавляет тонкую теплоту
   - Результат: Профессиональный звук вещания

### Lo-Fi Character
1. Bit Crusher -> Simple Jitter -> RS Reverb
   - Bit Crusher: Уменьшает битовую глубину для ретро-ощущения
   - Simple Jitter: Добавляет цифровые несовершенства
   - RS Reverb: Создает атмосферное пространство
   - Результат: Классическая lo-fi эстетика

## Troubleshooting

### Audio Issues
1. Выпадения или глитчи
   - Нажмите "Reset Audio" для перестройки аудио конвейера
   - Попробуйте уменьшить количество активных эффектов
   - Закройте другие вкладки браузера, использующие аудио

2. Высокая загрузка CPU
   - Отключите эффекты, которые вы активно не используете
   - Рассмотрите использование меньшего количества эффектов в цепочке

### Common Setup Issues
1. Нет входного аудио
   - Проверьте выбор входного устройства в EffeTune
   - Проверьте разрешения микрофона в браузере
   - Убедитесь, что аудио воспроизводится из вашего источника

2. Эффект не работает
   - Убедитесь, что эффект включен (кнопка ON/OFF)
   - Проверьте настройки параметров
   - Попробуйте удалить и добавить эффект заново

3. Проблемы с обменом
   - Используйте кнопку "Share" для генерации URL
   - Копируйте весь URL при обмене
   - Проверьте общий ссылку в новом окне браузера

## FAQ

Q. Поддерживает ли это приложение объемный звук?
A. В настоящее время, из-за ограничений браузера, мы не можем обрабатывать более 2 каналов в браузере, и нет проверенного опыта работы с объемным звуком. Хотя сама реализация плагинов поддерживает объемный звук, нам нужно дождаться будущей поддержки браузеров.

Q. Какая рекомендуемая длина цепочки эффектов?
A. Хотя строгого ограничения нет, мы рекомендуем держать вашу цепочку эффектов в пределах 8-10 эффектов для оптимальной производительности. Более сложные цепочки могут повлиять на производительность системы.

Q. Могу ли я сохранять мои любимые комбинации эффектов?
A. Да! Используйте кнопку "Share" для генерации URL, содержащего всю конфигурацию вашей цепочки эффектов. Добавьте этот URL в закладки, чтобы сохранить ваши настройки.

Q. Как достичь наилучшего качества звука?
A. Используйте частоту дискретизации 96kHz когда возможно, начинайте с тонких настроек эффектов и стройте вашу цепочку постепенно. Следите за уровнями, чтобы избежать искажений.

Q. Будет ли это работать с любым источником аудио?
A. Да, EffeTune может обрабатывать любое аудио, воспроизводимое через ваше выбранное входное устройство, включая стриминговые сервисы, локальные файлы и физические носители.

## Technical Information

### Browser Compatibility

Frieve EffeTune был протестирован и проверен на работу в Google Chrome. Приложение требует современный браузер с поддержкой:
- Web Audio API
- Audio Worklet
- getUserMedia API
- Drag and Drop API

### Browser Support Details
1. Chrome/Chromium
   - Полностью поддерживается и рекомендуется
   - Обновите до последней версии для лучшей производительности

2. Firefox/Safari
   - Ограниченная поддержка
   - Некоторые функции могут работать не так, как ожидается
   - Рассмотрите использование Chrome для лучшего опыта

### Recommended Sample Rate

Для оптимальной производительности с нелинейными эффектами рекомендуется использовать EffeTune с частотой дискретизации 96kHz или выше. Эта более высокая частота дискретизации помогает достичь идеальных характеристик при обработке аудио через нелинейные эффекты, такие как насыщение и компрессия.
