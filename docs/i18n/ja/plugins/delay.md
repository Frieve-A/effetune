# ディレイプラグイン

音のタイミングを調整したり、はっきりした反復音を加えたりするためのツール集です。再生タイミングの微調整、リズミカルなエコー、空間感や奥行きの追加に役立ちます。

## プラグイン一覧

- [Delay](#delay) - タイミング、音色、ステレオの広がりを調整してエコーを作ります
- [Time Alignment](#time-alignment) - スピーカーやリスニング位置に合わせて再生タイミングを微調整

## Delay

音にはっきりしたエコーを加えるエフェクトです。エコーの繰り返し間隔、消え方、スピーカー間での広がりを調整でき、音楽再生に控えめな奥行き、リズム感、または個性的な空間効果を加えられます。

### リスニング体験ガイド

- **控えめな奥行きと空間:**
  - 音をぼやけさせず、穏やかな空間感を加えます。
  - ボーカルやリード楽器を少し大きく、存在感のある音に感じさせることがあります。
  - 短めのDelay Sizeと低めのFeedback/Mixを使います。
- **リズムの強調:**
  - 音楽のテンポに合うエコーを作れます（耳で手動調整）。
  - 電子音楽、ドラム、ギターなどにグルーヴや動きを加えます。
  - 8分音符や4分音符に合うようなディレイ時間を、耳で探してみてください。
- **スラップバックエコー:**
  - ロックやカントリーのボーカル、ギターでよく使われる、とても短い単発エコーです。
  - パーカッシブで二重に聞こえる効果を加えます。
  - とても短いDelay Size（30-120ms）、Feedback 0%、中程度のMixを使います。
- **ステレオの広がり:**
  - Ping-Pongを使うと、エコーが左右のスピーカー間を跳ね返るように聞こえます。
  - より広く、引き込まれるステレオイメージを作れます。
  - 音をより動きのある面白い印象にできます。

### パラメータ

- **Pre-Delay (ms)** - 信号がエコーディレイに入る前に追加する時間です（0から100 ms）。最初のエコーはPre-Delay + Delay Sizeの後に聞こえます。
  - 低い値（0-20ms）: エコーパターンがほぼすぐに始まります。
  - 高い値（20-100ms）: エコーパターンの前にはっきりした間を作り、元の音から分離します。
- **Delay Size (ms)** - 各エコーの間隔です（1から5000 ms）。
  - 短い（1-100ms）: 厚み付けや「スラップバック」効果を作ります。
  - 中程度（100-600ms）: 標準的なエコー効果で、リズムの強調に向いています。
  - 長い（600ms以上）: 間隔の広い、はっきりしたエコーになります。
  - *ヒント:* 音楽に合わせてタップし、リズミカルに感じるディレイ時間を探してみてください。
- **Damping (%)** - 各エコーで高域と低域がどの程度薄れていくかを制御します（0から100%）。
  - 0%: エコーは元の音色を保ちます（明るめ）。
  - 50%: バランスの取れた自然な減衰。
  - 100%: エコーがすばやく暗く薄くなります（よりこもった印象）。
  - High/Low Dampと組み合わせて使います。
- **High Damp (Hz)** - エコーが明るさを失い始める周波数を設定します（20から20000 Hz）。
  - 低い値（例: 2000Hz）: エコーが早く暗くなります。
  - 高い値（例: 10000Hz）: エコーの明るさが長く残ります。
  - エコーの音色を整えるため、Dampingと合わせて調整します。
- **Low Damp (Hz)** - エコーが低域の厚みを失い始める周波数を設定します（20から20000 Hz）。
  - 低い値（例: 50Hz）: エコーに低音がより残ります。
  - 高い値（例: 500Hz）: エコーがより早く薄くなります。
  - エコーの音色を整えるため、Dampingと合わせて調整します。
  - 予測しやすく音色を整えるには、Low DampをHigh Dampより低くしておくのが基本です。値が交差した場合、処理側で内部的に順序を整えます。
- **Feedback (%)** - 聞こえるエコーの数、またはエコーの持続時間を決めます（0から99%）。
  - 0%: 1回だけエコーが聞こえます。
  - 10-40%: いくつかの分かりやすい繰り返し。
  - 40-70%: より長く、消えていくエコーの尾を作ります。
  - 70-99%: 非常に長い尾になり、自己発振に近づきます（注意して使ってください）。
- **Ping-Pong (%)** - エコーがステレオチャンネル間でどう跳ね返るかを制御します（0から100%）。ステレオ再生時のみ効果があります。
  - 0%: 通常のディレイ。左入力は左へ、右入力は右へエコーします。
  - 50%: モノ寄りのフィードバック。エコーはスピーカー中央にまとまります。
  - 100%: フルPing-Pong。エコーが左右のスピーカー間で交互に聞こえます。
  - 中間値では、さまざまな広がり方を作れます。
- **Mix (%)** - エコーの音量と元の音のバランスを調整します（0から100%）。
  - 0%: 効果なし。
  - 5-15%: 控えめな奥行きやリズム。
  - 15-30%: はっきり聞こえるエコー（始めやすい範囲）。
  - 30%以上: より強く目立つ効果。デフォルトは16%です。

### リスニング向け推奨設定

1.  **ボーカル/楽器に控えめな奥行き:**
    - Delay Size: 80-150ms
    - Feedback: 0-15%
    - Mix: 8-16%
    - Ping-Pong: 0%（少し広げたい場合は20-40%も試す）
    - Damping: 40-60%
2.  **リズムの強調（電子音楽/ポップ）:**
    - Delay Size: 耳でテンポに合わせる（例: 120-500ms）
    - Feedback: 20-40%
    - Mix: 15-25%
    - Ping-Pong: 0%または100%
    - Damping: 好みに合わせて調整（明るい反復には低め）
3.  **クラシックロック風スラップバック（ギター/ボーカル）:**
    - Delay Size: 50-120ms
    - Feedback: 0%
    - Mix: 15-30%
    - Ping-Pong: 0%
    - Damping: 20-40%
4.  **広いステレオエコー（アンビエント/パッド）:**
    - Delay Size: 300-800ms
    - Feedback: 40-60%
    - Mix: 20-35%
    - Ping-Pong: 70-100%
    - Damping: 50-70%（滑らかな余韻に）

### クイックスタートガイド

1.  **タイミングを決める:**
    - `Delay Size`で主なエコーのリズムを設定します。
    - `Feedback`でエコーの回数や長さを調整します。
    - `Pre-Delay`でエコーパターンが始まる前の追加の間を作ります。
2.  **音色を調整する:**
    - `Damping`、`High Damp`、`Low Damp`を組み合わせ、エコーが消えていくときの音色を整えます。Dampingは50%前後から始め、Damp周波数を調整します。
3.  **ステレオ位置を調整する（任意）:**
    - ステレオで聴いている場合は、`Ping-Pong`でエコーの広がりを試してください。
4.  **自然に混ぜる:**
    - `Mix`でエコーの音量を元の音楽と合わせます。低め（約16%）から始め、ちょうどよく感じるところまで上げます。

## Time Alignment

再生タイミングを少しだけ遅らせるエフェクトです。スピーカー距離の違いを補正したいときや、リスニング位置に届く音のタイミングを微調整したいときに使えます。

### 使用シーン
- スピーカーとリスニング位置の小さな距離差を補正する
- このプラグインを通るチャンネルのタイミングを微調整する
- 小さな遅延によってステレオイメージがより安定または自然に感じられるか確認する

### パラメータ
- **Delay** - このプラグインを通るチャンネルに適用する遅延時間を制御します（0から100 ms）
  - 0 ms: 遅延なし
  - 小さい値: スピーカー間のごく小さな到達時間差の補正に役立ちます
  - 高い値: よりはっきりしたタイミングのずれを作ります

### 推奨される使い方

1. スピーカー距離の補正
   - 片方のスピーカーやチャンネルがリスニング位置へ早く届く場合、小さな遅延を加えます
   - 中央定位のボーカルなど、中心にある音を聴きながら少しずつ調整します

2. リスニング位置の微調整
   - まずはごく小さい値から試します
   - センターの音像が安定し、音が自然に保たれるところで止めます

目的は、音楽をより楽しく聴けるようにすることです。コントロールを試しながら、お気に入りの音楽に面白さや奥行きを加えつつ、やりすぎない音を探してみてください。
