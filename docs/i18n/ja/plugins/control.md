# 制御エフェクト

制御エフェクトは、オーディオを直接処理するのではなく、他のエフェクトの動作を管理するための特殊なユーティリティです。複雑なエフェクトチェーンを整理し、扱いやすくするのに役立ちます。

## Section

Section エフェクトは複数のエフェクトをグループ化し、その部分のチェーン全体を1つのON/OFFトグルでバイパスしたり元に戻したりできます。各エフェクト自身のON/OFF設定は保持されます。

### 概要

- **名前**: Section
- **カテゴリ**: Control
- **説明**: エフェクトをグループ化し、セクション全体をまとめてバイパスしたり元に戻したりします

### パラメータ

| パラメータ | 説明 |
|-----------|------|
| Comment   | セクションの目的を示す名前または説明 |

### 使用方法

1. まとめて制御したいエフェクトグループの先頭に Section エフェクトを配置します
2. 「Comment」フィールドに、セクションの目的が分かる名前を入力します
3. Section エフェクトをOFFにすると、そのセクション内のエフェクトをバイパスします。ONに戻すと、各エフェクト自身のON/OFF状態を保ったままそのセクション内の処理を再び通します
4. Section エフェクトの後に置かれたエフェクトは、次の Section エフェクトが現れるまで、そのセクションの対象になります

### 応用例

- 関連するエフェクトをまとめる（例: 「EQ調整」「空間系エフェクト」）
- 簡単に切り替えられる別の処理チェーンを作る
- 複雑なエフェクトチェーンを分かりやすいまとまりに整理する
- エフェクトを削除せずに、一部のグループだけを一時的に無効化する
