# イコライザープラグイン

深い低音から鮮明な高音まで、音楽のサウンドの異なる側面を調整できるプラグインのコレクションです。これらのツールは、特定のサウンド要素を強調または抑制することで、リスニング体験をパーソナライズするのに役立ちます。

## プラグイン一覧

- [15Band GEQ](#15band-geq) - 15の精密なコントロールによる詳細なサウンド調整
- [5Band PEQ](#5band-peq) - 柔軟な制御を備えたプロフェッショナルなパラメトリックイコライザー
- [Tone Control](#tone-control) - シンプルな低音、中音、高音の調整
- [Narrow Range](#narrow-range) - サウンドの特定の部分に焦点を当てる

## 15Band GEQ

15の独立したコントロールを持つ詳細なサウンド調整ツールで、それぞれが音域の特定の部分に影響を与えます。音楽を好みの通りに微調整するのに最適です。

### リスニング向上ガイド
- 低音域(25Hz-160Hz):
  - バスドラムと深い低音のパワーを強調
  - ベース楽器の充実感を調整
  - 部屋を揺らすような超低音を制御
- 低中音域(250Hz-630Hz):
  - 音楽の温かみを調整
  - 全体的なサウンドの充実感を制御
  - サウンドの「厚み」を抑制または強調
- 高中音域(1kHz-2.5kHz):
  - ボーカルをよりクリアで存在感のあるものに
  - メイン楽器の突出度を調整
  - サウンドの「前方感」を制御
- 高音域(4kHz-16kHz):
  - クリスプさと細部を強調
  - 音楽の「輝き」と「空気感」を制御
  - 全体的な明るさを調整

### パラメータ
- **Band Gains** - 各周波数帯域の個別コントロール(-12dBから+12dB)
  - 超低音
    - 25Hz:最も低い低音の感覚
    - 40Hz:深い低音のインパクト
    - 63Hz:低音のパワー
    - 100Hz:低音の充実感
    - 160Hz:上部低音
  - 低音域
    - 250Hz:サウンドの温かみ
    - 400Hz:サウンドの充実感
    - 630Hz:サウンドのボディ
  - 中音域
    - 1kHz:メインサウンドのプレゼンス
    - 1.6kHz:サウンドの明瞭さ
    - 2.5kHz:サウンドの細部
  - 高音域
    - 4kHz:サウンドのクリスプさ
    - 6.3kHz:サウンドの輝き
    - 10kHz:サウンドの空気感
    - 16kHz:サウンドのスパークル

### 視覚的表示
- サウンド調整をリアルタイムで表示するグラフ
- 精密な制御が可能な使いやすいスライダー
- ワンクリックでデフォルト設定にリセット

## Tone Control

素早く簡単なサウンドのパーソナライズのための3バンドサウンドアジャスター。技術的な詳細に入り込まずに基本的なサウンドシェイピングを行うのに最適です。

### 音楽向上ガイド
- クラシック音楽:
  - 弦楽器の細部をより明確にするための軽い高音ブースト
  - オーケストラサウンドをより豊かにするための穏やかな低音ブースト
  - 自然なサウンドのためのニュートラルな中音
- ロック/ポップス音楽:
  - よりインパクトのある適度な低音ブースト
  - よりクリアなサウンドのための軽い中音カット
  - クリスプなシンバルと細部のための高音ブースト
- ジャズ音楽:
  - より豊かなサウンドのための温かい低音
  - 楽器の細部を明確にする中音
  - シンバルの輝きのための穏やかな高音
- 電子音楽:
  - 深いインパクトのための強い低音
  - よりクリーンなサウンドのための抑えめの中音
  - クリスプな細部のための強調された高音

### パラメータ
- **Bass** - 低音を制御(-24dBから+24dB)
  - 増加でよりパワフルな低音
  - 減少でより軽く、クリーンなサウンド
  - 音楽の「重み」に影響
- **Mid** - サウンドのメインボディを制御(-24dBから+24dB)
  - 増加でボーカル/楽器をより目立たせる
  - 減少でよりスペーシャスなサウンド
  - 音楽の「充実感」に影響
- **Treble** - 高音を制御(-24dBから+24dB)
  - 増加でより輝きと細部を強調
  - 減少でよりスムーズで柔らかいサウンド
  - 音楽の「明るさ」に影響

### 視覚的表示
- 調整を表示する見やすいグラフ
- 各コントロールのシンプルなスライダー
- クイックリセットボタン

## 5Band PEQ

科学的原理に基づいた、5つの完全に設定可能なバンドを持つプロフェッショナルグレードのパラメトリックイコライザーです。微妙なサウンドの洗練から補正的なオーディオ処理まで、幅広い用途に最適です。

### サウンド向上ガイド
- ボーカルと楽器の明瞭さ:
  - 自然なプレゼンスのために3.2kHzバンドを適度なQ(1.0-2.0)で使用
  - 共振を除去するために狭いQ(4.0-8.0)のカットを適用
  - 10kHzハイシェルフで穏やかな空気感を追加(+2から+4dB)
- 低音品質の制御:
  - 100Hzピーキングフィルターで基本音を整形
  - 特定の周波数で狭いQを使用して部屋の共振を除去
  - ローシェルフで滑らかな低音の拡張を作成
- 科学的なサウンド調整:
  - 特定の周波数を精密にターゲット
  - アナライザーを使用して問題のある領域を特定
  - 最小限の位相影響で測定された補正を適用

### 技術的パラメータ
- **精密設計されたバンド**
  - バンド1:100Hz(サブ & 低音制御)
  - バンド2:316Hz(低中域の定義)
  - バンド3:1.0kHz(中域のプレゼンス)
  - バンド4:3.2kHz(高中域の細部)
  - バンド5:10kHz(高周波の拡張)
- **バンドごとのプロフェッショナルコントロール**
  - 中心周波数:最適なカバレッジのための対数的な配置
  - ゲイン範囲:精密な±18dB調整
  - Qファクター:広い0.1から精密な10.0まで
  - 複数のフィルタータイプ:
    - Peaking:対称的な周波数調整
    - Low/High Pass:12dB/オクターブのスロープ
    - Low/High Shelf:穏やかなスペクトル整形
    - Band Pass:焦点を絞った周波数分離

### 技術的表示
- 高解像度の周波数応答可視化
- 精密なパラメータ表示を備えたインタラクティブな制御ポイント
- リアルタイムの伝達関数計算
- 校正された周波数とゲインのグリッド
- すべてのパラメータの正確な数値表示

## Narrow Range

不要な周波数をフィルタリングすることで、音楽の特定の部分に焦点を当てることができるツールです。特殊なサウンドエフェクトの作成や不要なサウンドの除去に便利です。

### リスニング向上ガイド
- ユニークなサウンドエフェクトの作成:
  - 「電話の声」エフェクト
  - 「古いラジオ」サウンド
  - 「水中」エフェクト
- 特定の楽器に焦点を当てる:
  - 低音域を分離
  - ボーカル帯域に焦点を当てる
  - 特定の楽器を強調
- 不要なサウンドの除去:
  - 低周波のランブルを低減
  - 過剰な高周波のヒスを低減
  - 音楽の最も重要な部分に焦点を当てる

### パラメータ
- **HPF Frequency** - 低音が減衰し始める周波数を制御します（20Hz～1000Hz）
  - 高い値：より多くの低音を除去します
  - 低い値：より多くの低音を保持します
  - 低い値から始めて、お好みに合わせて調整してください
- **HPF Slope** - 低音がどの程度速く減衰するかを制御します（0～-48 dB/オクターブ）
  - 0dB：減衰なし（オフ）
  - -6dB～-48dB：6dB刻みで徐々に強い減衰
- **LPF Frequency** - 高音が減衰し始める周波数を制御します（200Hz～20000Hz）
  - 低い値：より多くの高音を除去します
  - 高い値：より多くの高音を保持します
  - 高い値から始め、必要に応じて下げて調整してください
- **LPF Slope** - 高音がどの程度速く減衰するかを制御します（0～-48 dB/オクターブ）
  - 0dB：減衰なし（オフ）
  - -6dB～-48dB：6dB刻みで徐々に強い減衰

### 視覚的表示
- 周波数応答を表示する明確なグラフ
- 調整が簡単な周波数コントロール
- シンプルなスロープ選択ボタン
