# Basic Audio Plugins

音楽再生の基本的な要素を調整するための必須ツール集。これらのプラグインは音量やバランスなど、リスニング体験の基本的な側面を制御できます。

## プラグイン一覧

* [Channel Divider](#channel-divider) - 複数チャンネルにわたってオーディオを周波数帯域に分割する
* [DC Offset](#dc-offset) - 音が不均衡に感じるオーディオを修正するのに役立つ
* [Matrix](#matrix) - 柔軟にオーディオチャンネルをルーティングおよびミックスする
* [Mute](#mute) - オーディオ出力を無音にする
* [Polarity Inversion](#polarity-inversion) - ステレオ音楽の音質を改善する場合がある
* [Stereo Balance](#stereo-balance) - 音楽の左右バランスを調整する
* [Volume](#volume) - 音楽の再生音量を制御する

## Channel Divider

ステレオ信号を複数の周波数帯域に分割し、それぞれを異なる出力チャンネルにルーティングする専用ツールです。マルチチャンネルシステムやカスタムクロスオーバー構成に最適です。

このエフェクトを使用するにはデスクトップアプリを使用し、オーディオ設定で出力チャンネル数を4以上に設定し、エフェクトバスルーティングのチャンネルを「All」に設定する必要があります。

### 使用シーン

* マルチチャンネルオーディオ出力（4、6、8チャンネル）を使用する場合
* 周波数ベースのカスタムチャンネルルーティングを作成する場合
* 複数アンプまたは複数スピーカー構成の場合

### パラメータ

* **Band Count** - 生成する周波数帯域の数（2～4バンド）

  * 2バンド: 低域/高域に分割
  * 3バンド: 低域/中域/高域に分割
  * 4バンド: 低域/低中域/高中域/高域に分割

* **Crossover Frequencies** - 音声が各帯域に分割される周波数を定義する

  * F1: 最初のクロスオーバーポイント
  * F2: 2番目のクロスオーバーポイント（3バンド以上用）
  * F3: 3番目のクロスオーバーポイント（4バンド用）

* **Slopes** - 帯域分離の急峻さを制御する

  * オプション: 1オクターブあたり-12dB〜-96dB
  * より急峻なスロープはより明確な分離を提供する
  * 緩やかなスロープはより自然な遷移を実現する

### 技術ノート

* 最初の2つの入力チャンネルのみを処理する
* 出力チャンネル数は2の倍数である必要がある（4、6、8）
* 高品質なLinkwitz-Rileyクロスオーバーフィルターを使用
* 設定を容易にする視覚的な周波数応答グラフ

## DC Offset

音が不均衡または不自然に聞こえるオーディオを修正するのに役立つユーティリティです。ほとんどのリスナーは頻繁に使用する必要はありませんが、音がどうもしっくりこない場合に有用です。

### 使用シーン

* 音楽のバランスが異常に不均衡に聞こえる場合
* 片方のチャンネルが異常に大きく聞こえる場合
* 他のエフェクトが期待通りに動作しない場合

### パラメータ

* **Offset** - オーディオバランスを調整する（-1.0〜+1.0）

  * 0.0: 通常設定
  * 違和感がある場合に調整
  * 小さい調整が最適な場合が多い

## Matrix

入力チャンネルと出力チャンネル間でカスタム信号経路を作成できる強力なチャネルルーティングツールです。オーディオ信号の接続やミックスに完全な柔軟性をもたらします。

### 使用シーン

* チャンネル間でカスタムルーティングを作成する場合
* 特定の方法で信号をミックスまたは分割する必要がある場合
* チャンネル間の相互作用を使ったクリエイティブなサウンドデザインに

### 機能

* 最大8チャンネル対応の柔軟なルーティングマトリックス
* 任意の入出力ペア間の個別接続制御
* 各接続に対する位相反転オプション
* 直感的に設定できる視覚的マトリックスインターフェース

### 動作原理

* 各接続ポイントは入力行から出力列へのルーティングを表す
* 有効な接続はチャンネル間の信号の流れを可能にする
* 位相反転オプションは信号の極性を反転させる
* 1つの出力への複数の入力接続は混合される

### 実用的な応用例

* カスタムダウンミックスまたはアップミックス構成
* 特定のチャンネルの分離または結合
* チャンネル間の位相関係の作成
* 複雑なルーティング要件の解決

## Mute

バッファをゼロで埋めることで全オーディオ出力を無音にするシンプルなユーティリティです。オーディオ信号を即座にミュートするのに役立ちます。

### 使用シーン

* フェードなしで即座にオーディオを無音にする場合
* 無音箇所やポーズ中に
* 不要なノイズ出力を防ぐために

## Polarity Inversion

特定の状況でステレオ音楽の音質を改善できるツールです。オーディオ波形を「反転」させるようなもので、音質を改善できる可能性があります。

エフェクターの共通設定で処理するチャンネルを限定することで、特定のチャンネルのみの極性を反転することもできます。

### 使用シーン

* ステレオ音楽が「こもっている」または「違和感がある」場合
* 他のステレオエフェクトと組み合わせる場合
* ステレオイメージを改善しようとする場合

## Stereo Balance

左右のスピーカーやヘッドホン間で音楽の配分を調整できます。不均一なステレオの修正や好みのサウンド配置に最適です。

### リスニング強化ガイド

* 理想的なバランス:

  * 自然なステレオのための中央位置
  * 両耳で同じ音量
  * ほとんどの音楽に最適
* 調整後のバランス:

  * 部屋の音響特性を補正
  * 聴力の違いを調整
  * 好みのサウンドステージを作成

### パラメータ

* **Balance** - 左右の配分を制御する（-100%〜+100%）

  * 中央（0%）：両側で同じ音量
  * 左（-100%）：左側の音量が多くなる
  * 右（+100%）：右側の音量が多くなる

### ビジュアル表示

* 使いやすいスライダー
* 明確な数値表示
* ステレオ位置を示す視覚的インジケーター

### 推奨使用例

1. 一般的なリスニング

   * バランスを中央（0%）に保つ
   * ステレオの不均一感を感じたら調整する
   * 微調整を行う

2. ヘッドフォンリスニング

   * 快適さのために微調整する
   * 聴力の違いを補正する
   * 好みのステレオイメージを作成する

3. スピーカーリスニング

   * 部屋の環境に合わせて調整する
   * 聴取位置に合わせたバランスに設定する
   * 部屋の音響特性を補正する

## Volume

音楽の再生音量を調整できる、シンプルかつ基本的なコントロールです。さまざまなシチュエーションで適切なリスニングレベルを見つけるのに最適です。

### リスニング強化ガイド

* 様々なリスニングシナリオに合わせて調整：

  * 仕事中のバックグラウンドミュージック
  * 集中リスニングセッション
  * 深夜の静かなリスニング
* 聴きやすいレベルに保ち、以下を防ぐ：

  * 聴取疲労
  * 音の歪み
  * 聴覚への潜在的ダメージ

### パラメータ

* **Volume** - 全体の音量を制御する（-60dB〜+24dB）

  * 下限値：再生音量が小さくなる
  * 上限値：再生音量が大きくなる
  * 0dB：元の音量レベル

覚えておいてください：これらの基本コントロールが良いサウンドの基礎です。より複雑なエフェクトを使用する前に、まずこれらの調整を行いましょう！
