# リバーブプラグイン

音楽に空間と雰囲気を加えるプラグインのコレクションです。これらのエフェクトは、親密な部屋から壮大なコンサートホールまで、様々な環境で演奏されているかのように音楽を響かせ、自然な空気感と奥行きでリスニング体験を向上させます。

## プラグイン一覧

- [Dattorro Plate Reverb](#dattorro-plate-reverb) - Dattorroアルゴリズムに基づくクラシックなプレートリバーブ
- [FDN Reverb](#fdn-reverb) - 高度な拡散マトリックスを使用したフィードバック・ディレイ・ネットワーク・リバーブ
- [RS Reverb](#rs-reverb) - 自然な室内の空気感と空間を作成

## Dattorro Plate Reverb

Jon Dattorroの1997年の論文「Effect Design, Part 1: Reverberator and Other Filters」に基づいた伝統的なプレートリバーブの実装です。このアルゴリズムは、その豊かで滑らかな音質で高く評価されており、デジタルリバーブ設計の基準となっています。音楽に豊かできらめくアンビエンスを追加するのに最適です。

### リスニング体験ガイド
- 豊かなプレートサウンド:
  - クラシックなスタジオプレートリバーブの特性
  - 金属的なアーティファクトのない滑らかで密度の高いリバーブテール
  - プレートリバーブ特有の美しいシマーと温かみ
- 万能なアンビエンス:
  - 微妙な空間強調から広大なホールまで
  - あらゆる音楽ジャンルに美しくマッチ
  - 録音にプロフェッショナルな仕上げを追加
- 自然な動き:
  - モジュレーションがリバーブに有機的な生命感を追加
  - 静的で人工的なテールを防止
  - 音楽の周りに呼吸するような生きた空間を作成

### パラメータ
- **Pre Delay** - リバーブ開始前の初期無音（0.0から100.0 ms）
  - 0-10ms: 即座のリバーブ、親密な感覚
  - 10-30ms: 自然な空間感
  - 30-100ms: より大きな空間の印象を作成
- **Bandwidth** - 入力信号のフィルタリング（0.0から1.0）
  - 低い値: より暗く温かい入力音色
  - 高い値（1.0に近い）: より明るくフル周波数の入力
  - デフォルト0.9995: Dattorroが推奨する最適値
- **Input Diff 1** - 第1段階入力拡散（0.0から1.0）
  - 入力信号の初期拡散を制御
  - デフォルト0.75: Dattorro論文からの推奨値
  - 高い値: より拡散した滑らかな初期反射
- **Input Diff 2** - 第2段階入力拡散（0.0から1.0）
  - 入力信号をさらに拡散
  - デフォルト0.625: Dattorro論文からの推奨値
  - Input Diff 1と連携して複雑な拡散を作成
- **Decay** - リバーブテールの持続時間（0.0から1.0）
  - 低い（0.1-0.3）: 短く制御された減衰
  - 中程度（0.4-0.6）: 自然な部屋のような減衰
  - 高い（0.7-1.0）: 長く広がりのあるテール
- **Decay Diff 1** - タンク内の減衰拡散（0.0から1.0）
  - 減衰フェーズ中の密度を制御
  - デフォルト0.70: Dattorro論文からの推奨値
  - リバーブテールの滑らかさに影響
- **Damping** - 時間経過による高周波吸収（0.0から1.0）
  - 0.0: ダンピングなし、全体的に明るいリバーブ
  - 0.0005（デフォルト）: 非常に微妙で自然なダンピング
  - 高い値: より暗く温かい減衰
- **Mod Depth** - ディレイモジュレーションの量（0.0から16.0サンプル）
  - 0.0: モジュレーションなし、静的なリバーブ
  - 1.0-4.0: 微妙な動き、生命感を追加
  - 8.0-16.0: より顕著なコーラスのような効果
- **Mod Rate** - モジュレーションの速度（0.0から10.0 Hz）
  - 0.5-1.5Hz: ゆっくりとした穏やかな動き
  - 2.0-4.0Hz: より活発なモジュレーション
  - 高い値: 速く、きらめくような効果
- **Wet Mix** - 追加されるリバーブの量（0から100%）
  - 10-30%: 微妙な強調
  - 30-50%: 顕著な存在感
  - 50-100%: 支配的なリバーブ効果
- **Dry Mix** - 原音信号の量（0から100%）
  - 通常のリスニングでは通常100%に保持
  - 特殊効果やアンビエントウォッシュでは削減

### 異なる音楽スタイルでの推奨設定

1. クラシックピアノ
   - Decay: 0.6-0.7
   - Damping: 0.001
   - Mod Depth: 1.0
   - Wet Mix: 25-35%
   - 最適な用途: ソロピアノ、室内楽

2. ボーカルとアコースティック
   - Decay: 0.4-0.5
   - Damping: 0.002
   - Pre Delay: 15-25ms
   - Wet Mix: 20-30%
   - 最適な用途: ボーカル、アコースティックギター

3. アンビエントと雰囲気重視
   - Decay: 0.8-0.95
   - Mod Depth: 4.0-8.0
   - Mod Rate: 0.5-1.0Hz
   - Wet Mix: 50-70%
   - 最適な用途: アンビエント、エレクトロニック、サウンドスケープ

4. 汎用強調
   - Decay: 0.5
   - Damping: 0.0005
   - Mod Depth: 1.0
   - Wet Mix: 20-30%
   - 最適な用途: オールラウンドな使用、微妙な仕上げ

### クイックスタートガイド

1. 基本的な特性の設定
   - Decayでリバーブの長さを制御
   - Pre Delayで知覚される距離を調整
   - Wet Mixで希望するリバーブの存在感を設定

2. 音色の形成
   - Bandwidthで入力の明るさを制御
   - Dampingで高周波数の減衰を調整
   - 拡散パラメータで密度を微調整

3. 動きの追加
   - 微妙な変化のためにMod Depthを設定（1.0を試す）
   - 速度のためにMod Rateを調整（1.0Hzを試す）
   - これらのパラメータがリバーブに生命感を追加

4. 最終的なバランス
   - 好みに合わせてWet/Dryミックスを調整
   - 最終設定は耳を信頼
   - デフォルト値は素晴らしい出発点

Dattorro Plate Reverbは、クラシックでプロフェッショナル品質のリバーブをリスニング体験にもたらします。その滑らかで豊かな特性は、美しく自然に響くアンビエンスであらゆる録音を強調するのに最適です！

## FDN Reverb

アダマール拡散マトリックスを使用したフィードバック・ディレイ・ネットワーク（FDN）アーキテクチャをベースとした洗練されたリバーブエフェクトです。優れた密度と自然な減衰特性を持つ豊かで複雑な残響を作り出し、没入感のある空間エフェクトでリスニング体験を向上させるのに最適です。

### リスニング体験ガイド
- 自然な部屋の感覚:
  - リアルな音響空間にいる感覚を作り出す
  - 音楽に奥行きと立体感を追加
  - ステレオ録音をより広々として生き生きとした感じにする
- 雰囲気の強調:
  - 平坦な録音を没入感のある体験に変える
  - 音楽的な音符に美しいテールとサステインを追加
  - 演奏空間にいるような感覚を作り出す
- カスタマイズ可能な空気感:
  - 親密な部屋から壮大なコンサートホールまで調整可能
  - 空間の特性と色合いの細かい制御
  - 穏やかなモジュレーションが自然な動きと生命感を追加

### パラメータ
- **Reverb Time** - リバーブエフェクトの持続時間（0.20から10.00秒）
  - 短い（0.2-1.0秒）: 素早く制御された減衰で明瞭性を保持
  - 中程度（1.0-3.0秒）: 自然な部屋のような残響
  - 長い（3.0-10.0秒）: 広がりのある雰囲気的なテール
- **Density** - 複雑さのためのエコーパスの数（4から8ライン）
  - 4ライン: よりシンプルで明確な個別エコー
  - 6ライン: 複雑さと明瞭さの良いバランス
  - 8ライン: 最大の滑らかさと密度
- **Pre Delay** - リバーブ開始前の初期無音（0.0から100.0ミリ秒）
  - 0-20ms: 即座のリバーブ、親密な感覚
  - 20-50ms: 自然な部屋の距離感
  - 50-100ms: より大きな空間の印象を作成
- **Base Delay** - リバーブネットワークの基礎タイミング（10.0から60.0ミリ秒）
  - 低い値: よりタイトで集中したリバーブ特性
  - 高い値: より広々として開放的な音質
  - 基本的なタイミング関係に影響
- **Delay Spread** - ライン間のディレイタイムのばらつき（0.0から25.0ミリ秒）
  - 実装: 各ディレイラインはパワーカーブ（0.8指数）を使用して徐々に長くなる
  - 0.0ms: すべてのラインが同じ基本タイミング（より規則的なパターン）
  - 高い値: より自然で不規則な反射パターン
  - リアルな音響空間に見られる現実的な変化を追加
- **HF Damp** - 時間の経過とともに高周波数がフェードする程度（0.0から12.0 dB/秒）
  - 0.0: ダンピングなし、減衰全体を通して明るいサウンド
  - 3.0-6.0: 自然な空気吸収シミュレーション
  - 12.0: 温かくメローな特性のためのヘビーダンピング
- **Low Cut** - リバーブから低周波数を除去（20から500 Hz）
  - 20-50Hz: リバーブで完全なベースレスポンス
  - 100-200Hz: 濁りを避けるための制御されたベース
  - 300-500Hz: タイトで明確な低域
- **Mod Depth** - コーラスエフェクトのピッチモジュレーション量（0.0から10.0セント）
  - 0.0: モジュレーションなし、純粋な静的リバーブ
  - 2.0-5.0: 生命感とリアリズムを追加する微妙な動き
  - 10.0: 顕著なコーラスのようなエフェクト
- **Mod Rate** - モジュレーションの速度（0.10から5.00 Hz）
  - 0.1-0.5Hz: 非常にゆっくりとした穏やかな動き
  - 1.0-2.0Hz: 自然に聞こえるバリエーション
  - 3.0-5.0Hz: 素早く、より明白なモジュレーション
- **Diffusion** - ハダマールマトリックスが信号をミックスする程度（0から100%）
  - 実装: 適用されるマトリックスミキシングの量を制御
  - 0%: 最小限のミキシング、より明確なエコーパターン
  - 50%: 自然なサウンドのためのバランスの取れた拡散
  - 100%: 最も滑らかな密度のための最大ミキシング
- **Wet Mix** - サウンドに追加されるリバーブの量（0から100%）
  - 10-30%: 微妙な空間強調
  - 30-60%: 顕著なリバーブの存在感
  - 60-100%: 支配的なリバーブエフェクト
- **Dry Mix** - 保持される原音信号の量（0から100%）
  - 通常のリスニングでは通常100%に保持
  - 特別な雰囲気エフェクトのために削減可能
- **Stereo Width** - リバーブがステレオで広がる幅（0から200%）
  - 実装: 0% = モノ、100% = 通常のステレオ、200% = 誇張された幅
  - 0%: リバーブが中央に表示（モノ）
  - 100%: 自然なステレオ広がり
  - 200%: 超ワイドなステレオイメージ

### 異なるリスニング体験のための推奨設定

1. クラシック音楽の強調
   - Reverb Time: 2.5-3.5秒
   - Density: 8ライン
   - Pre Delay: 30-50ms
   - HF Damp: 4.0-6.0
   - 最適な用途: オーケストラ録音、室内楽

2. ジャズクラブの雰囲気
   - Reverb Time: 1.2-1.8秒
   - Density: 6ライン
   - Pre Delay: 15-25ms
   - HF Damp: 2.0-4.0
   - 最適な用途: アコースティックジャズ、親密なパフォーマンス

3. ポップス/ロックの強調
   - Reverb Time: 1.0-2.0秒
   - Density: 6-7ライン
   - Pre Delay: 10-30ms
   - Wet Mix: 20-40%
   - 最適な用途: モダン録音、空間の追加

4. アンビエントサウンドスケープ
   - Reverb Time: 4.0-8.0秒
   - Density: 8ライン
   - Mod Depth: 3.0-6.0
   - Wet Mix: 60-80%
   - 最適な用途: 雰囲気的な音楽、リラクゼーション

### クイックスタートガイド

1. 空間特性の設定
   - 希望する空間サイズに合わせてReverb Timeから始める
   - 滑らかで自然なサウンドのためにDensityを6-8に設定
   - 距離の知覚を制御するためにPre Delayを調整

2. 音色の形成
   - 自然な空気吸収をシミュレートするためにHF Dampを使用
   - ベースのビルドアップを防ぐためにLow Cutを設定
   - 滑らかさのためにDiffusionを調整（70-100%を試す）

3. 自然な動きの追加
   - 微妙な生命感のためにMod Depthを2-4セントに設定
   - 穏やかなバリエーションのためにMod Rateを0.3-1.0 Hz前後で使用
   - 空間的な印象のためにStereo Widthを調整

4. エフェクトのバランス
   - 30%のWet Mixから開始
   - 通常のリスニングではDry Mixを100%に保持
   - 音楽と好みに基づいて微調整

FDN Reverbは、どんな録音にも美しく自然に聞こえる残響で現実的な音響空間を追加することで、リスニング体験を変革します。お気に入りの楽曲を美しく自然なサウンドの残響で強調したい音楽愛好家に最適です！

## RS Reverb

居心地の良い部屋から荘厳なホールまで、様々な空間に音楽を移し替えることができるエフェクトです。自然なエコーと反射を加えることで、音楽をより立体的で没入感のあるものにします。

### リスニング体験ガイド
- 親密な空間:
  - 温かく居心地の良い部屋で演奏されているような感覚
  - 近距離での親密なリスニングに最適
  - 明瞭さを失うことなく微妙な奥行きを追加
- コンサートホール体験:
  - ライブパフォーマンスの壮大さを再現
  - クラシックやオーケストラ音楽に荘厳な空間を追加
  - 没入感のあるコンサート体験を作成
- 雰囲気の強調:
  - 夢のような、幻想的な質感を追加
  - アンビエントや雰囲気重視の音楽に最適
  - 魅力的なサウンドスケープを作成

### パラメータ
- **Pre-Delay** - 空間効果が始まるまでの時間を制御(0から50 ms)
  - 低い値:即座の、親密な感覚
  - 高い値:より強い距離感
  - 自然なサウンドには10msから開始
- **Room Size** - 空間の大きさを設定(2.0から50.0 m)
  - 小(2-5m):居心地の良い部屋の感覚
  - 中(5-15m):ライブルームの雰囲気
  - 大(15-50m):コンサートホールの壮大さ
- **Reverb Time** - エコーの持続時間(0.1から10.0 s)
  - 短い(0.1-1.0s):クリアでフォーカスされたサウンド
  - 中程度(1.0-3.0s):自然な部屋の響き
  - 長い(3.0-10.0s):広々とした雰囲気
- **Density** - 空間の豊かさ(4から8)
  - 低い値:よりはっきりしたエコー
  - 高い値:よりスムーズな雰囲気
  - 自然なサウンドには6から開始
- **Diffusion** - サウンドの広がり方(0.2から0.8)
  - 低い値:よりはっきりしたエコー
  - 高い値:よりスムーズなブレンド
  - バランスの取れたサウンドには0.5を試してみる
- **Damping** - エコーの減衰方法(0から100%)
  - 低い値:より明るく開放的なサウンド
  - 高い値:より温かく親密な感じ
  - 自然な感じには40%前後から開始
- **High Damp** - 空間の明るさを制御(1000から20000 Hz)
  - 低い値:より暗く温かい空間
  - 高い値:より明るく開放的
  - 自然なサウンドには8000Hz前後から開始
- **Low Damp** - 空間の充実感を制御(20から500 Hz)
  - 低い値:より充実した豊かなサウンド
  - 高い値:よりクリアで制御された感じ
  - バランスの取れた低音には100Hz前後から開始
- **Mix** - エフェクトとオリジナルサウンドのバランス(0から100%)
  - 10-30%:微妙な強調
  - 30-50%:目立つ空間
  - 50-100%:ドラマチックな効果

### 異なる音楽スタイルでの推奨設定

1. コンサートホールでのクラシック音楽
   - Room Size: 30-40m
   - Reverb Time: 2.0-2.5s
   - Mix: 30-40%
   - 最適な用途:オーケストラ作品、ピアノ協奏曲

2. 親密なジャズクラブ
   - Room Size: 8-12m
   - Reverb Time: 1.0-1.5s
   - Mix: 20-30%
   - 最適な用途:ジャズ、アコースティックパフォーマンス

3. モダンなポップス/ロック
   - Room Size: 15-20m
   - Reverb Time: 1.2-1.8s
   - Mix: 15-25%
   - 最適な用途:現代音楽

4. アンビエント/エレクトロニック
   - Room Size: 25-40m
   - Reverb Time: 3.0-6.0s
   - Mix: 40-60%
   - 最適な用途:雰囲気重視のエレクトロニック音楽

### クイックスタートガイド

1. 空間の選択
   - Room Sizeで基本的な空間を設定
   - Reverb Timeで望ましい雰囲気を調整
   - Mixで適切なバランスを調整

2. サウンドの形成
   - Dampingで温かみを制御
   - High/Low Dampで音色を調整
   - DensityとDiffusionでテクスチャを設定

3. エフェクトの微調整
   - Pre-Delayでより深みを追加
   - Mixで最終的なバランスを調整
   - 耳を信頼して好みに合わせて調整

注意:目標は自然な空間と雰囲気で音楽を強調することです。控えめな設定から始めて、リスニング体験に最適なバランスを見つけるまで調整してください!
